# Solution to ex_2_15: Asymptotic Expansion of the Bessel Function I_0(a)

## Problem Setup

We are given the integral representation:
$$I_0(a) = \frac{1}{2\pi} \int_0^{2\pi} e^{a \cos \theta} d\theta$$

and the Taylor series expansion:
$$I_0(a) = \sum_{n=0}^{\infty} \frac{a^{2n}}{2^{2n}(n!)^2}$$

We need to find the asymptotic expansion using steepest descent/stationary phase methods for $a \to +\infty$ and $a \to i\infty$.

## Case 1: $a \to +\infty$ (Real Positive)

For large positive $a$, we apply the saddle point method to:
$$I_0(a) = \frac{1}{2\pi} \int_0^{2\pi} e^{a f(\theta)} d\theta$$

where $f(\theta) = \cos \theta$ is the "phase" function.

**Finding the saddle point:** The stationary points satisfy:
$$f'(\theta) = -\sin \theta = 0$$

This gives $\theta = 0$ and $\theta = \pi$. The maximum of $\cos \theta$ over $[0, 2\pi]$ occurs at $\theta = 0$ where $\cos 0 = 1$. Since $a > 0$, the dominant contribution comes from $\theta = 0$.

**Second derivative:** At $\theta = 0$:
$$f''(\theta) = -\cos \theta \bigg|_{\theta=0} = -1$$

**Laplace's formula:** For the integral $\int e^{a f(\theta)} d\theta$ with a maximum at $\theta_0$ where $f''(\theta_0) < 0$:
$$\int_0^{2\pi} e^{a f(\theta)} d\theta \sim e^{a f(\theta_0)} \sqrt{\frac{2\pi}{a |f''(\theta_0)|}} = e^a \sqrt{\frac{2\pi}{a}}$$

Thus:
$$I_0(a) \sim \frac{1}{2\pi} e^a \sqrt{\frac{2\pi}{a}} = \frac{e^a}{\sqrt{2\pi a}}$$

**Second term:** To find the next correction, we expand $\cos \theta = 1 - \theta^2/2 + \theta^4/24 + \ldots$ around $\theta = 0$. The integral becomes:
$$I_0(a) = \frac{1}{2\pi} \int_{-\infty}^{\infty} e^{a(1 - \theta^2/2 + \theta^4/24 + \ldots)} d\theta$$

$$= \frac{e^a}{2\pi} \int_{-\infty}^{\infty} e^{-a\theta^2/2} e^{a\theta^4/24 + \ldots} d\theta$$

$$= \frac{e^a}{2\pi} \int_{-\infty}^{\infty} e^{-a\theta^2/2} \left(1 + \frac{a\theta^4}{24} + \ldots\right) d\theta$$

The zeroth order integral gives $\sqrt{2\pi/a}$. The $\theta^4$ term integral is:
$$\int_{-\infty}^{\infty} \theta^4 e^{-a\theta^2/2} d\theta = \frac{3}{4a^2} \sqrt{\frac{2\pi a}{1}} = \frac{3\sqrt{2\pi}}{4a^{5/2}}$$

This contributes: $\frac{a}{24} \cdot \frac{3\sqrt{2\pi}}{4a^{5/2}} = \frac{\sqrt{2\pi}}{32a^{3/2}}$

Therefore, the second term is $+\frac{1}{8a}$ times the leading term.

**Result for $a \to +\infty$:**
$$I_0(a) \sim \frac{e^a}{\sqrt{2\pi a}} \left(1 + \frac{1}{8a} + O(a^{-2})\right)$$

## Case 2: $a \to i\infty$ (Purely Imaginary)

Write $a = ib$ where $b \to +\infty$ is real and positive.

$$I_0(ib) = \frac{1}{2\pi} \int_0^{2\pi} e^{ib \cos \theta} d\theta$$

The phase function is now $f(\theta) = i \cos \theta$.

**Stationary points:** $f'(\theta) = -i \sin \theta = 0$ gives $\theta = 0, \pi$.

At $\theta = 0$: $f(0) = i$ and $f''(0) = -i$
At $\theta = \pi$: $f(\pi) = -i$ and $f''(\pi) = -i$

**Contour deformation:** These are saddle points of equal height (in terms of the real part: $\text{Re}(f) = 0$). Near each saddle:
- Near $\theta = 0$: $\cos \theta \approx 1 - \theta^2/2$, so $e^{ib \cos \theta} \approx e^{ib} e^{-ib\theta^2/2}$
- Near $\theta = \pi$: $\cos \theta \approx -1 + (\theta - \pi)^2/2$, so $e^{ib \cos \theta} \approx e^{-ib} e^{-ib(\theta-\pi)^2/2}$

**Combining contributions:** The two saddle points contribute equally and coherently. Using the steepest descent formula with two saddles at equal height contributing with phases $\pm ib$:

$$I_0(ib) \sim \frac{1}{2\pi} \left[ e^{ib} \sqrt{\frac{2\pi}{b}} + e^{-ib} \sqrt{\frac{2\pi}{b}} \right] = \sqrt{\frac{2}{\pi b}} \cos b$$

With higher-order corrections from the $\theta^4$ terms:
$$I_0(ib) \sim \sqrt{\frac{2}{\pi b}} \cos(b - \pi/4) + O(b^{-3/2})$$

**Result for $a \to i\infty$ (with $a = ib$, $b \to +\infty$):**
$$I_0(ib) \sim \sqrt{\frac{2}{\pi b}} \cos(b - \pi/4) + O(b^{-3/2})$$

or equivalently:
$$I_0(ib) \sim \sqrt{\frac{2}{\pi |a|}} \cos(|a| - \pi/4)$$

## Summary

$\boxed{\text{For } a \to +\infty: I_0(a) \sim \frac{e^a}{\sqrt{2\pi a}}\left(1 + \frac{1}{8a} + \ldots\right) \text{ ; For } a \to i\infty: I_0(ib) \sim \sqrt{\frac{2}{\pi b}}\cos(b - \pi/4) + \text{lower-order terms}}$
