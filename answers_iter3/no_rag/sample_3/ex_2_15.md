# Solution to ex_2_15: Asymptotic Expansion of Bessel Function I_0(a)

## Problem Setup

The Bessel function I_0(a) is defined as:
$$I_0(a) = \frac{1}{2\pi} \int_0^{2\pi} e^{a\cos\theta} d\theta$$

We need to find the asymptotic expansion as $a \to +\infty$ and as $a \to i\infty$ using steepest descent / stationary phase methods.

## Part 1: Asymptotic as $a \to +\infty$

For large positive $a$, we apply the steepest descent method to:
$$I_0(a) = \frac{1}{2\pi} \int_0^{2\pi} e^{a\cos\theta} d\theta$$

The exponent is $S(\theta) = a\cos\theta$. The stationary point occurs where:
$$\frac{dS}{d\theta} = -a\sin\theta = 0 \implies \theta = 0 \text{ or } \theta = \pi$$

For $a > 0$:
- At $\theta = 0$: $S(0) = a$ (maximum)
- At $\theta = \pi$: $S(\pi) = -a$ (minimum)

Since $a \to +\infty$, the maximum dominates. The second derivative at $\theta = 0$:
$$\frac{d^2S}{d\theta^2}\bigg|_{\theta=0} = -a\cos\theta\bigg|_{\theta=0} = -a$$

By steepest descent, we expand near $\theta = 0$:
$$e^{a\cos\theta} \approx e^a e^{-a\theta^2/2} \text{ for } \theta \text{ near } 0$$

Therefore:
$$I_0(a) \approx \frac{e^a}{2\pi} \int_{-\infty}^{\infty} e^{-a\theta^2/2} d\theta = \frac{e^a}{2\pi} \sqrt{\frac{2\pi}{a}}$$

**Leading term:**
$$I_0(a) \sim \frac{e^a}{\sqrt{2\pi a}}$$

**Second term:** A more refined expansion including the $O(a^{-1})$ correction gives:
$$I_0(a) \sim \frac{e^a}{\sqrt{2\pi a}} \left(1 + \frac{1}{8a} + \cdots\right)$$

## Part 2: Asymptotic as $a \to i\infty$ (write $a = ib$ with $b$ real, $b \to +\infty$)

With $a = ib$, we have:
$$I_0(ib) = \frac{1}{2\pi} \int_0^{2\pi} e^{ib\cos\theta} d\theta$$

The exponent is $S(\theta) = ib\cos\theta$. The stationary points occur where:
$$\frac{dS}{d\theta} = -ib\sin\theta = 0 \implies \theta = 0 \text{ or } \theta = \pi$$

For imaginary argument, both stationary points contribute (they are of equal "weight" on the imaginary axis):
- At $\theta = 0$: $S(0) = ib$ with $\frac{d^2S}{d\theta^2}\big|_0 = -ib$
- At $\theta = \pi$: $S(\pi) = -ib$ with $\frac{d^2S}{d\theta^2}\big|_\pi = ib$

Near $\theta = 0$:
$$e^{ib\cos\theta} \approx e^{ib} e^{-ib\theta^2/2}$$

Near $\theta = \pi$ (write $\theta = \pi + \phi$):
$$e^{ib\cos\theta} \approx e^{-ib} e^{-ib\phi^2/2}$$

The Gaussian integrals combine:
$$I_0(ib) \approx \frac{1}{2\pi}\left(e^{ib}\int_{-\infty}^{\infty}e^{-ib\theta^2/2}d\theta + e^{-ib}\int_{-\infty}^{\infty}e^{-ib\phi^2/2}d\phi\right)$$

Using $\int_{-\infty}^{\infty} e^{-ib\theta^2/2} d\theta = \sqrt{\pi/ib}$ (with phase):
$$\int_{-\infty}^{\infty} e^{-ib\theta^2/2} d\theta = \sqrt{\frac{2\pi}{ib}} = \sqrt{\frac{2\pi}{b}} e^{-i\pi/4}$$

Therefore:
$$I_0(ib) \approx \frac{1}{2\pi} \sqrt{\frac{2\pi}{b}} \left(e^{ib}e^{-i\pi/4} + e^{-ib}e^{i\pi/4}\right)$$

$$= \sqrt{\frac{1}{2\pi b}} \left(e^{i(b-\pi/4)} + e^{-i(b-\pi/4)}\right)$$

$$= \sqrt{\frac{2}{2\pi b}} \cos\left(b - \frac{\pi}{4}\right)$$

**Leading term:**
$$I_0(ib) \sim \sqrt{\frac{2}{\pi b}} \cos\left(b - \frac{\pi}{4}\right)$$

This includes oscillatory behavior characteristic of the Bessel function $J_0$ (which is related to $I_0$ by an imaginary argument).

## Final Answers

$$\boxed{a \to +\infty: \quad I_0(a) \sim \frac{e^a}{\sqrt{2\pi a}}\left(1 + \frac{1}{8a} + \cdots\right)}$$

$$\boxed{a \to i\infty: \quad I_0(ib) \sim \sqrt{\frac{2}{\pi b}} \cos\left(b - \frac{\pi}{4}\right) + O(b^{-3/2})}$$
