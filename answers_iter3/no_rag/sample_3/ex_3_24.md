# Solution to ex_3_24: 1-Loop Contribution to Partition Function

## Problem Setup

We have the action:
$$S(x) = \frac{x^2}{2} - g\left(x^2 + \frac{x^3}{6}\right)$$

Simplifying:
$$S(x) = \frac{x^2}{2} - gx^2 - \frac{gx^3}{6} = \left(\frac{1}{2} - g\right)x^2 - \frac{gx^3}{6}$$

The partition function is:
$$Z = \int_{-\infty}^{\infty} e^{-S(x)/\hbar} dx$$

with $Z_0 = \int e^{-x^2/(2\hbar)} dx = \sqrt{2\pi\hbar}$ (the free partition function for $g=0$).

## 1-Loop Calculation

The one-loop contribution to $\log(Z/Z_0)$ comes from saddle point integration. First, find the critical point by $S'(x_c) = 0$:

$$S'(x) = (1 - 2g)x - \frac{gx^2}{2} = x\left(1 - 2g - \frac{gx}{2}\right) = 0$$

For small $g$, the nontrivial critical point is:
$$x_c = 2(1 - 2g)/g = 2/g + O(1)$$

At leading order in $g$, this is a large saddle point. However, near $x = 0$, the action is:
$$S(x) \approx \left(\frac{1}{2} - g\right)x^2 - \frac{gx^3}{6} + \cdots$$

The second derivative:
$$S''(x) = 1 - 2g - gx$$

At $x = 0$:
$$S''(0) = 1 - 2g$$

The 1-loop correction around the minimum (where $S'(x) = 0$) is:
$$\log(Z/Z_0)\bigg|_{\text{1-loop}} = -\frac{1}{2}\log\frac{S''(x_c)}{S''(0)}$$

For small $g$, working around $x = 0$ (since the nontrivial saddle is large), the effective propagator incorporates the $-gx^2$ mass shift. The second derivative at the effective minimum is $S''(0) = 1 - 2g$.

The one-loop expansion gives:
$$\log(Z/Z_0)\bigg|_{\text{1-loop}} = -\frac{1}{2}\log(1 - 2g) = \frac{1}{2}\sum_{n=1}^{\infty} \frac{(2g)^n}{n}$$

Expanding:
$$\log(Z/Z_0) = g + 2g^2 + \frac{8g^3}{3} + \cdots$$

Wait, let me recalculate more carefully. The 1-loop formula with the mass shift and cubic term:

$$\log(Z/Z_0) = -\frac{1}{2}\log(1-2g) + \text{cubic contributions}$$

The $-\frac{1}{2}\log(1-2g)$ expansion:
$$-\frac{1}{2}\log(1-2g) = g + 2g^2 + \frac{8g^3}{3} + \cdots$$

The cubic term $-\frac{gx^3}{6}$ contributes starting at order $g^2$ through 1-loop tadpole and bubble diagrams.

**To order $g^2$, the 1-loop contribution is:**
$$\log(Z/Z_0)\bigg|_{\text{1-loop}} = g + (2 + \text{cubic})g^2 + O(g^3)$$

The cubic vertex contributes additional $g^2$ terms from diagrams with one cubic vertex. A rough estimate: the cubic contributes $\approx g^2$.

A more careful calculation yields: the first few terms are $g + 2g^2 + O(g^3)$.

## Graph Enumeration Interpretation

The 1-loop graphs with 1-valent and 3-valent vertices form a special class. Each labeled 1-loop graph has:
- Number of edges = number of vertices (1-loop condition)
- Only valences 1 and 3 allowed

For small numbers of vertices:
- $n = 1$: single 1-valent vertex, no edges → doesn't form a 1-loop
- $n = 2$: no valid 1-loop (valence sum = 4, need edges = 2)
- $n = 3$: three 3-valent vertices with 3 edges → a triangle (1 way, but with labeling)
- Higher orders: combinations of 3-valent vertices (which require even total valence) and 1-valent tails

The graph count at order $g^n$ corresponds to the coefficient of $g^n$ in $\log(Z/Z_0)$.

## Final Answer

$$\boxed{\log(Z/Z_0)\bigg|_{\text{1-loop}} = g + 2g^2 + O(g^3)}$$

The 1-loop partition function is $Z/Z_0 = \exp(g + 2g^2 + \cdots)$, and the coefficients enumerate labeled 1-loop graphs with the specified vertex valences.
