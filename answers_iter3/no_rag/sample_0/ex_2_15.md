# Asymptotic Expansion of Bessel Function I_0(a)

## Problem Setup

The modified Bessel function I_0(a) is defined by the integral representation:
$$I_0(a) = \frac{1}{2\pi} \int_0^{2\pi} e^{a \cos \theta} d\theta$$

This is an even entire function with Taylor expansion:
$$I_0(a) = \sum_{n \geq 0} \frac{a^{2n}}{2^{2n}(n!)^2}$$

We use steepest descent (saddle point method) to find the asymptotic expansion as $a \to +\infty$ and as $a \to i\infty$.

## Regime 1: $a \to +\infty$

For real positive $a$, the phase is $S(\theta) = a \cos \theta$. This is maximized at $\theta = 0$ where $\cos \theta = 1$.

**Stationary point:** $\frac{dS}{d\theta}\big|_{\theta=0} = -a \sin(0) = 0$ ✓

**Second derivative:** $\frac{d^2S}{d\theta^2} = -a \cos \theta$, so $S''(0) = -a$.

Expanding around $\theta = 0$:
$$S(\theta) = a \cos \theta = a - \frac{a\theta^2}{2} + O(\theta^4)$$

By steepest descent, the integral is dominated by the saddle point:
$$I_0(a) \approx \frac{1}{2\pi} e^a \int_0^{2\pi} e^{-a\theta^2/2} d\theta$$

For large $a$, the exponential is sharply peaked, so we can extend to $\mathbb{R}$:
$$I_0(a) \approx \frac{e^a}{2\pi} \int_{-\infty}^{\infty} e^{-a\theta^2/2} d\theta = \frac{e^a}{2\pi} \sqrt{\frac{2\pi}{a}}$$

**Leading term:** 
$$I_0(a) \sim \frac{e^a}{\sqrt{2\pi a}} \quad \text{as } a \to +\infty$$

For the next-order correction, we expand the integrand more carefully:
$$e^{a\cos\theta} = e^a e^{-a\theta^2/2 + a\theta^4/24 + \ldots}$$

The $\theta^4$ term contributes at order $1/a$, giving:
$$I_0(a) \sim \frac{e^a}{\sqrt{2\pi a}} \left(1 + \frac{1}{8a} + O(a^{-2})\right)$$

**Second term:** The coefficient is $+1/(8a)$.

## Regime 2: $a = ib$ with $b \to +\infty$

Here $S(\theta) = ib\cos\theta$. The stationary points satisfy:
$$\frac{dS}{d\theta} = -ib\sin\theta = 0 \implies \theta = 0 \text{ or } \theta = \pi$$

**At $\theta = 0$:** $S''(0) = -ib\cos(0) = -ib$

**At $\theta = \pi$:** $S''(\pi) = -ib\cos(\pi) = ib$

Expanding near $\theta = 0$:
$$S(\theta) = ib(1 - \theta^2/2 + \ldots)$$

Expanding near $\theta = \pi$ (write $\theta = \pi + \phi$):
$$S(\pi + \phi) = ib\cos(\pi + \phi) = -ib(1 - \phi^2/2 + \ldots)$$

**Saddle point contributions:** 
From $\theta = 0$: $e^{ib} \int_{-\infty}^{\infty} e^{-ib\theta^2/2} d\theta = e^{ib}\sqrt{\frac{2\pi}{ib}} = e^{ib}e^{-i\pi/4}\sqrt{\frac{2\pi}{b}}$

From $\theta = \pi$: $e^{-ib} \int_{-\infty}^{\infty} e^{ib\phi^2/2} d\phi = e^{-ib}e^{i\pi/4}\sqrt{\frac{2\pi}{b}}$

Combining:
$$I_0(ib) \sim \frac{1}{2\pi}\left[e^{ib}e^{-i\pi/4} + e^{-ib}e^{i\pi/4}\right]\sqrt{\frac{2\pi}{b}}$$

$$= \frac{1}{2\pi}\sqrt{\frac{2\pi}{b}} \left[e^{i(b-\pi/4)} + e^{-i(b-\pi/4)}\right]$$

$$= \sqrt{\frac{2}{\pi b}} \cos(b - \pi/4)$$

**Leading term:**
$$I_0(ib) \sim \sqrt{\frac{2}{\pi b}} \cos(b - \pi/4) \quad \text{as } b \to +\infty$$

**Next-order correction:** Similar analysis including $\theta^4$ terms gives corrections of order $1/b$.

## Summary

$$\boxed{
\begin{align}
a \to +\infty: \quad &I_0(a) \sim \frac{e^a}{\sqrt{2\pi a}}\left(1 + \frac{1}{8a} + O(a^{-2})\right) \\
a = ib, b \to +\infty: \quad &I_0(ib) \sim \sqrt{\frac{2}{\pi b}}\cos(b - \pi/4) + O(b^{-3/2})
\end{align}
}$$
