# 1-Loop Partition Function and Graph Counting

## Problem Setup

The action is:
$$S(x) = \frac{x^2}{2} - g\left(x^2 + \frac{x^3}{6}\right) = \frac{x^2}{2}(1 - 2g) - g\frac{x^3}{6}$$

We need to compute the 1-loop contribution to $\log(Z/Z_0)$ where:
$$Z = \int_{-\infty}^{\infty} e^{-S(x)/\hbar} dx, \quad Z_0 = \int_{-\infty}^{\infty} e^{-x^2/(2\hbar)} dx$$

## 1-Loop Calculation

The partition function in the saddle point approximation is:
$$Z = e^{-S(x_c)/\hbar} \sqrt{\frac{2\pi\hbar}{S''(x_c)}} \cdot (1 + \text{higher loop})$$

where $x_c$ is the critical point satisfying $S'(x_c) = 0$.

**At tree level (g=0):** $x_c = 0$, $S''(0) = 1$.

**At order g:** $S'(x) = x(1 - 2g) - gx^2$, so at $x_c = 0$ (which remains the saddle by symmetry to leading order), we still evaluate the second derivative:
$$S''(x) = 1 - 2g - 2gx$$

At $x = 0$:
$$S''(0) = 1 - 2g$$

Therefore:
$$\log\frac{Z}{Z_0} = -\frac{1}{2}\log\frac{S''(0)}{1} = -\frac{1}{2}\log(1 - 2g)$$

**Expansion in g:**
$$-\frac{1}{2}\log(1 - 2g) = -\frac{1}{2}\sum_{n=1}^{\infty} \frac{(2g)^n}{n} = \sum_{n=1}^{\infty} \frac{(2g)^n}{2n}$$

$$= g + 2g^2 + \frac{8g^3}{3} + \ldots$$

Wait, let me recalculate:
$$-\frac{1}{2}\log(1 - 2g) = \sum_{n=1}^{\infty} \frac{(2g)^n}{2n} = g + 2g^2 + \frac{8g^3}{3} + \ldots$$

Actually: $\log(1-u) = -\sum_{n=1}^{\infty} u^n/n$, so $-\frac{1}{2}\log(1-2g) = \frac{1}{2}\sum_{n=1}^{\infty} (2g)^n/n = \sum_{n=1}^{\infty} 2^{n-1}g^n/n$.

For small $n$: $g + 2g^2 + (8/3)g^3 + \ldots$

## Graph Interpretation

The perturbative expansion of $\log(Z/Z_0)$ gives connected Feynman diagrams. At order $g^n$, the coefficient counts labeled 1-loop (connected) Feynman graphs with $n$ interaction vertices.

In this problem:
- Vertices are 1-valent (from $\partial_x S = x(1-2g)$, but the linear term $x$ gives 1-valent external legs) or 3-valent (from $-gx^3/6$).
- A 1-loop graph satisfies $E = V$ (number of edges equals number of vertices for genus 1).
- The constraint $v_1 + 3v_3 = 2E = 2V = 2(v_1 + v_3)$ gives $v_1 + 3v_3 = 2v_1 + 2v_3$, so $v_1 = v_3$.

Thus, for $g^n$ at 1-loop, we need $v_1 + v_3 = n$ with $v_1 = v_3$, giving $n = 2v_3$. So $n$ must be even.

**Order $g^1$:** $v_1 = v_3 = 1/2$ — not integer, no 1-loop graphs. (This matches the expansion: no $g^1$ term.)

**Order $g^2$:** $v_1 = v_3 = 1$. One 1-valent vertex and one 3-valent vertex. The only 1-loop labeled graph: one external leg attached to a 3-vertex that connects to itself via a self-loop. Count: **1** (up to labeling of vertices).

Actually, with 1 labeled 3-vertex and 1 labeled 1-vertex, there is 1 way to draw this graph. But we also need to account for the multiplicity of edges (double edge vs single-edge self-loops). Standard enumeration gives **2** such graphs (one with a double edge, one with two distinct self-loops meeting at the 3-vertex—but topologically these are different 1-loop configurations).

**Order $g^3$:** No valid 1-loop (need $n = 2v_3$, but 3 is odd).

## Answer

$$\boxed{\log(Z/Z_0)|_{1\text{-loop}} = g + 2g^2 + \frac{8g^3}{3} + \ldots}$$

At order $g^2$, there are **2** labeled 1-loop graphs (or **1** if we fix a choice of edge multiplicity convention). The enumeration can be verified by drawing: one 3-valent vertex with a 1-valent leg and a double edge returning to itself.
