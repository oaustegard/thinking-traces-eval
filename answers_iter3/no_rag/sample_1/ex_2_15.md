# Solution to ex_2_15: Asymptotic Expansion of the Bessel Function I_0(a)

## Problem Setup

We are given the Bessel function defined by:
$$I_0(a) = \frac{1}{2\pi} \int_0^{2\pi} e^{a \cos \theta} d\theta$$

We need to find the asymptotic expansion as $a \to +\infty$ and as $a \to i\infty$ using steepest descent.

## Case 1: $a \to +\infty$ (Real and Positive)

**Finding the stationary point:**
The phase is $S(\theta) = a \cos \theta$. The stationary point occurs where:
$$\frac{dS}{d\theta} = -a \sin \theta = 0$$

For $\theta \in [0, 2\pi]$, this gives $\theta = 0$ and $\theta = \pi$.

At $\theta = 0$: $S(0) = a$ and $\frac{d^2S}{d\theta^2}\bigg|_{\theta=0} = -a \cos(0) = -a$ (maximum)
At $\theta = \pi$: $S(\pi) = -a$ and $\frac{d^2S}{d\theta^2}\bigg|_{\theta=\pi} = -a \cos(\pi) = a$ (minimum, not on steepest descent from $\theta=0$)

For $a > 0$, the maximum is at $\theta = 0$. The steepest descent path passes through this point.

**Applying steepest descent formula:**
Near $\theta = 0$:
$$\cos \theta = 1 - \frac{\theta^2}{2} + O(\theta^4)$$

The Gaussian approximation gives:
$$I_0(a) \approx \frac{e^a}{2\pi} \int_{-\infty}^{\infty} e^{-a\theta^2/2} d\theta = \frac{e^a}{2\pi} \sqrt{\frac{2\pi}{a}} = \frac{e^a}{\sqrt{2\pi a}}$$

**Second term:** Including the next order in the expansion of $\cos \theta = 1 - \theta^2/2 + \theta^4/24 + \ldots$:
$$I_0(a) = \frac{e^a}{\sqrt{2\pi a}} \left(1 + \frac{1}{8a} + O(a^{-2})\right)$$

## Case 2: $a \to i\infty$ (Pure Imaginary, $a = ib$ with $b \to +\infty$)

With $a = ib$, we have:
$$I_0(ib) = \frac{1}{2\pi} \int_0^{2\pi} e^{ib \cos \theta} d\theta$$

The phase is now $S(\theta) = b \cos \theta$ (real oscillatory term).

**Stationary points:** 
$\frac{dS}{d\theta} = -b \sin \theta = 0$ gives $\theta = 0$ and $\theta = \pi$.

Both are saddle points with $S''(0) = -b$ and $S''(\pi) = b$.

**Applying stationary phase:**
For large $b$, both saddle contributions oscillate. At $\theta = 0$: phase is $b$. At $\theta = \pi$: phase is $-b$.

$$I_0(ib) = \frac{1}{2\pi}\left[e^{ib} \sqrt{\frac{2\pi}{b}} e^{i\pi/4} + e^{-ib} \sqrt{\frac{2\pi}{b}} e^{-i\pi/4}\right] + \text{lower order}$$

Combining the two contributions:
$$I_0(ib) = \sqrt{\frac{2}{\pi b}} \cos\left(b - \frac{\pi}{4}\right) + O(b^{-3/2})$$

---

## Final Answer

$$\boxed{\text{For } a \to +\infty: \quad I_0(a) \sim \frac{e^a}{\sqrt{2\pi a}}\left(1 + \frac{1}{8a} + \ldots\right)}$$

$$\boxed{\text{For } a = ib \to i\infty: \quad I_0(ib) \sim \sqrt{\frac{2}{\pi b}} \cos\left(b - \frac{\pi}{4}\right) + \ldots}$$
