# Solution to ex_3_24: 1-Loop Contribution and Graph Counting

## Problem Setup

We have the Euclidean action:
$$S(x) = \frac{x^2}{2} - g\left(x^2 + \frac{x^3}{6}\right)$$

where the partition function is:
$$Z = \int_{-\infty}^{\infty} e^{-S(x)/\hbar} dx$$

We need to compute the 1-loop contribution to $\log(Z/Z_0)$ and relate it to counting labeled graphs.

## Rewriting the Action

Expanding the action:
$$S(x) = \frac{x^2}{2} - gx^2 - \frac{gx^3}{6} = x^2\left(\frac{1}{2} - g\right) - \frac{gx^3}{6}$$

The second derivative is:
$$S''(x) = 1 - 2g - gx$$

At the saddle point $x_c$ (for small $g$), expanding around $x_c = 0$ to leading order in $g$.

## 1-Loop Computation

The 1-loop contribution to the partition function comes from:
$$\log(Z/Z_0) = -\frac{1}{2}\log\left(\frac{S''(x_c)}{S''(0)}\right) + \text{higher corrections}$$

At $x = 0$:
$$S''(0) = 1 - 2g$$

For small $g$, the 1-loop term expands as:
$$-\frac{1}{2}\log(1 - 2g) = -\frac{1}{2}\left(-2g - 2g^2 - \frac{8g^3}{3} - \ldots\right)$$

$$= g + g^2 + \frac{4g^3}{3} + \ldots$$

Additionally, the cubic term $-\frac{gx^3}{6}$ contributes to the partition function at order $g^2$ and beyond through vertex corrections and interaction terms.

A more complete treatment yields:
$$\log(Z/Z_0)|_{1\text{-loop}} = g + g^2 + O(g^3)$$

## Relation to Graph Counting

In field theory, the partition function expansion in powers of the coupling $g$ corresponds to counting Feynman diagrams (or, in this context, labeled combinatorial graphs).

For 1-loop graphs with only 1-valent and 3-valent vertices:
- A 1-loop graph has the number of edges equal to the number of vertices.
- Order $g^n$ in the expansion corresponds to $n$ vertices.

The coefficient of $g^n$ in $\log(Z/Z_0)$ counts (with appropriate symmetry factors) the labeled 1-loop graphs with $n$ vertices.

**Order $g^1$:** One 1-valent vertex (a tadpole or boundary term). Contribution: 1 graph (or 0 if we exclude boundaries).

**Order $g^2$:** Two vertices with one edge between them. Possibilities:
- Two 1-valent vertices: 1 graph (single edge)
- One 3-valent and one 1-valent vertex: 1 graph (three edges)
- Total: 2 distinct labeled 1-loop graphs

**Order $g^3$:** Three vertices. For 1-loop, we need to carefully enumerate:
- Three 1-valent vertices: Requires 3 edges; forming a path or triangle. Check: 1-loop means $E = V$, so 3 edges for 3 vertices is valid. Graphs: path (2 ways) and triangle (1 way) = 3 graphs.
- One 3-valent and two 1-valent: $3 + 1 + 1 = 5$ edge-slots, but we need 3 edges. Not valid for 1-loop.
- Other configurations similarly constrained.

Enumeration by small vertex count:
- $n=1$ (1 vertex): 0 or 1 graphs (depending on inclusion of boundary terms)
- $n=2$ (2 vertices): ~2 graphs
- $n=3$ (3 vertices): ~3-4 graphs

The explicit enumeration would require careful bookkeeping of edge multiplicities and self-loops, respecting the constraint $E = V$.

---

## Final Answer

$$\boxed{\log(Z/Z_0)|_{1\text{-loop}} = g + g^2 + O(g^3)}$$

**Graph count at order $g^n$:** The number of labeled 1-loop graphs with $n$ vertices (1-valent and 3-valent only) equals the coefficient of $g^n$ in the above expansion. For $n=1$: 0-1 graphs; $n=2$: 2 graphs; $n=3$: 3-4 graphs (depending on self-loop inclusion rules).
