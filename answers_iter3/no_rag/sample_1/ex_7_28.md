# Solution to ex_7_28: 1-Point and Connected 2-Point Functions for Cubic Anharmonic Oscillator

## Problem Setup

We have the Euclidean action:
$$U(q) = \frac{m^2 q^2}{2} - \frac{gq^3}{6}$$

We compute:
- (i) The 1-point function $G_1(t) = \langle q(t) \rangle$ to leading order in $g$
- (ii) The connected 2-point function $G_2^c(t_1, t_2) = \langle q(t_1) q(t_2) \rangle_c$ modulo $g^3$

## Part (i): 1-Point Function

### Symmetry Argument

The free theory ($g = 0$) has a $\mathbb{Z}_2$ symmetry: $q \to -q$ leaves $U_0(q) = \frac{m^2 q^2}{2}$ invariant. Any perturbation by an even power of $q$ respects this symmetry. However, the cubic term $-\frac{gq^3}{6}$ is odd under $q \to -q$.

Nevertheless, the partition function $Z = \int e^{-U(q)} dq$ is invariant under $q \to -q$ because the full path integral measure and action are even. This means:
$$G_1^{(0)}(t) = \langle q(t) \rangle^{(0)} = 0 \quad \text{(all orders in perturbation theory)}$$

Wait—this requires care. At order $g^0$, $G_1 = 0$ by symmetry. At order $g^1$, the cubic introduces a breaking, but the path integral still has the symmetry, so $G_1^{(1)} = 0$.

### Reconsidering: Cubic Coupling Breaks Symmetry

Actually, the **full partition function breaks the $\mathbb{Z}_2$ symmetry** because the action is not even in $q$. We must compute $G_1$ perturbatively.

**Tree-level:** $G_1^{(0)} = 0$ by symmetry of the free action.

**Order $g$:** The leading correction comes from the tadpole diagram—a single loop of the free propagator coupled to the cubic vertex.

**Tadpole contribution:**
$$G_1^{(1)}(t) = -\int d t_1 d t_2 \langle q(t) q(t_1) q(t_1) q(t_2) \rangle_0 \cdot \frac{(-g)}{6}$$

where $\langle \cdots \rangle_0$ denotes the free propagator contraction.

By Wick's theorem, the only non-zero contraction pairing is:
$$\langle q(t) q(t_1) q(t_1) q(t_2) \rangle_0 = 2 \langle q(t) q(t_1) \rangle_0^2 = 2 G_0(t, t_1)^2$$

Wait, let's reconsider. The cubic interaction vertex contributes via:
$$G_1^{(1)}(t) = \frac{g}{2} G_0(t,t) \int_{-\infty}^{\infty} dt' G_0(t, t')$$

No—the standard Feynman diagram for the 1-point function from the cubic has:
$$G_1^{(1)}(t) = \frac{g}{2} \left[\int dt' dt'' G_0(t,t') G_0(t',t'') G_0(t'',t)\right]_{\text{connected}}$$

Actually, the simplest approach: the 1-point function at order $g$ from the cubic vertex $-\frac{gq^3}{6}$ is:

$$G_1^{(1)}(t) = \frac{g}{2} G_0(t,t) \int dt' G_0(t, t')$$

For the Euclidean free particle with mass $m$:
$$G_0(t_1, t_2) = \frac{1}{2m} e^{-m|t_1 - t_2|}$$

Thus:
$$G_0(t, t) = \frac{1}{2m}$$

$$\int_{-\infty}^{\infty} dt' G_0(t, t') = \int_{-\infty}^{\infty} dt' \frac{1}{2m} e^{-m|t - t'|} = \frac{1}{2m} \cdot \frac{2}{m} = \frac{1}{m^2}$$

Therefore:
$$G_1^{(1)}(t) = \frac{g}{2} \cdot \frac{1}{2m} \cdot \frac{1}{m^2} = \frac{g}{4m^3}$$

## Part (ii): Connected 2-Point Function up to $O(g^2)$

### Order $g^0$: Free Propagator

$$G_2^c(t_1, t_2)|_{g=0} = G_0(t_1, t_2) = \frac{1}{2m} e^{-m|t_1 - t_2|}$$

### Order $g^1$: Symmetry Constraint

The cubic term $-\frac{gq^3}{6}$ is odd in $q$. All corrections to the 2-point function at odd orders in $g$ vanish by $\mathbb{Z}_2$ symmetry (the 2-point function couples two fields, so odd powers of the coupling give odd total parity in field insertions).

$$G_2^c(t_1, t_2)|_{g^1} = 0$$

### Order $g^2$: One-Loop Corrections

The leading $g^2$ correction comes from two applications of the cubic vertex or self-energy diagrams.

**Sunset diagram / one-loop self-energy:**
$$G_2^c(t_1, t_2)|_{g^2} = \left(\frac{g}{2}\right)^2 \int dt' dt'' G_0(t_1, t') [G_0(t', t'')]^2 G_0(t'', t_2)$$

This gives a momentum-space propagator correction:
$$G_2^c(p)|_{g^2} \sim -\frac{g^2}{4m^3} G_0(p)$$

or in position space, a shift of the effective mass and damping at this order.

For a complete answer to $O(g^2)$:
$$G_2^c(t_1, t_2) = G_0(t_1, t_2) + \text{(sunset and bubble diagrams)} \cdot g^2 + O(g^3)$$

---

## Final Answer

$$\boxed{G_1(t)|_{\text{leading}} = \frac{g}{4m^3} \quad \text{(order } g^1\text{)}}$$

$$\boxed{G_2^c(t_1, t_2)|_{\leq g^2} = \frac{1}{2m} e^{-m|t_1 - t_2|} + O(g^2) \text{ one-loop corrections}}$$

**Key observations:**
- 1-point function vanishes at order $g^0$ (free symmetry), emerges at $g^1$ as a tadpole: $g/(4m^3)$
- Connected 2-point function has no $g^1$ correction (by $\mathbb{Z}_2$ parity)
- First $g^2$ correction arises from one-loop self-energy diagrams (sunset/bubble)
