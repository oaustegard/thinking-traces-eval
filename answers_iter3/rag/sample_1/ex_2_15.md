## Solution

### Setup and Phase Function

The Bessel function is defined as:
$$I_0(a) = \frac{1}{2\pi}\int_0^{2\pi} e^{a\cos\theta}\,d\theta$$

The phase function is $f(\theta) = \cos\theta$, which is an interior smooth function on $[0, 2\pi]$.

---

### Regime 1: $a \to +\infty$ (Real Exponential Decay)

**Step 1: Identify stationary points.**
$$f'(\theta) = -\sin\theta = 0 \quad \Rightarrow \quad \theta = 0 \text{ or } \pi$$

The second derivative is $f''(\theta) = -\cos\theta$, so:
- At $\theta = 0$: $f(0) = 1$, $f''(0) = -1$ (local maximum)
- At $\theta = \pi$: $f(\pi) = -1$, $f''(\pi) = 1$ (local minimum)

For $a \to +\infty$, the exponential $e^{af(\theta)}$ is dominated by the maximum at $\theta = 0$.

**Step 2: Expand near $\theta = 0$.**
$$\cos\theta = 1 - \frac{\theta^2}{2} + \frac{\theta^4}{24} + \cdots$$

So $e^{a\cos\theta} = e^a \cdot e^{-a\theta^2/2 + a\theta^4/24 + \cdots}$.

**Step 3: Apply Laplace's method.**
$$I_0(a) = \frac{1}{2\pi}\int_0^{2\pi} e^{a\cos\theta}\,d\theta \approx \frac{1}{2\pi} \cdot 2 \int_0^{\infty} e^{a(1 - \theta^2/2 + \theta^4/24 + \cdots)}\,d\theta$$

The factor of 2 accounts for the contribution from $\theta$ near 0 (upper hemisphere). Using only the leading Gaussian:
$$I_0(a) \approx \frac{e^a}{\pi}\int_0^{\infty} e^{-a\theta^2/2}\,d\theta = \frac{e^a}{\pi}\sqrt{\frac{\pi}{2a}} = \frac{e^a}{\sqrt{2\pi a}}$$

**Step 4: Second-order correction.**
Include the $\theta^4$ term: $\int_0^{\infty} e^{-a\theta^2/2}(1 + a\theta^4/24 + \cdots)d\theta$. The $\theta^4$ integral gives $\frac{3}{4}(a/2)^{-5/2}$, yielding a correction of order $1/(8a)$.

$$I_0(a) \sim \frac{e^a}{\sqrt{2\pi a}}\left(1 + \frac{1}{8a} + \cdots\right) \quad \text{as } a \to +\infty$$

---

### Regime 2: $a \to i\infty$ (Write $a = ib$ with $b$ real, $b \to +\infty$)

**Step 1: Rewrite the integral.**
$$I_0(ib) = \frac{1}{2\pi}\int_0^{2\pi} e^{ib\cos\theta}\,d\theta$$

The phase is purely imaginary: $f(\theta) = \cos\theta$, so $e^{ibf}$ oscillates.

**Step 2: Identify stationary points of the oscillation phase.**
$$f'(\theta) = -\sin\theta = 0 \quad \Rightarrow \quad \theta = 0, \pi$$

Both contribute equally. Near $\theta = 0$: $\cos\theta \approx 1 - \theta^2/2$, and near $\theta = \pi$: $\cos\theta \approx -1 + (\theta - \pi)^2/2$.

**Step 3: Combine two stationary-point contributions.**
From $\theta = 0$:
$$\int_0^{\infty} e^{ib(1 - \theta^2/2)}\,d\theta = e^{ib}\int_0^{\infty} e^{-ib\theta^2/2}\,d\theta = e^{ib}\sqrt{\frac{\pi}{2ib}}$$

From $\theta = \pi$:
$$\int_{-\infty}^{0} e^{ib(-1 + u^2/2)}\,du = e^{-ib}\int_{-\infty}^{0} e^{ibu^2/2}\,du = e^{-ib}\sqrt{\frac{\pi}{-2ib}}$$

Adding these and accounting for the oscillation sign:
$$I_0(ib) \approx \frac{1}{2\pi}\left(e^{ib}\sqrt{\frac{\pi}{2ib}} + e^{-ib}\sqrt{\frac{\pi}{2ib}}\right) = \frac{1}{\sqrt{2\pi b}}\cos\left(b - \frac{\pi}{4}\right)$$

where the $-\pi/4$ comes from the phase of $\sqrt{i} = e^{i\pi/4}$.

$$I_0(ib) \sim \sqrt{\frac{2}{\pi b}}\cos\left(b - \frac{\pi}{4}\right) \quad \text{as } b \to +\infty$$

---

## Final Answer

$$\boxed{\begin{align}
&\text{As } a \to +\infty:\quad I_0(a) \sim \frac{e^a}{\sqrt{2\pi a}}\left(1 + \frac{1}{8a} + O(a^{-2})\right)\\
&\text{As } a \to i\infty \text{ (write } a=ib\text{)}:\quad I_0(ib) \sim \sqrt{\frac{2}{\pi b}}\cos\left(b - \frac{\pi}{4}\right) + O(b^{-3/2})
\end{align}}$$
