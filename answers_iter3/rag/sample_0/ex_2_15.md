# Solution to ex_2_15: Asymptotic Expansion of Bessel Function I_0

## Retrieved Examples and Relevance

**Example 1: ex_2_14_wallis (Wallis formula)** — Highly relevant. Uses steepest descent on an integral of the form $\int_0^{\pi/2} e^{n \log \sin x} dx$ with phase function $f(x) = \log \sin x$. Demonstrates identifying the maximum of the phase and expanding around the saddle point to extract leading-order asymptotics.

**Example 2: ex_9_13_amitsur_levitzki (trace argument via determinant)** — Not relevant. This is an algebraic identity argument using graph theory and determinants; no asymptotic analysis.

**Example 3: ex_2_20_morse (Morse coordinates)** — Not relevant. Concerns local coordinate transformations via the Morse lemma; no asymptotic integrals.

**Key differences from ex_2_14_wallis:** The Wallis example treats $\int_0^{\pi/2} e^{n\log \sin x} dx$ with the maximum at the boundary. The Bessel problem has the integral $I_0(a) = \frac{1}{2\pi}\int_0^{2\pi} e^{a\cos\theta} d\theta$, which has interior saddle points and must be analyzed separately for $a \to +\infty$ (real) and $a \to i\infty$ (imaginary).

## Solution

### Part 1: Regime $a \to +\infty$ (real $a > 0$)

The integral is
$$I_0(a) = \frac{1}{2\pi}\int_0^{2\pi} e^{a\cos\theta} d\theta.$$

Rewrite as an integral of the exponential of $a\phi(\theta)$ where $\phi(\theta) = \cos\theta$:
$$I_0(a) = \frac{1}{2\pi}\int_0^{2\pi} e^{a\cos\theta} d\theta.$$

**Identify the saddle points:** $\phi'(\theta) = -\sin\theta = 0$ at $\theta = 0$ and $\theta = \pi$. Evaluate $\phi$ at these points:
- At $\theta = 0$: $\phi(0) = 1$ (maximum)
- At $\theta = \pi$: $\phi(\pi) = -1$ (minimum)

For $a \to +\infty$, the integral is dominated by the maximum of $\phi$ at $\theta = 0$.

**Steepest descent near $\theta = 0$:** Expand $\cos\theta$ around $\theta = 0$:
$$\cos\theta = 1 - \frac{\theta^2}{2} + \frac{\theta^4}{24} + O(\theta^6).$$

Thus
$$a\cos\theta = a - \frac{a\theta^2}{2} + \frac{a\theta^4}{24} + O(a\theta^6).$$

The integral is
$$I_0(a) = \frac{e^a}{2\pi}\int_0^{2\pi} e^{-a\theta^2/2 + a\theta^4/24 + \cdots} d\theta.$$

For large $a$, the main contribution comes from $|\theta| \lesssim 1/\sqrt{a}$. Extend the limits to $(-\infty, \infty)$ (the tails decay exponentially):
$$I_0(a) \approx \frac{e^a}{2\pi}\int_{-\infty}^{\infty} e^{-a\theta^2/2} d\theta = \frac{e^a}{2\pi}\sqrt{\frac{2\pi}{a}} = \frac{e^a}{\sqrt{2\pi a}}.$$

**First correction term:** Include the $\theta^4$ term in the exponent:
$$\int_{-\infty}^{\infty} e^{-a\theta^2/2 + a\theta^4/24} d\theta = \int_{-\infty}^{\infty} e^{-a\theta^2/2} e^{a\theta^4/24} d\theta.$$

Expand $e^{a\theta^4/24} = 1 + \frac{a\theta^4}{24} + O(a^2\theta^8)$. Using Gaussian moments $\int_{-\infty}^{\infty} e^{-a\theta^2/2} \theta^{2k} d\theta = (2k-1)!! \sqrt{\frac{2\pi}{a}} \cdot \frac{1}{a^k}$:

$$\int_{-\infty}^{\infty} e^{-a\theta^2/2} d\theta = \sqrt{\frac{2\pi}{a}},$$
$$\int_{-\infty}^{\infty} e^{-a\theta^2/2} \theta^4 d\theta = 3 \cdot \frac{1}{a^2} \sqrt{\frac{2\pi}{a}} = \frac{3\sqrt{2\pi}}{a^{5/2}}.$$

Thus
$$I_0(a) \approx \frac{e^a}{2\pi}\left[\sqrt{\frac{2\pi}{a}}\left(1 + \frac{a}{24} \cdot \frac{3\sqrt{2\pi}}{a^{5/2}} / \sqrt{\frac{2\pi}{a}} + \cdots\right)\right].$$

The correction is $\frac{a}{24} \cdot \frac{3}{a^2} = \frac{1}{8a}$.

**Result for $a \to +\infty$:**
$$\boxed{I_0(a) \sim \frac{e^a}{\sqrt{2\pi a}}\left(1 + \frac{1}{8a} + O(a^{-2})\right)}$$

### Part 2: Regime $a \to i\infty$ (write $a = ib$ with $b \to +\infty$ real)

For $a = ib$, the integral becomes
$$I_0(ib) = \frac{1}{2\pi}\int_0^{2\pi} e^{ib\cos\theta} d\theta.$$

The phase is $\phi(\theta) = \cos\theta$, which is real. There are two saddle points of equal height: $\theta = 0$ (max, $\phi = 1$) and $\theta = \pi$ (min, $\phi = -1$).

**Expand around both saddle points:**

Near $\theta = 0$:
$$ib\cos\theta = ib\left(1 - \frac{\theta^2}{2} + O(\theta^4)\right) = ib - \frac{ib\theta^2}{2} + \cdots$$

Near $\theta = \pi$ (set $\theta = \pi + \tau$, $\tau$ small):
$$\cos(\pi + \tau) = -\cos\tau = -1 + \frac{\tau^2}{2} + O(\tau^4),$$
$$ib\cos(\pi + \tau) = -ib + \frac{ib\tau^2}{2} + \cdots$$

The two contributions are:
- From $\theta = 0$: $e^{ib} \int e^{-ib\theta^2/2} d\theta$
- From $\theta = \pi$: $e^{-ib} \int e^{-ib\tau^2/2} d\tau$

For the Gaussian integrals with imaginary quadratic form, use $\int e^{-i\alpha t^2} dt = \sqrt{\pi/(i\alpha)} = e^{-i\pi/4}\sqrt{\pi/\alpha}$:
$$\int_{-\infty}^{\infty} e^{-ib\theta^2/2} d\theta = \sqrt{\frac{2\pi}{b}} e^{-i\pi/4}.$$

Thus the two saddle-point contributions are:
$$I_0(ib) \approx \frac{1}{2\pi}\left[e^{ib} \sqrt{\frac{2\pi}{b}} e^{-i\pi/4} + e^{-ib} \sqrt{\frac{2\pi}{b}} e^{-i\pi/4}\right].$$

Simplify:
$$I_0(ib) \approx \frac{e^{-i\pi/4}}{2\pi}\sqrt{\frac{2\pi}{b}}\left(e^{ib} + e^{-ib}\right) = \frac{e^{-i\pi/4}\sqrt{2\pi}}{2\pi\sqrt{b}} \cdot 2\cos b.$$

$$I_0(ib) \approx \sqrt{\frac{2}{\pi b}} e^{-i\pi/4} \cos b.$$

Rewrite using $e^{-i\pi/4} = \cos(\pi/4) - i\sin(\pi/4) = \frac{1}{\sqrt{2}}(1 - i)$ and $\cos b = \text{Re}(e^{ib})$:
$$I_0(ib) \approx \sqrt{\frac{2}{\pi b}} \cos\left(b - \frac{\pi}{4}\right).$$

**Result for $a = ib \to i\infty$:**
$$\boxed{I_0(ib) \sim \sqrt{\frac{2}{\pi b}} \cos\left(b - \frac{\pi}{4}\right) + O(b^{-3/2})}$$

---

## Summary

For $a \to +\infty$ real: The maximum of $\cos\theta$ at $\theta = 0$ dominates, yielding exponential growth $e^a/\sqrt{2\pi a}$ with $1/(8a)$ correction.

For $a = ib \to i\infty$: The two equal-height saddle points at $\theta = 0, \pi$ contribute coherently via steepest descent through imaginary directions, producing oscillatory cosine behavior $\sqrt{2/(\pi b)} \cos(b - \pi/4)$.

The key difference from the Wallis example is handling interior multiple saddle points of equal height in the imaginary regime.
