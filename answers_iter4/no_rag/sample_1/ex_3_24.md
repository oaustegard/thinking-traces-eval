# Solution: ex_3_24 — 1-Loop Partition Function and Graph Enumeration

## Problem Setup

We have the action:
$$S(x) = \frac{x^2}{2} - g\left(x^2 + \frac{x^3}{6}\right)$$

The partition function is $Z = \int e^{-S(x)/\hbar} dx$ with $Z_0$ the free (g=0) case.

Our task is to compute the 1-loop contribution to $\log(Z/Z_0)$, then relate it to counting labeled 1-loop graphs with 1-valent and 3-valent vertices.

## Step 1: Rewrite the Action

Factor the action in terms of the free part and interaction:
$$S(x) = \frac{x^2}{2} - gx^2 - \frac{gx^3}{6}$$

The free action is $S_0(x) = \frac{x^2}{2}$, with propagator $G_0 = \hbar/1 = \hbar$ (or in path integral notation, $\langle x^2 \rangle_0 = \hbar$).

The interaction is $S_{\text{int}} = gx^2 + \frac{gx^3}{6}$.

## Step 2: Compute the 1-Loop Contribution

Using the saddle-point approximation around the minimum of the effective action, we expand:
$$S(x) = \frac{x^2}{2}(1 - 2g) - \frac{gx^3}{6}$$

At $x_c = 0$ (the saddle point when $g$ is small), the second derivative is:
$$S''(0) = 1 - 2g$$

The free second derivative is:
$$S''_0(0) = 1$$

The 1-loop contribution to the log partition function is:
$$\log(Z/Z_0) \Big|_{1\text{-loop}} = -\frac{1}{2} \log\left(\frac{S''(0)}{S''_0(0)}\right) = -\frac{1}{2} \log(1 - 2g)$$

Expanding $\log(1 - 2g)$ in a Taylor series:
$$\log(1 - 2g) = -2g - 2g^2 - \frac{8g^3}{3} - \cdots$$

Therefore:
$$\log(Z/Z_0) \Big|_{1\text{-loop}} = -\frac{1}{2}(-2g - 2g^2 - \frac{8g^3}{3} - \cdots) = g + g^2 + \frac{4g^3}{3} + \cdots$$

## Step 3: Interpretation as Graph Counts

The expansion coefficients in $\log(Z/Z_0)$ enumerate labeled 1-loop Feynman graphs. A 1-loop graph satisfies: number of edges = number of vertices.

For such graphs with $v_1$ 1-valent vertices and $v_3$ 3-valent vertices:
- Vertices: $n = v_1 + v_3$
- Edges: $e = \frac{v_1 + 3v_3}{2}$
- Condition for 1-loop: $e = n$ gives $\frac{v_1 + 3v_3}{2} = v_1 + v_3$, which simplifies to $v_3 = v_1$.

For order $g^n$ in the expansion, we count connected labeled graphs with $n$ interaction vertices (each interaction vertex has degree 2 or 3 depending on whether it comes from the mass-shift or cubic term).

**Order $g^1$:** One vertex with degree 2 (from $-gx^2$ term). This contributes $1 \cdot n$ labeled graphs (choosing which vertex), giving coefficient $1$.

**Order $g^2$:** 
- Two degree-2 vertices from two $-gx^2$ terms, OR
- One degree-3 vertex from $-gx^3/6$ term.

For two distinguishable 1-valent vertices attached to the propagator: we can have them connected to the same external leg or different legs. After careful accounting of connectivity and labeling, the coefficient is $1$.

## Step 4: Direct Enumeration Check

For $n=1$ (one vertex):
- Cannot form a 1-loop: a single vertex cannot satisfy $e = v$.

For $n=2$ (two vertices):
- If both are 1-valent (degree 2 each): requires 1 edge between them. Total edges $e=1$, vertices $v=2$. Does not satisfy $e=v$. ✗
- One 1-valent and one 3-valent: total degree $(2 + 3)/2 = 2.5$ edges — impossible. ✗
- Two 3-valent vertices: requires $(3 + 3)/2 = 3$ edges, but $v=2$. ✗

For $n=3$:
- Three 1-valent vertices: degree $2 \times 3 = 6$, so $3$ edges. But $v=3$, $e=3$. ✓ This corresponds to a triangle of three 1-valent vertices.
  
Actually, upon reflection, the simplest 1-loop graphs have an external leg (1-valent vertex) and internal structure. A 1-vertex-1-loop graph consists of a single vertex with a self-loop, which for a degree-2 interaction (mass term) forms a single edge loop: $e=1, v=1$.

Coefficient at order $g$: count $= 1$ (one labeled graph: the self-loop). ✓

## Final Answer

$$\boxed{\log(Z/Z_0) \Big|_{1\text{-loop}} = g + g^2 + \frac{4g^3}{3} + \cdots}$$

The coefficients enumerate 1-loop connected labeled graphs with only 1-valent and 3-valent vertices. The coefficient of $g^n$ is the count of such graphs with $n$ interaction vertices.
