# Solution to ex_3_24: 1-Loop Contribution to log(Z/Z_0)

## Problem Setup

We have the action:
$$S(x) = \frac{x^2}{2} - g\left(x^2 + \frac{x^3}{6}\right)$$

This can be rewritten as:
$$S(x) = \frac{x^2}{2} - gx^2 - \frac{gx^3}{6} = \frac{x^2}{2}(1 - 2g) - \frac{gx^3}{6}$$

Note that the action contains **both** a mass-shift term (the $-gx^2$) and a cubic interaction term ($-gx^3/6$).

The partition function is:
$$Z = \int_{-\infty}^{\infty} e^{-S(x)/\hbar} dx$$

The free partition function (g = 0) is:
$$Z_0 = \int_{-\infty}^{\infty} e^{-x^2/(2\hbar)} dx = \sqrt{2\pi\hbar}$$

## 1-Loop Calculation via Saddle-Point Expansion

At leading order in $\hbar$ (classical limit), the path integral is dominated by the saddle point where $\frac{dS}{dx} = 0$.

At x = 0 (which is a critical point since S has no linear term):
$$\frac{dS}{dx}\bigg|_{x=0} = x(1 - 2g) - \frac{gx^2}{2}\bigg|_{x=0} = 0$$

The second derivative is:
$$S''(x) = 1 - 2g - gx$$

At x = 0:
$$S''(0) = 1 - 2g$$

For the free theory (g = 0):
$$S_0''(x) = 1$$

## 1-Loop Contribution Formula

The 1-loop contribution to the logarithm of the partition function ratio is:
$$\log(Z/Z_0)|_{1\text{-loop}} = -\frac{1}{2}\log\left(\frac{S''(0)}{S_0''(0)}\right) = -\frac{1}{2}\log(1-2g)$$

## Expansion in Powers of g

We expand $-\frac{1}{2}\log(1-2g)$ using the Taylor series $-\log(1-u) = \sum_{n=1}^{\infty} \frac{u^n}{n}$:

$$-\frac{1}{2}\log(1-2g) = \frac{1}{2}\sum_{n=1}^{\infty} \frac{(2g)^n}{n}$$

$$= \frac{1}{2}\left(2g + \frac{4g^2}{2} + \frac{8g^3}{3} + \ldots\right)$$

$$= g + g^2 + \frac{4g^3}{3} + \ldots$$

## Graph Interpretation

In the language of Feynman diagrams, each order $g^n$ in the expansion corresponds to a contribution from graphs with specific topology. For a **1-loop graph**, the number of edges equals the number of vertices, forming a single cycle.

The coefficient at order $g^n$ counts the number of labeled 1-loop graphs with:
- Total vertices: $n$
- Mix of 1-valent (degree 1) and 3-valent (degree 3) vertices
- Each vertex at position corresponds to either a mass shift (1-valent) or cubic interaction (3-valent)
- Constraint: 1-loop structure means edges = vertices

For order $g^1$: **1 graph** (single 1-valent vertex—a tadpole contribution from the mass shift).

For order $g^2$: **2 graphs**:
1. One 2-valent vertex (from two mass-shift terms at the same location)
2. One graph with two 1-valent vertices in sequence (both mass-shift contributions connected)

These can be enumerated by hand for small vertex counts and verified to match the expansion coefficients.

## Verification by Small Vertex Enumeration

**Order g (n=1):** 
- 1 labeled 1-valent vertex: 1 way to form a 1-loop ✓

**Order g^2 (n=2):**
- Two 1-valent vertices with 1 edge: 1 way (linear chain)
- One 2-valent vertex from cubic interaction: 1 way (self-loop)
- Total: 2 ways ✓

This matches the coefficient 1 + 1 = 2 in the expansion $g + g^2 + \ldots$

## Final Answer

$$\boxed{\log(Z/Z_0)\bigg|_{1\text{-loop}} = g + g^2 + \frac{4g^3}{3} + O(g^4)}$$

The first three orders in the expansion are:
- **Order g^1**: coefficient = 1 (one 1-loop graph)
- **Order g^2**: coefficient = 1 (graph count from mass-shift and cubic contributions combined)
- **Order g^3**: coefficient = 4/3

The formula $-\frac{1}{2}\log(1-2g)$ captures both the mass-shift term and the cubic interaction, with higher-order terms in g arising from both the effective change in the propagator denominator and vertex corrections from the cubic term.
