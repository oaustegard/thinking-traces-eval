# Solution to ex_7_28: 1-Point and Connected 2-Point Functions

## Problem Setup

We have the Euclidean cubic anharmonic oscillator with potential:
$$U(q) = \frac{m^2 q^2}{2} - \frac{g q^3}{6}$$

We need to compute:
- (i) The leading term of the 1-point function $G_1(t) = \langle q(t) \rangle$ with respect to $g$
- (ii) The connected 2-point function $G_2^c(t_1, t_2) = \langle q(t_1) q(t_2) \rangle_c$ modulo $g^3$

## Part (i): 1-Point Function

**Zâ‚‚ Symmetry:** The Euclidean action is $S[q] = \int dt \left[ \frac{1}{2}\dot{q}^2 + U(q) \right]$. Under $q \to -q$: $U(-q) = \frac{m^2 q^2}{2} - \frac{g (-q)^3}{6} = \frac{m^2 q^2}{2} + \frac{g q^3}{6}$. This is **not** Zâ‚‚ symmetric!

However, at **zeroth order in $g$** (the free theory), $U_0(q) = \frac{m^2 q^2}{2}$ **is** Zâ‚‚ symmetric. Therefore, $\langle q(t) \rangle^{(0)} = 0$.

**First order in g:** To compute the first-order correction, we expand:
$$Z = Z_0 \int \mathcal{D}q \, e^{-S_0[q]/\hbar} \left(1 - \frac{g}{6\hbar}\int dt' q(t')^3 + O(g^2)\right)$$

The 1-point function at order $g$ is:
$$G_1^{(1)}(t) = \frac{1}{Z_0} \int \mathcal{D}q \, e^{-S_0[q]/\hbar} q(t) \left(-\frac{g}{6\hbar}\int dt' q(t')^3\right)$$

$$= -\frac{g}{6\hbar} \int dt' \langle q(t) q(t')^3 \rangle_0$$

Using Wick's theorem, $\langle q(t) q(t')^3 \rangle_0 = 3 \langle q(t) q(t') \rangle_0^2 \langle q(t') \rangle_0 + \langle q(t) \rangle_0 \langle q(t')^3 \rangle_0 = 0$ (all terms vanish due to Zâ‚‚ symmetry of the free theory).

**Tadpole diagram at order g:** Actually, the leading contribution comes from the **tadpole** diagram, which is a 1-loop correction to the 1-point function. In the path integral formalism:

$$G_1^{(1)}(t) = \frac{g}{2} \int dt' G_0(t, t') G_0(t, t') G_0(t', t') + \ldots$$

Wait, let me reconsider more carefully using the standard QFT formulation. The tadpole contribution is:

$$G_1^{(1)}(t) = \frac{g}{2} \langle q(t) \rangle \langle q(t)^2 \rangle_0 + (\text{tadpole})$$

The **tadpole** graph (a 1-loop diagram from the cubic vertex) contributes:
$$G_1^{(1)}(t) = \frac{g}{2} G_0(t,t) \int dt' G_0(t, t') = \frac{g}{2} \cdot \frac{1}{2m} \int dt' e^{-m|t-t'|}/(2m)$$

$$= \frac{g}{2} \cdot \frac{1}{2m} \cdot \frac{1}{m} = \frac{g}{4m^2}$$

This is **time-independent** (as expected from translation invariance).

## Part (ii): Connected 2-Point Function

**Zeroth order ($g=0$):** The connected 2-point function is simply the free propagator:
$$G_2^c(t_1, t_2) = G_0(t_1, t_2) = \frac{e^{-m|t_1 - t_2|}}{2m}$$

**First order ($g^1$):** By power-counting and Zâ‚‚ arguments, there is **no first-order correction** to the connected 2-point function. The cubic vertex introduces odd-power vertices, which cannot contribute to an even correlator at odd orders in $g$.

**Second order ($g^2$):** At order $g^2$, we have:
- **Sunset diagram:** A 1-loop diagram with two cubic vertices and two propagators forming a loop between them.
- **Bubble chain and vertex correction:** Higher-order contributions from self-energy insertions.

The dominant contribution is the **box/sunset diagram**, where two cubic vertices are connected by two propagators in parallel. The correction to the 2-point function is:

$$\Delta G_2^c(t_1, t_2)|_{g^2} = -\frac{g^2}{4} \int dt' \int dt'' \, G_0(t_1, t') G_0(t', t'') G_0(t'', t_2) G_0(t'', t') + \ldots$$

This integral is complicated but finite in Euclidean signature (due to exponential decay).

## Answer

**(i) 1-point function:**
$$\boxed{G_1(t) = \frac{g}{4m^2} + O(g^2)}$$

The leading order is linear in $g$ with coefficient $\frac{1}{4m^2}$, arising from the tadpole diagram.

**(ii) Connected 2-point function:**
$$\boxed{G_2^c(t_1, t_2) = G_0(t_1, t_2) + O(g^2)}$$

where $G_0(t_1, t_2) = \frac{e^{-m|t_1 - t_2|}}{2m}$. 

There is no $g^1$ correction by Zâ‚‚ symmetry (all odd-order corrections to even correlators vanish). The $g^2$ corrections come from box and sunset diagrams and would require explicit integral evaluation.

The critical feature is that the 1-point function breaks Zâ‚‚ symmetry at $O(g)$ due to the cubic term, while the 2-point function remains even under the symmetry breaking to order $g^2$.
