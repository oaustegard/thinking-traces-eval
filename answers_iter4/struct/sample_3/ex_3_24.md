## Solution

**Problem Setup:** 
The action is $S(x) = \frac{x^2}{2} - g(x^2 + \frac{x^3}{6}) = \frac{x^2}{2} - gx^2 - \frac{gx^3}{6} = \frac{(1-2g)x^2}{2} - \frac{gx^3}{6}$.

The 1-loop contribution to $\log(Z/Z_0)$ is $-\frac{1}{2}\log\frac{S''(x_c)}{S''(0)}$, where $S''(x) = (1-2g) - gx$.

At the saddle point $x_c$ (determined by $S'(x_c) = 0$), we have:
$$S'(x) = (1-2g)x - \frac{gx^2}{2} = 0 \implies x_c = 0 \text{ or } x_c = 2(1-2g)/g.$$

For small $g$, the dominant saddle is $x_c \approx 2/g$ (for $g < 1/2$). However, the 1-loop partition function at order $g^n$ is determined by expanding around $x_c$ to all orders. Following ex_3_26's approach, the 1-loop term factors as:

$$\log(Z/Z_0)|_{1-loop} = -\frac{1}{2}\log\frac{S''(x_c)}{S''(0)} = -\frac{1}{2}\log\left(\frac{(1-2g) - gx_c}{1-2g}\right).$$

**Expansion in powers of $g$:**

Using $S'(x) = 0$ to eliminate the mass-shift term, we find that the 1-loop action expands to $-\frac{1}{2}\sum_{k=1}^{\infty} c_k g^k$, where the coefficients $c_k$ encode the **labeled graph counts** at order $g^k$.

From ex_3_26, the resummed contribution is $-\frac{1}{2}\log(1 - gx_c)$, which expands as:
$$\log(Z/Z_0)|_{1-loop} = \frac{1}{2}\sum_{k=1}^{\infty} \frac{(gx_c)^k}{k}.$$

Now, $gx_c = g \cdot \frac{2}{g}(1-O(g)) = 2(1-O(g))$ to leading order. But we must be careful: the full expansion in $g$ (not in $gx_c$) is what we need.

Alternatively, following the graph-counting interpretation from ex_3_25 (dissymmetry theorem for tree-rooted structures) and ex_3_26 (necklace diagrams), the 1-loop graphs with $n$ vertices all labeled with 1-valent or 3-valent types contribute at order $g^n$.

**Direct enumeration at small vertex count:**

At order $g^1$: A single 3-valent vertex with 1-loop (self-edge) contributes a **tadpole**. There is 1 such labeled graph (choosing which vertex is the 3-valent one). Coefficient: $1$.

At order $g^2$: Two 3-valent vertices connected by edges, with constraint that edges = vertices (i.e., 1-loop topology). Possibilities:
- Two 3-valent vertices with one edge between them (each vertex needs two more half-edges): this is a **single edge** between two degree-2 nodes—but in a pure graph sense with both 3-valent, we'd have 4 edges. For $E = V$ in a connected 1-loop, we need a **loop** structure. 
- More carefully: a 1-loop with $n$ vertices and $n$ edges means a single cycle of $n$ vertices. If all $n$ vertices are 3-valent, each uses 3 edges at that vertex. But $n$ vertices, each with degree 3, sums to $3n$ half-edge endpoints, which requires $3n/2$ edges. For a 1-loop cycle, $E = n$, so $3n \ne 2n$ unless $n=0$. This suggests **not all vertices can be 3-valent in a 1-loop**.

**Constraint:** Recall from the problem: 1-loop means $E = V$ for a connected graph. Summing degrees: $\sum_v \text{deg}(v) = 2E = 2V$. With $v_1$ vertices of degree 1 and $v_3$ vertices of degree 3: $v_1 + 3v_3 = 2(v_1 + v_3)$, so $v_1 + 3v_3 = 2v_1 + 2v_3$, giving $v_3 = v_1$. So in any 1-loop labeled graph, the number of 3-valent vertices equals the number of 1-valent vertices.

At order $g^n$: We have $v_1 = v_3 = n/2$ (so $n$ must be even). The total vertex count is $n$, giving a coefficient of:
$$\text{Coeff}[g^n] = \frac{n!}{(n/2)! \cdot (n/2)!} \cdot N(n/2, n/2),$$
where $N(n/2, n/2)$ is the number of unlabeled 1-loop connected graphs on $n/2$ 1-valent and $n/2$ 3-valent vertices. For $n=2$: $N(1,1) = 1$ (one 1-valent and one 3-valent vertex form a single edge, but that's degree 1 each—not 3 and 1). 

Actually, revisiting: a 1-valent vertex has degree 1; a 3-valent vertex has degree 3. For a 1-loop with $E=V=n$, the Handshaking Lemma gives $\sum \text{deg} = 2E = 2n$. With $v_1$ and $v_3$ as above: $v_1 + 3v_3 = 2n$ and $v_1 + v_3 = n$, so $v_3 = n/2$ and $v_1 = n/2$. Each order $g^{n/2}$ (since each 3-valent vertex in the action contributes a factor of $-g$) gives graphs with $n/2$ three-valent vertices and $n/2$ one-valent vertices, i.e., $n$ total vertices.

**Final formula:**
$$\log(Z/Z_0)|_{1-loop} = \sum_{n=1}^{\infty} a_n g^n,$$
where $a_n$ is the number of labeled connected 1-loop graphs on $n$ vertices with $n/2$ one-valent and $n/2$ three-valent vertices (or 0 if $n$ is odd).

**Small cases:**
- $n=2$: $v_1 = v_3 = 1$. The single edge connects them. $a_2 = 1 \cdot 1 = 1$ (two labeled vertices, one way to connect).
- $n=4$: $v_1 = v_3 = 2$. A 4-cycle with alternating 1-valent and 3-valent vertices. There are $2 \cdot 2! = 4$ labeled cycles on 4 vertices (fix one, arrange the other 3: $3! = 6$, but half for rotations... actually $\frac{4!}{2 \cdot 4} = 3$ for unlabeled, but we want labeled, so $\frac{4!}{2 \cdot 1} = 12$ for directed cycles, or $\frac{4 \cdot 3!}{2} = 12$ simple cycles on labeled vertices, divided by 2 for clockwise/counterclockwise = 6 distinct 4-cycles. But we also choose which 2 of 4 vertices are 3-valent: $\binom{4}{2} = 6$. However, not all assignments yield valid 1-loop graphs—the constraint is that degree must match the vertex type. For a 4-cycle, each vertex has degree 2, but we need 1-valent and 3-valent. Contradiction!)

**Correction:** I misread "1-valent and 3-valent vertices"—these refer to **vertex types** in the action and graph-counting sense, not graph-theoretic degree. Re-reading the problem: "number of labeled $n$-vertex 1-loop graphs with 1-valent and 3-valent vertices only."

In the context of Feynman-diagram counting (from ex_3_26 and ex_3_25 language), vertices are **interaction vertices** (from the action), not degree in the graph sense. For a 1-loop diagram on $E$ external legs and $V$ interaction vertices: Euler relation gives $V = E$ (for a connected tree or loop). The 1-loop partition function coefficients are:
$$\log(Z/Z_0)|_{1-loop} = \sum_{V=1}^{\infty} N_V g^V,$$
where $N_V$ is the number of labeled 1-loop one-particle-irreducible (1PI) topologies on $V$ three-valent vertices.

For $V=1$: the **tadpole** (one vertex with a self-loop). $N_1 = 1$.

For $V=2$: a **two-vertex 1-loop**. The two 3-valent vertices must be connected to form a loop. Each vertex has 3 edges; one connects between the two, leaving 2 + 2 = 4 internal edges forming closed loops. But we need exactly one loop (1-loop means $L=1$). Revisiting Euler: for a connected graph with $V$ vertices, $E$ edges, and $L$ independent loops (cyclomatic complexity), $L = E - V + 1$. For 1-loop ($L=1$), we have $E = V$. This matches our earlier constraint.

So for 1-loop: $E = V$. With two 3-valent vertices ($V=2$, so $E=2$ edges), each vertex has degree 3. The sum of degrees is $3+3=6 = 2E = 4$, contradiction. So **$V=2$ is impossible**.

**Resolution:** I think the problem uses "1-valent and 3-valent vertices" in a **vertex-type** sense (marking external vs interaction), not degree. In Feynman diagrams, a "1-loop graph" can have **multiple** external legs (1-valent), and the vertices are 3-valent interactions. Let me re-interpret:

A 1-loop diagram has one independent cycle. With external legs and internal edges, we sum over all allowed topologies. The generating function coefficient at order $g^k$ counts labeled **$k$-vertex** 1-loop Feynman diagrams with 3-valent vertices.

For $k=1$ (one 3-valent vertex): the **tadpole** with any number of external legs. But for partition function (no external legs), the tadpole is a single loop. Contribution: $N_1 = 1$.

**Final Answer:**

Following ex_3_26, the 1-loop effective action is 
$$\log(Z/Z_0)|_{1-loop} = -\frac{1}{2} \log(1 - 2g),$$
which expands as 
$$= \frac{1}{2}\sum_{k=1}^{\infty} \frac{(2g)^k}{k} = \sum_{k=1}^{\infty} \frac{2^{k-1}}{k} g^k.$$

The labeled graph count at order $g^k$ is $N_k = 2^{k-1} / k \cdot k! = (k-1)! \cdot 2^{k-1}$ (after converting EGF to combinatorial count). However, the problem likely asks for EGF coefficients or a direct combinatorial formula.
