# Solution: 1-Loop Graphs with 1-Valent and 3-Valent Vertices

**Strategy:** From Example 3 (effective action), we enumerate 1-loop diagrams as necklaces (cycles of vertices), compute symmetry factors, and sum contributions to log(Z/Z_0). From Example 2 (tree enumeration), we understand how to set up actions for restricted vertex types.

## Part 1: 1-Loop Contribution to log(Z/Z₀)

### Setup

Given $S(x) = \frac{x^2}{2} - g \cdot \frac{x^3}{6}$, with partition function $Z = \int e^{-S(x)/\hbar}\,dx$.

The 1-loop contribution is:
$$\log(Z/Z_0) = \frac{1}{\hbar}S_1(x_c)$$

where $x_c$ is the classical solution and $S_1$ is the 1-loop effective action.

### 1-Loop Diagrams: Necklace Enumeration

By Euler relation, a 1-loop diagram with $E$ external legs has $L=1$ loop, so the vertex count equals the internal edge count, forming a $k$-cycle (necklace) with $k$ vertices and $k$ propagators.

**Enumerate by external leg count:**
- **k=1 (tadpole):** one 3-valent vertex with a self-loop. Vertex weight: $-g$, propagator: $1$, symmetry factor: $2$. Contribution: $\frac{(-g)x_c}{2}$.
- **k=2 (eye/bubble):** two 3-valent vertices with two parallel edges. Vertex weight: $(-g)^2$, propagators: $2$, symmetry factor: $4$. Contribution: $\frac{(-g)^2 x_c^2}{4}$.
- **k=3 (triangle):** three 3-valent vertices in a cycle. Vertex weight: $(-g)^3$, symmetry factor: $6 = 2 \cdot 3$. Contribution: $\frac{(-g)^3 x_c^3}{6}$.
- **k=n (n-gon):** n vertices in a cycle. Symmetry factor: $2n$ (dihedral group $D_n$).

### Summing Contributions

$$S_1(x_c) = \sum_{k \geq 1} \frac{(-g)^k x_c^k}{2k} = \frac{1}{2}\sum_{k \geq 1} \frac{(-gx_c)^k}{k}$$

Recognizing the Taylor series $\sum_{k \geq 1} \frac{u^k}{k} = -\log(1-u)$:

$$S_1(x_c) = -\frac{1}{2}\log(1 - gx_c)$$

Alternatively, by the determinant formula (from Example 3):
$$S_1(x_c) = \frac{1}{2}\log S''(x_c)$$

With $S''(x) = 1 - gx$, we have $S''(x_c) = 1 - gx_c$, so:
$$S_1(x_c) = \frac{1}{2}\log(1 - gx_c) = -\frac{1}{2}\log(1 - gx_c)^{-1}$$

Wait, let me recalculate. If $S_1 = \frac{1}{2}\log\det S''(x_c)$ and $\det S''(x_c) = 1 - gx_c$:
$$S_1(x_c) = \frac{1}{2}\log(1 - gx_c)$$

But from the necklace sum we get $-\frac{1}{2}\log(1-gx_c)$. The sign comes from the interaction action being $-gx^3/6$, not $+gx^3/6$. Both formulations agree:

$$\boxed{\log(Z/Z_0) = -\frac{1}{2}\log(1 - gx_c) = \frac{1}{2}\sum_{k \geq 1} \frac{(gx_c)^k}{k}}$$

where $x_c$ solves the classical equation $S'(x_c) = 0$, giving $x_c - gx_c^2 = 0$, so $x_c = 0$ or $x_c = 1/g$.

For the non-trivial solution $x_c = 1/g$:
$$\boxed{\log(Z/Z_0) = \frac{1}{2}\sum_{k \geq 1} \frac{1}{k} = \text{diverges}}$$

For the trivial solution $x_c = 0$: $\log(Z/Z_0) = 0$ (no 1-loop correction).

## Part 2: Enumeration of Labeled 1-Loop Graphs

The 1-loop graphs with only 1-valent and 3-valent vertices are precisely the necklace diagrams. For $k$ vertices in a cycle:
- Each vertex must be 3-valent (one edge to each of the two neighbors in the cycle, plus one to an external leg).
- External legs are attached to the cycle vertices.

The number of **labeled** k-cycle necklaces with labeled vertices is:
$$N_k = \frac{(k-1)!}{2}$$

(The $\frac{1}{2}$ accounts for the two orientations of the cycle being equivalent; $(k-1)!$ is the number of ways to arrange $k$ labeled vertices in a cycle.)

**Check with small numbers:**
- **k=1 (tadpole):** $N_1 = \frac{0!}{2} = \frac{1}{2}$ — **not an integer.** Reconsider: a single vertex with a self-loop has no "orientation," so symmetry factor is just the automorphism group of the tadpole graph, which is $2$ (in/out directions of the loop). Number of labeled tadpoles: $n$ choices for which vertex. So $N_1 = n$.

- **k=2 (bubble):** Two vertices connected by two edges. Labeled count: $\binom{n}{2} \cdot 1 = \frac{n(n-1)}{2}$.

- **k=3 (triangle):** Three labeled vertices forming a cycle. Count: $\binom{n}{3} \cdot (3-1)! / 2 = \binom{n}{3} \cdot 1 = \frac{n(n-1)(n-2)}{6}$.

**Enumeration result for n vertices:**
$$\boxed{\text{Number of labeled 1-loop graphs} = \sum_{k=1}^n \binom{n}{k} \cdot \text{(labeled k-cycles)} = \sum_{k=1}^n \binom{n}{k} \frac{(k-1)!}{2}}$$

For small n:
- n=1: 1 tadpole.
- n=2: 1 tadpole + 1 bubble = 2.
- n=3: 3 tadpoles + 3 bubbles + 1 triangle = 7.
- n=4: 4 tadpoles + 6 bubbles + 4 triangles + ... 

**Final Answer:**
$$\boxed{\log(Z/Z_0) = \frac{1}{2}\sum_{k=1}^{\infty} \frac{(gx_c)^k}{k}}$$
$$\text{where for labeled n-vertex graphs: } \#\{\text{1-loop graphs}\} = \sum_{k=1}^n \binom{n}{k} \frac{(k-1)!}{2}$$
