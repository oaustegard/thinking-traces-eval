Example 1: **Approach:** Using Vertex Form

**Problem:** Find the increasing order of the values of the quadratic function $f(x) = x^2 + bx + c$ at three points $x=1, 2, 4$, given that $f(2 + t) = f(2 - t)$ for all real numbers $t$.

**Step 1 — Express the Function in Vertex Form:**
Since the axis of symmetry is $x = 2$ and the coefficient of $x^2$ is 1, express the function as $f(x) = (x - 2)^2 + k$.

**Step 2 — Expand and Simplify:**
Expand $f(x) = (x - 2)^2 + k$ to get $f(x) = x^2 - 4x + 4 + k$.

**Step 3 — Identify Relationships Between Coefficients:**
Compare this to the given $f(x) = x^2 + bx + c$. Thus $b = -4$ and $c = 4 + k$.

**Step 4 — Determine the Vertex:**
The vertex of the parabola is at $(2, k)$. The minimum of the quadratic occurs at $x=2$ where $f(2) = k$.

**Step 5 — Evaluate f(1) and f(4):**
Evaluate $f(1) = (1 - 2)^2 + k = 1 + k$ and $f(4) = (4 - 2)^2 + k = 4 + k$.

**Step 6 — Re-express Function Values in Terms of c:**
Since $k = c - 4$, rewrite $f(1) = 1 + (c - 4) = c - 3$ and $f(4) = 4 + (c - 4) = c$. Also $f(2) = k = c-4$.

**Step 7 — Compare Function Values and State the Order:**
Compare $f(1) = c - 3, f(2) = c - 4,$ and $f(4) = c$. Therefore, $f(2) < f(1) < f(4)$.

**Answer:** $\boxed{f(2), f(1), f(4)}$
Example 2: **Approach:** Algebraic Manipulation of Piecewise Function

**Problem:** Find $a+b$, where $a$ and $b$ are positive real numbers, given the piecewise function:
$f(x) = \begin{cases}
\frac{a}{b}x & \text{ if }x\le-4, \\
abx^2 & \text{ if }x>-4.
\end{cases}$
and the conditions $f(-4) = -\frac{60}{13}$ and $f(4) = 3120$.

**Step 1 — Apply Function Definition to First Condition:**
Since $-4 \le -4$, use the first case of the function definition to get $f(-4) = \frac{a}{b}(-4) = -\frac{4a}{b}$. Set this equal to $-\frac{60}{13}$ to get $-\frac{4a}{b} = -\frac{60}{13}$.

**Step 2 — Simplify First Equation:**
Simplify $-\frac{4a}{b} = -\frac{60}{13}$ to $\frac{a}{b} = \frac{15}{13}$.

**Step 3 — Apply Function Definition to Second Condition:**
Since $4 > -4$, use the second case of the function definition to get $f(4) = ab(4^2) = 16ab$. Set this equal to $3120$ to get $16ab = 3120$.

**Step 4 — Simplify Second Equation:**
Simplify $16ab = 3120$ to $ab = \frac{3120}{16} = 195$.

**Step 5 — Substitute to Eliminate a Variable:**
From $\frac{a}{b} = \frac{15}{13}$, express $a$ as $a = \frac{15}{13}b$ and substitute this into the equation $ab = 195$ to get $(\frac{15}{13}b)b = 195$.

**Step 6 — Solve for b:**
Simplify and solve for $b$: $\frac{15}{13}b^2 = 195 \implies b^2 = 195 \cdot \frac{13}{15} = 13 \cdot 13 = 169$. Since $b > 0$, $b = 13$.

**Step 7 — Solve for a and Find a+b:**
Substitute $b=13$ into $a = \frac{15}{13}b$ to get $a = \frac{15}{13} \cdot 13 = 15$. Finally, compute $a+b = 15 + 13 = 28$.

**Answer:** $\boxed{28}$
Example 3: **Approach:** Calculus and Optimization

**Problem:** Find the least area of a region bounded by a line passing through the point $(1, c)$ and the parabola $y = x^2$.  The constant $c > 1$.

**Step 1 — Define the Line's Equation:** Define the line passing through $(1, c)$ as $y = m(x - 1) + c$, where $m$ is the slope.

**Step 2 — Find Intersection Points:** Find the intersection points of the line and parabola by setting $x^2 = m(x - 1) + c$, which simplifies to $x^2 - mx + m - c = 0$.

**Step 3 — Express Area as an Integral:** The area $A$ between the line and parabola is given by $A = \int_{x_1}^{x_2} [(mx - m + c) - x^2] dx$, where $x_1$ and $x_2$ are the roots of the quadratic equation.

**Step 4 — Calculate the Area Formula:** Evaluate the integral: $A = \frac{1}{6} (m^2 - 4m + 4c)^{3/2}$.

**Step 5 — Minimize the Area:** To minimize $A$, minimize the expression inside the parentheses $g(m) = m^2 - 4m + 4c$.

**Step 6 — Find the Critical Point:** Find the derivative of $g(m)$ with respect to $m$ and set it equal to 0: $g'(m) = 2m - 4 = 0$, giving $m = 2$.

**Step 7 — Compute the Minimum Area:**  Substitute $m = 2$ into the area formula: $A = \frac{1}{6}(4c - 4)^{3/2} = \frac{4}{3}(c-1)^{3/2}$.

**Answer:** $\boxed{\frac{4}{3} (c - 1)^{3/2}}$
