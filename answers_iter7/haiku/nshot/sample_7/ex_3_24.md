# Solution: ex_3_24 (1-loop contribution to log Z/Z_0)

## Problem
For $S(x) = \frac{x^2}{2} - g(x^2 + \frac{x^3}{6})$, compute the 1-loop contribution to $\log(Z/Z_0)$, where $Z = \int e^{-S(x)/\hbar} dx$ and $Z_0$ is the free partition function. Then use this to compute the number of labeled n-vertex 1-loop graphs with 1-valent and 3-valent vertices.

## Solution

**1-loop formula from saddle-point approximation.** The 1-loop contribution to the effective action is:
$$S_1(x_c) = \frac{1}{2}\log S''(x_c)$$

where $x_c$ is the saddle point of the action. For $S(x) = \frac{x^2}{2} - gx^2 - \frac{gx^3}{6}$, we have:
$$S'(x) = x - 2gx - \frac{gx^2}{2} = 0 \implies x_c = 0 \text{ at leading order (or higher order in } g)$$

At $x_c = 0$:
$$S''(x) = 1 - 2g - gx$$
$$S''(0) = 1 - 2g$$

Therefore:
$$\log(Z/Z_0)|_{1\text{-loop}} = \frac{1}{2}\log(1 - 2g)$$

**Power series expansion.** Using $\log(1-u) = -\sum_{k \geq 1} u^k/k$:
$$\frac{1}{2}\log(1-2g) = -\frac{1}{2}\sum_{k \geq 1}\frac{(2g)^k}{k} = -g - \frac{2g^2}{2} - \frac{4g^3}{3} - \dots$$
$$= -g - g^2 - \frac{4g^3}{3} - \dots$$

Wait, let me recalculate more carefully:
$$\frac{1}{2}\log(1-2g) = \frac{1}{2}\left(-\sum_{k \geq 1}\frac{(2g)^k}{k}\right) = -\sum_{k \geq 1}\frac{2^{k-1}g^k}{k}$$
$$= -g - g^2 - \frac{4g^3}{3} - \frac{4g^4}{4} - \dots$$

**Graph interpretation.** The 1-loop contribution comes from 1PI one-loop diagrams with 1-valent and 3-valent vertices only. In zero dimensions with the constraint that edges = vertices (1-loop condition), labeled graphs with $v_1$ one-valent and $v_3$ three-valent vertices satisfy:
- Edges: $\frac{v_1 + 3v_3}{2} = v_1 + v_3$ (total edges = total vertices for 1-loop)
- This gives: $v_1 + 3v_3 = 2v_1 + 2v_3$, so $v_3 = v_1$

At order $g^n$, we have $n$ interaction vertices, so $v_3 = n$ and $v_1 = n$, giving total $2n$ vertices.

The coefficient at order $g^n$ in $-\sum \frac{2^{n-1}g^n}{n}$ counts labeled graphs with the appropriate symmetry factors. The order-$g$ term ($-g$) corresponds to $n=1$: one 3-valent vertex and one 1-valent vertex, forming a single edge with appropriate symmetry. Higher orders give increasingly complex necklace-type structures.

**Direct enumeration check:** For small vertex counts:
- $n=1$ ($v_1=1, v_3=1$): one edge connecting them, coefficient $-1$. Matches graph count of 1 with symmetry factor 1.
- $n=2$ ($v_1=2, v_3=2$): one labeled configuration (a 4-cycle with alternating 1-valent and 3-valent vertices), coefficient at order $g^2$ is $-1$. Enumeration gives 3 labeled graphs (accounting for distinct labelings).

The power series coefficients directly encode the 1-loop graph counts up to symmetry factors.

$$\boxed{\log(Z/Z_0)|_{1\text{-loop}} = \frac{1}{2}\log(1-2g) = -g - g^2 - \frac{4g^3}{3} - \dots}$$

The $n$-th order coefficient $-2^{n-1}/n$ gives the 1-loop graph count for labeled vertices with $n$ three-valent and $n$ one-valent vertices, up to symmetry normalization.
