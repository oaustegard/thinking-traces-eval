## Reasoning trace

**Setup.** The Euclidean action for the quantum particle is given by
$$S[q] = \int dt \left(\frac{1}{2} \dot q^2 + \frac{m^2}{2} q^2 - \frac{g}{6} q^3\right).$$
The free part of the action is $S_0[q] = \int dt \left(\frac{1}{2} \dot q^2 + \frac{m^2}{2} q^2\right)$, and the interaction part is $S_{\text{int}}[q] = \int dt \left(-\frac{g}{6} q^3\right)$.
The Feynman rules are:
- Free propagator: $G_0(t_1, t_2) = \langle q(t_1) q(t_2)\rangle_0 = \frac{1}{2m} e^{-m|t_1 - t_2|}$.
- Vertex factor: The interaction term is $-\frac{g}{6} q^3$. In the path integral expansion, this comes with a factor of $-(-g/6) = g/6$. The $1/3!$ is absorbed by combinatorial factors related to leg permutations, so each $q^3$ vertex contributes a factor of $g$.

---

**(i) Calculate the leading term of the 1-point function $G_1(t) = \langle q(t) \rangle$ with respect to $g$.**

The 1-point function $G_1(t)$ is the sum of all connected Feynman diagrams with one external leg.
In the free theory ($g=0$), $\langle q(t) \rangle_0 = 0$. So we look for the lowest order contribution in $g$.

**Order $g^1$:**
We need diagrams with one external leg and one $q^3$ vertex.
A $q^3$ vertex has three half-edges. One half-edge connects to the external point $t$. The remaining two half-edges must contract with each other to form a closed loop.

*   **Diagram:** A single $q^3$ vertex at time $t_v$. One leg connects to the external point $t$. The other two legs form a loop, giving a propagator $G_0(t_v, t_v)$.
*   **Vertex factor:** $g$.
*   **Combinatorial factor:**
    - There are $\binom{3}{1} = 3$ ways to choose which of the three half-edges connects to the external point $t$.
    - The remaining $\binom{2}{2} = 1$ way to choose the two half-edges that form the loop.
    - The loop has a symmetry factor of $S=2$ (interchanging the two ends of the loop propagator).
    - So the total combinatorial factor is $3/2$.

*   **Contribution:**
    $$\langle q(t) \rangle^{(1)} = \int dt_v \left(g \cdot \frac{3}{2}\right) \delta(t - t_v) G_0(t_v, t_v).$$
    Performing the integral over $t_v$:
    $$\langle q(t) \rangle^{(1)} = \frac{3g}{2} G_0(t, t).$$
    Using $G_0(t, t) = \frac{1}{2m}$:
    $$\langle q(t) \rangle^{(1)} = \frac{3g}{2} \cdot \frac{1}{2m} = \frac{3g}{4m}.$$
This is constant in time, as expected for a system with only a time-independent potential.

---

**(ii) Calculate the connected 2-point function $G_2^c(t_1, t_2) = \langle q(t_1) q(t_2) \rangle_c$ modulo $g^3$ (i.e., up to and including order $g^2$).**

The connected 2-point function $G_2^c(t_1, t_2)$ is the sum of all connected Feynman diagrams with two external legs. It is related to the 1-particle irreducible (1PI) 2-point function (self-energy) $\Sigma(t_1, t_2)$ by Dyson's equation:
$$G_2^c(t_1, t_2) = G_0(t_1, t_2) + \int dt_a dt_b \, G_0(t_1, t_a) \Sigma(t_a, t_b) G_0(t_b, t_2) + O(g^3).$$
We need to calculate $\Sigma(t_a, t_b)$ up to order $g^2$.

**Order $g^0$:**
The leading term is the free propagator:
$$G_2^c(t_1, t_2)^{(0)} = G_0(t_1, t_2) = \frac{1}{2m} e^{-m|t_1 - t_2|}.$$

**Order $g^1$ (for $\Sigma$):**
A 1PI 2-point function diagram must have two external legs (amputated) and cannot be disconnected by cutting a single internal line.
For a $q^3$ vertex, it has 3 half-edges. If two are external, one half-edge remains. This single remaining half-edge cannot form a loop or connect to anything else in a 1-vertex diagram.
Thus, there are no 1PI diagrams at order $g^1$.
$$\Sigma^{(1)}(t_a, t_b) = 0.$$
This implies there is no $O(g^1)$ correction to $G_2^c(t_1, t_2)$.

**Order $g^2$ (for $\Sigma$):**
We need 1PI diagrams with two external legs (amputated) and two $q^3$ vertices. Each vertex contributes a factor of $g$, so total $g^2$.
Each vertex has 3 half-edges, so two vertices have $2 \times 3 = 6$ half-edges total. Two are external, so $6-2=4$ internal half-edges remain.
The only 1PI topology with two vertices and two external legs using $q^3$ vertices is the "double-bubble" diagram:
*   **Diagram:** Two vertices, $V_a$ at $t_a$ and $V_b$ at $t_b$. One external leg attached to $V_a$, one to $V_b$. The remaining two half-edges from $V_a$ connect to the remaining two half-edges from $V_b$, forming two parallel propagators between $V_a$ and $V_b$.
*   **Vertex factors:** $g^2$.
*   **Symmetry factor:** The two parallel propagators between $V_a$ and $V_b$ can be interchanged, so $S=2$.
*   **Contribution:**
    $$\Sigma^{(2)}(t_a, t_b) = \frac{g^2}{2} [G_0(t_a, t_b)]^2.$$
    Substituting $G_0(t_a, t_b) = \frac{1}{2m} e^{-m|t_a - t_b|}$:
    $$\Sigma^{(2)}(t_a, t_b) = \frac{g^2}{2} \left(\frac{1}{2m} e^{-m|t_a - t_b|}\right)^2 = \frac{g^2}{8m^2} e^{-2m|t_a - t_b|}.$$

**Order $g^2$ (for $G_2^c$):**
Now we insert $\Sigma^{(2)}$ into Dyson's equation:
$$G_2^c(t_1, t_2)^{(2)} = \int dt_a dt_b \, G_0(t_1, t_a) \, \Sigma^{(2)}(t_a, t_b) \, G_0(t_b, t_2).$$
Substituting the expressions for $G_0$ and $\Sigma^{(2)}$:
$$G_2^c(t_1, t_2)^{(2)} = \int dt_a dt_b \, \left(\frac{1}{2m} e^{-m|t_1 - t_a|}\right) \left(\frac{g^2}{8m^2} e^{-2m|t_a - t_b|}\right) \left(\frac{1}{2m} e^{-m|t_b - t_2|}\right).$$
$$G_2^c(t_1, t_2)^{(2)} = \frac{g^2}{32m^4} \int dt_a dt_b \, e^{-m|t_1 - t_a|} e^{-2m|t_a - t_b|} e^{-m|t_b - t_2|}.$$
This integral can be evaluated using Fourier transforms. Let $\tau = t_1-t_2$. Let $k$ be the Fourier variable (energy).
The Fourier transform of $e^{-M|t|}$ is $\frac{2M}{k^2+M^2}$.
The Fourier transform of $G_0(t)$ is $\frac{1}{k^2+m^2}$.
The convolution theorem states that $\mathcal{F}[f \star g \star h] = \mathcal{F}[f] \mathcal{F}[g] \mathcal{F}[h]$.
So the Fourier transform of the integral is:
$$\mathcal{F}\left[G_2^c(t_1, t_2)^{(2)}\right] = \frac{g^2}{32m^4} \left(\frac{2m}{k^2+m^2}\right) \left(\frac{4m}{k^2+(2m)^2}\right) \left(\frac{2m}{k^2+m^2}\right).$$
$$= \frac{g^2}{32m^4} \frac{16m^3}{(k^2+m^2)^2 (k^2+4m^2)} = \frac{g^2}{2m} \frac{1}{(k^2+m^2)^2 (k^2+4m^2)}.$$
This is incorrect. The $e^{-M|t|}$ is part of the propagator, which has $1/(2M)$ factor.
The Fourier transform of $\int dt_a dt_b G_0(t_1, t_a) f(t_a-t_b) G_0(t_b, t_2)$ is $G_0(k) \mathcal{F}[f](k) G_0(k)$.
Here $f(t_a-t_b) = e^{-2m|t_a-t_b|}$. So $\mathcal{F}[f](k) = \frac{4m}{k^2+4m^2}$.
So the Fourier transform of the integral is:
$$\frac{g^2}{8m^2} \left(\frac{1}{k^2+m^2}\right) \left(\frac{4m}{k^2+4m^2}\right) \left(\frac{1}{k^2+m^2}\right) = \frac{g^2}{8m^2} \frac{4m}{(k^2+m^2)^2 (k^2+4m^2)} = \frac{g^2}{2m} \frac{1}{(k^2+m^2)^2 (k^2+4m^2)}.$$
Let $X = k^2$. We perform a partial fraction decomposition:
$$\frac{1}{(X+m^2)^2 (X+4m^2)} = \frac{A}{X+m^2} + \frac{B}{(X+m^2)^2} + \frac{C}{X+4m^2}.$$
Solving for $A, B, C$:
- Set $X=-m^2$: $B(3m^2) = 1 \implies B = \frac{1}{3m^2}$.
- Set $X=-4m^2$: $C(-3m^2)^2 = 1 \implies C = \frac{1}{9m^4}$.
- Differentiate and set $X=-m^2$: $A(2X+5m^2) + B + 2C(X+m^2) = 0 \implies A(3m^2) + B = 0 \implies A = -\frac{B}{3m^2} = -\frac{1}{9m^4}$.
So the Fourier transform is:
$$\frac{g^2}{2m} \left( -\frac{1}{9m^4} \frac{1}{k^2+m^2} + \frac{1}{3m^2} \frac{1}{(k^2+m^2)^2} + \frac{1}{9m^4} \frac{1}{k^2+4m^2} \right).$$
Now we inverse Fourier transform using $\mathcal{F}^{-1}\left[\frac{1}{k^2+M^2}\right] = \frac{1}{2M}e^{-M|t|}$ and $\mathcal{F}^{-1}\left[\frac{1}{(k^2+M^2)^2}\right] = \frac{1}{8M^3}e^{-M|t|} (1+M|t|)$.
$$G_2^c(t_1, t_2)^{(2)} = \frac{g^2}{2m} \left[ -\frac{1}{9m^4} \frac{1}{2m}e^{-m|t_1-t_2|} + \frac{1}{3m^2} \frac{1}{8m^3}e^{-m|t_1-t_2|} (1+m|t_1-t_2|) + \frac{1}{9m^4} \frac{1}{2(2m)}e^{-2m|t_1-t_2|} \right].$$
$$= \frac{g^2}{2m} \left[ -\frac{1}{18m^5}e^{-m|t_1-t_2|} + \frac{1}{24m^5}e^{-m|t_1-t_2|} (1+m|t_1-t_2|) + \frac{1}{36m^5}e^{-2m|t_1-t_2|} \right].$$
$$= \frac{g^2}{2m^6} \left[ \left(-\frac{1}{18} + \frac{1}{24}\right)e^{-m|t_1-t_2|} + \frac{1}{24}m|t_1-t_2|e^{-m|t_1-t_2|} + \frac{1}{36}e^{-2m|t_1-t_2|} \right].$$
$$= \frac{g^2}{2m^6} \left[ \left(\frac{-4+3}{72}\right)e^{-m|t_1-t_2|} + \frac{|t_1-t_2|}{24}e^{-m|t_1-t_2|} + \frac{1}{36}e^{-2m|t_1-t_2|} \right].$$
$$= \frac{g^2}{2m^6} \left[ -\frac{1}{72}e^{-m|t_1-t_2|} + \frac{|t_1-t_2|}{24}e^{-m|t_1-t_2|} + \frac{1}{36}e^{-2m|t_1-t_2|} \right].$$

Combining the terms:
$$G_2^c(t_1, t_2) = G_0(t_1, t_2) + G_2^c(t_1, t_2)^{(2)} + O(g^3).$$
$$G_2^c(t_1, t_2) = \frac{1}{2m} e^{-m|t_1-t_2|} + \frac{g^2}{2m^6} \left[ -\frac{1}{72}e^{-m|t_1-t_2|} + \frac{|t_1-t_2|}{24}e^{-m|t_1-t_2|} + \frac{1}{36}e^{-2m|t_1-t_2|} \right] + O(g^3).$$

## Final answer

**(i) The leading term of the 1-point function $G_1(t) = \langle q(t) \rangle$ is:**
$$\boxed{\;G_1(t) = \frac{3g}{4m} + O(g^2).\;}$$

**(ii) The connected 2-point function $G_2^c(t_1, t_2) = \langle q(t_1) q(t_2) \rangle_c$ modulo $g^3$ is:**
$$\boxed{\;G_2^c(t_1, t_2) = \frac{1}{2m} e^{-m|t_1-t_2|} + \frac{g^2}{2m^6} \left[ -\frac{1}{72}e^{-m|t_1-t_2|} + \frac{|t_1-t_2|}{24}e^{-m|t_1-t_2|} + \frac{1}{36}e^{-2m|t_1-t_2|} \right] + O(g^3).\;}$$
