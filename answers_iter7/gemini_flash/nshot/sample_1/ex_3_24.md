## Reasoning trace

The problem has two parts:
1.  Compute the 1-loop contribution to $\log(Z/Z_0)$ for a given action.
2.  Compute the number of labeled $n$-vertex 1-loop graphs with specific valences, and verify for small $n$.

---

### Part 1: 1-loop contribution to $\log(Z/Z_0)$

**Setup:**
The action is $S(x) = \frac{x^2}{2} - g\left(x^2 + \frac{x^3}{6}\right) = \frac{(1-g)x^2}{2} - \frac{gx^3}{6}$.
The partition function is $Z = \int dx \, e^{-S(x)/\hbar}$.
The free partition function $Z_0$ corresponds to $g=0$, so $S_0(x) = x^2/2$.
$Z_0 = \int dx \, e^{-x^2/(2\hbar)} = \sqrt{2\pi\hbar}$.

**1-loop contribution to $\log Z$ (zero-dimensional QFT):**
In zero dimensions, the 1-loop contribution to $\log Z$ is given by the Gaussian integral around the classical solution $x_c$.
The formula is $\log Z = -\frac{1}{\hbar}S(x_c) + \frac{1}{2}\log(2\pi\hbar) - \frac{1}{2}\log S''(x_c) + O(\hbar)$.
Here, $x_c$ is a classical solution, meaning $S'(x_c) = 0$. We choose the solution that minimizes $S(x)$.

First, calculate $S'(x)$ and $S''(x)$:
$S'(x) = \frac{d}{dx}\left(\frac{(1-g)x^2}{2} - \frac{gx^3}{6}\right) = (1-g)x - \frac{gx^2}{2}$.
$S''(x) = \frac{d}{dx}\left((1-g)x - \frac{gx^2}{2}\right) = (1-g) - gx$.

Now, find the classical solutions $x_c$ by setting $S'(x_c) = 0$:
$(1-g)x_c - \frac{gx_c^2}{2} = 0 \Rightarrow x_c\left((1-g) - \frac{gx_c}{2}\right) = 0$.
This gives two solutions: $x_c = 0$ or $x_c = \frac{2(1-g)}{g}$.

To determine which solution minimizes $S(x)$, we check the sign of $S''(x_c)$:
- At $x_c = 0$: $S''(0) = (1-g)$. For a minimum, we need $1-g > 0$, i.e., $g < 1$.
- At $x_c = \frac{2(1-g)}{g}$: $S''\left(\frac{2(1-g)}{g}\right) = (1-g) - g\left(\frac{2(1-g)}{g}\right) = (1-g) - 2(1-g) = -(1-g)$. For a minimum, we need $-(1-g) > 0$, i.e., $g > 1$.

Assuming $g < 1$, the minimum of $S(x)$ is at $x_c = 0$.
In this case:
- $S(x_c) = S(0) = 0$.
- $S''(x_c) = S''(0) = 1-g$.

Substitute these into the formula for $\log Z$:
$\log Z = -\frac{1}{\hbar}(0) + \frac{1}{2}\log(2\pi\hbar) - \frac{1}{2}\log(1-g) + O(\hbar) = \frac{1}{2}\log(2\pi\hbar) - \frac{1}{2}\log(1-g)$.

For the free theory ($g=0$), $S_0(x) = x^2/2$. The classical solution is $x_c=0$.
$S_0(0) = 0$, $S_0''(0) = 1$.
So, $\log Z_0 = \frac{1}{2}\log(2\pi\hbar) - \frac{1}{2}\log(1) = \frac{1}{2}\log(2\pi\hbar)$.

Finally, compute $\log(Z/Z_0)$:
$\log(Z/Z_0) = \log Z - \log Z_0 = \left(\frac{1}{2}\log(2\pi\hbar) - \frac{1}{2}\log(1-g)\right) - \frac{1}{2}\log(2\pi\hbar) = -\frac{1}{2}\log(1-g)$.

This expression represents the full 1-loop contribution, resumming all orders in $g$.

### Part 2: Number of labeled $n$-vertex 1-loop graphs with 1-valent and 3-valent vertices only.

Let $n_1$ be the number of 1-valent vertices and $n_3$ be the number of 3-valent vertices.
The total number of vertices is $n = n_1 + n_3$.

For any graph, the sum of valences is $2E$, where $E$ is the number of edges:
$1 \cdot n_1 + 3 \cdot n_3 = 2E$.

For a connected graph with $V$ vertices and $E$ edges, the number of loops $L$ is given by $L = E - V + 1$.
In this problem, we are looking for 1-loop graphs, so $L=1$.
$1 = E - n + 1 \Rightarrow E = n$.

Substitute $E=n$ into the valence sum equation:
$n_1 + 3n_3 = 2n$.

We have a system of two equations:
1. $n_1 + n_3 = n$
2. $n_1 + 3n_3 = 2n$

Subtracting (1) from (2):
$(n_1 + 3n_3) - (n_1 + n_3) = 2n - n$
$2n_3 = n \Rightarrow n_3 = n/2$.
Substitute $n_3 = n/2$ into (1):
$n_1 + n/2 = n \Rightarrow n_1 = n/2$.

This implies that $n$ must be an even number. If $n$ is odd, there are no such graphs.
Also, we require $n_1 \ge 0$ and $n_3 \ge 0$, so $n \ge 2$ (the smallest cycle requires at least 1 vertex with a self-loop, or 2 vertices with a double edge).

**Structure of the graph:**
Since 1-valent vertices cannot be part of a cycle, the single cycle in the graph must be formed entirely by 3-valent vertices.
Each 3-valent vertex in the cycle uses two of its incident edges to form the cycle. The remaining one edge must connect to another part of the graph.
Since there is only one loop, any path branching off the cycle must eventually terminate in a 1-valent vertex (a leaf).
Given $n_1 = n_3 = n/2$, it must be that each of the $n_3$ 3-valent vertices in the cycle is connected to exactly one of the $n_1$ 1-valent vertices. If any 3-valent vertex were not part of the cycle, it would have to connect to 3 other vertices, leading to a structure inconsistent with $n_1 = n_3 = n/2$ and only one loop.
Thus, the graph structure is uniquely determined: an $n_3$-cycle (length $n/2$) where each vertex of the cycle has exactly one 1-valent vertex attached to it.

**Counting labeled graphs:**
Let $k = n/2$ (so $k = n_1 = n_3$).
1.  **Choose $k$ vertices to be 3-valent (and the remaining $k$ to be 1-valent):** There are $\binom{n}{k} = \binom{n}{n/2}$ ways to do this.
2.  **Form a cycle of length $k$ using the $k$ chosen 3-valent vertices:**
    *   If $k=1$ (i.e., $n=2$): The single 3-valent vertex forms a self-loop. There is $C(1)=1$ way.
    *   If $k=2$ (i.e., $n=4$): The two 3-valent vertices form a double edge between them. There is $C(2)=1$ way.
    *   If $k \ge 3$ (i.e., $n \ge 6$): The $k$ 3-valent vertices form a simple cycle. The number of ways to form a simple cycle on $k$ specified labeled vertices is $C(k) = (k-1)!/2$.
3.  **Connect the $k$ 1-valent vertices to the $k$ 3-valent vertices:** Each 3-valent vertex in the cycle has one free half-edge, and each 1-valent vertex has one free half-edge. These must be paired up. Since there are $k$ of each, this is a perfect matching, and there are $k! = (n/2)!$ ways to do this.

Combining these steps, the total number of such labeled graphs $N_n$ for even $n$ is:
$N_n = \binom{n}{n/2} \times C(n/2) \times (n/2)!$.
Substitute $\binom{n}{n/2} = \frac{n!}{(n/2)!(n/2)!}$:
$N_n = \frac{n!}{(n/2)!(n/2)!} \times C(n/2) \times (n/2)! = \frac{n!}{(n/2)!} \times C(n/2)$.

**Verification for small $n$ (even):**
*   **For $n=2$:** $k=n/2=1$. $N_2 = \frac{2!}{1!} \times C(1) = 2 \times 1 = 2$.
    *   Graphs: (1) Vertex 1 is 1-valent, Vertex 2 is 3-valent. Edges are (1,2) and (2,2) (self-loop on 2).
    *   (2) Vertex 2 is 1-valent, Vertex 1 is 3-valent. Edges are (1,2) and (1,1) (self-loop on 1).
    *   Matches.
*   **For $n=4$:** $k=n/2=2$. $N_4 = \frac{4!}{2!} \times C(2) = 12 \times 1 = 12$.
    *   Choose 2 vertices to be 1-valent (e.g., $\{1,2\}$) and 2 to be 3-valent (e.g., $\{3,4\}$). $\binom{4}{2}=6$ ways.
    *   The 3-valent vertices $\{3,4\}$ form a double edge. (1 way).
    *   The 1-valent vertices $\{1,2\}$ connect to $\{3,4\}$. (1 connects to 3, 2 connects to 4) OR (1 connects to 4, 2 connects to 3). $2!=2$ ways.
    *   Total: $6 \times 1 \times 2 = 12$. Matches.
*   **For $n=6$:** $k=n/2=3$. $N_6 = \frac{6!}{3!} \times C(3) = 120 \times ((3-1)!/2) = 120 \times 1 = 120$.
    *   Choose 3 vertices to be 1-valent (e.g., $\{1,2,3\}$) and 3 to be 3-valent (e.g., $\{4,5,6\}$). $\binom{6}{3}=20$ ways.
    *   The 3-valent vertices $\{4,5,6\}$ form a simple cycle (e.g., edges $(4,5),(5,6),(6,4)$). (1 way for specified vertices).
    *   The 1-valent vertices $\{1,2,3\}$ connect to $\{4,5,6\}$. $3!=6$ ways.
    *   Total: $20 \times 1 \times 6 = 120$. Matches.

## Final answer

### Part 1: 1-loop contribution to $\log(Z/Z_0)$

The 1-loop contribution to $\log(Z/Z_0)$ is given by:
$$ \boxed{-\frac{1}{2}\log(1-g)} $$
This result is obtained by evaluating the Gaussian part of the path integral around the classical solution $x_c=0$, which is valid for $g<1$.

### Part 2: Number of labeled $n$-vertex 1-loop graphs with 1-valent and 3-valent vertices only

For such graphs to exist, the number of vertices $n$ must be **even**. If $n$ is odd, the number of such graphs is 0.

For even $n$, there must be $n_1 = n/2$ 1-valent vertices and $n_3 = n/2$ 3-valent vertices. The structure of such a graph is an $n_3$-cycle where each vertex of the cycle has exactly one 1-valent vertex attached to it.

The number of such labeled graphs, $N_n$, is given by:
$$ \boxed{ N_n = \frac{n!}{(n/2)!} \cdot C(n/2) } $$
where $k = n/2$, and $C(k)$ is the number of ways to form a cycle on $k$ specified labeled vertices:
*   $C(1) = 1$ (for $k=1$, a self-loop)
*   $C(2) = 1$ (for $k=2$, a double edge)
*   $C(k) = \frac{(k-1)!}{2}$ (for $k \ge 3$, a simple cycle)

**Verification for small $n$ (even):**
*   For $n=2$: $N_2 = \frac{2!}{1!} \cdot C(1) = 2 \cdot 1 = 2$.
*   For $n=4$: $N_4 = \frac{4!}{2!} \cdot C(2) = 12 \cdot 1 = 12$.
*   For $n=6$: $N_6 = \frac{6!}{3!} \cdot C(3) = 120 \cdot \frac{(3-1)!}{2} = 120 \cdot 1 = 120$.
