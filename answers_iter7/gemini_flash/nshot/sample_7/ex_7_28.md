## Reasoning trace

**Setup.** The Euclidean action for the quantum particle is given by
$$S[q] = \int dt \left(\frac{1}{2} \dot q^2 + \frac{m^2}{2} q^2 - \frac{g}{6} q^3\right).$$
The free part of the action is $S_0[q] = \int dt \left(\frac{1}{2} \dot q^2 + \frac{m^2}{2} q^2\right)$.
The interaction part is $S_{\text{int}}[q] = -\int dt \frac{g}{6} q^3$.
The free propagator (Green's function) is:
$$G_0(t_1, t_2) = \langle q(t_1) q(t_2)\rangle_0 = \frac{1}{2m} e^{-m|t_1 - t_2|}.$$
The Feynman rule for a $q^3$ vertex, originating from the expansion $e^{-S_{\text{int}}}$, is a factor of $g$. (The $1/6$ in the action cancels the $3!$ permutations of legs at the vertex).

**Part (i): Leading term of the 1-point function $G_1(t) = \langle q(t) \rangle$.**

The 1-point function is the sum of all connected Feynman diagrams with one external leg.

**Order $g^0$.** In the absence of interactions ($g=0$), the ground state is symmetric around $q=0$, so $\langle q(t) \rangle_0 = 0$.

**Order $g^1$.** We need diagrams with one vertex and one external leg.
A $q^3$ vertex has three half-edges. One is connected to the external time $t$. The remaining two must connect to each other to form a closed loop (a tadpole diagram).

The diagram consists of:
- One $q^3$ vertex at time $t_v$. Vertex factor: $g$.
- One external propagator $G_0(t, t_v)$.
- One internal loop propagator $G_0(t_v, t_v)$.
- Symmetry factor: The two internal legs forming the loop are indistinguishable, so $S=2$.

The contribution is:
$$G_1^{(1)}(t) = \int dt_v \, \frac{1}{2} \cdot g \cdot G_0(t, t_v) \, G_0(t_v, t_v).$$
We use $G_0(t_v, t_v) = \frac{1}{2m}$.
$$G_1^{(1)}(t) = \frac{g}{2} \cdot \frac{1}{2m} \int dt_v \, G_0(t, t_v) = \frac{g}{4m} \int dt_v \, \frac{1}{2m} e^{-m|t - t_v|}.$$
The integral $\int_{-\infty}^\infty e^{-m|t-t_v|} dt_v = \int_{-\infty}^t e^{m(t-t_v)} dt_v + \int_t^\infty e^{-m(t-t_v)} dt_v = \left[\frac{1}{m}e^{m(t-t_v)}\right]_{-\infty}^t + \left[-\frac{1}{m}e^{-m(t-t_v)}\right]_t^\infty = \frac{1}{m} - 0 + 0 - (-\frac{1}{m}) = \frac{2}{m}$.
So,
$$G_1^{(1)}(t) = \frac{g}{4m} \cdot \frac{1}{2m} \cdot \frac{2}{m} = \frac{g}{4m^3}.$$
This is the leading term of $G_1(t)$. It is a constant, as expected for a translationally invariant system.

**Part (ii): Connected 2-point function $G_2^c(t_1, t_2) = \langle q(t_1) q(t_2) \rangle_c$ modulo $g^3$.**

The connected 2-point function is the sum of all connected Feynman diagrams with two external legs.

**Order $g^0$.** This is the free propagator:
$$G_2^{c(0)}(t_1, t_2) = G_0(t_1, t_2) = \frac{1}{2m} e^{-m|t_1 - t_2|}.$$

**Order $g^1$.** We need diagrams with one vertex and two external legs.
A $q^3$ vertex has three legs. If two are external (to $t_1, t_2$), the third leg must form a loop. But this would mean the vertex has 1 external leg and 2 internal legs, which is a contribution to $G_1(t)$, not $G_2(t_1, t_2)$.
A $q^3$ vertex must have an odd number of external legs. Therefore, no $g^1$ contribution to $G_2^c$.
$$G_2^{c(1)}(t_1, t_2) = 0.$$

**Order $g^2$.** We need diagrams with two $q^3$ vertices and two external legs. Each vertex contributes a factor $g$. Total 6 half-edges from the two vertices. Two are external (to $t_1, t_2$). The remaining four must form internal connections.

There are two distinct topologies:
1.  **Two vertices connected by one propagator, each with a tadpole.**
    -   Diagram: $q(t_1) - V_a - V_b - q(t_2)$, with a loop attached to $V_a$ and a loop attached to $V_b$.
    -   Vertex factors: $g^2$.
    -   Propagators: $G_0(t_1, t_a)$, $G_0(t_a, t_b)$, $G_0(t_b, t_2)$, $G_0(t_a, t_a)$, $G_0(t_b, t_b)$.
    -   Symmetry factor: The graph has a $\mathbb{Z}_2$ symmetry by swapping $V_a$ and $V_b$ (which also swaps their attached loops and external legs). So $S=2$.
    -   Contribution:
        $$\frac{1}{2} g^2 \int dt_a dt_b \, G_0(t_1, t_a) G_0(t_a, t_b) G_0(t_b, t_2) \, G_0(t_a, t_a) G_0(t_b, t_b).$$
        Using $G_0(t_a, t_a) = G_0(t_b, t_b) = \frac{1}{2m}$:
        $$C_1 = \frac{g^2}{2} \left(\frac{1}{2m}\right)^2 \int dt_a dt_b \, G_0(t_1, t_a) G_0(t_a, t_b) G_0(t_b, t_2) = \frac{g^2}{8m^2} (G_0 \star G_0 \star G_0)(t_1, t_2).$$
        The $N$-fold convolution of $G_0(t)$ is $\mathcal{F}^{-1}[ (1/(\omega^2+m^2))^N ]$. For $N=3$, this is:
        $$(G_0 \star G_0 \star G_0)(t) = \frac{e^{-m|t|}}{8m^5} (3 + 3m|t| + m^2t^2).$$
        Let $T = |t_1-t_2|$. So $C_1 = \frac{g^2}{8m^2} \frac{e^{-mT}}{8m^5} (3 + 3mT + m^2T^2) = \frac{g^2 e^{-mT}}{64m^7} (3 + 3mT + m^2T^2).$$

2.  **Two vertices connected by two propagators (a "double-line" connection).**
    -   Diagram: $q(t_1) - V_a = V_b - q(t_2)$.
    -   Vertex factors: $g^2$.
    -   Propagators: $G_0(t_1, t_a)$, $G_0(t_b, t_2)$, and two propagators $G_0(t_a, t_b)^2$.
    -   Symmetry factor: The graph has a $\mathbb{Z}_2$ symmetry for swapping $V_a$ and $V_b$ (and $t_1, t_2$). It also has a $\mathbb{Z}_2$ symmetry for swapping the two propagators between $V_a$ and $V_b$. So $S = 2 \times 2 = 4$.
    -   Contribution:
        $$\frac{1}{4} g^2 \int dt_a dt_b \, G_0(t_1, t_a) [G_0(t_a, t_b)]^2 G_0(t_b, t_2).$$
        This is $\frac{g^2}{4} (G_0 \star G_0^2 \star G_0)(t_1, t_2)$.
        The convolution $(G_0 \star G_0^2 \star G_0)(t)$ is $\mathcal{F}^{-1}[ (1/(\omega^2+m^2))^4 ]$. For $N=4$:
        $$(G_0 \star G_0^2 \star G_0)(t) = \frac{e^{-m|t|}}{96m^7} (15 + 15m|t| + 6m^2t^2 + m^3|t|^3).$$
        So $C_2 = \frac{g^2}{4} \frac{e^{-mT}}{96m^7} (15 + 15mT + 6m^2T^2 + m^3T^3) = \frac{g^2 e^{-mT}}{384m^7} (15 + 15mT + 6m^2T^2 + m^3T^3).$$

**Total $G_2^c$ up to $O(g^2)$.**
$G_2^c(t_1, t_2) = G_0(t_1, t_2) + C_1 + C_2 + O(g^3)$.
Let $T = |t_1-t_2|$.
$G_2^{c(2)}(t_1, t_2) = \frac{g^2 e^{-mT}}{64m^7} (3 + 3mT + m^2T^2) + \frac{g^2 e^{-mT}}{384m^7} (15 + 15mT + 6m^2T^2 + m^3T^3).$
To sum these, find a common denominator (384):
$\frac{6}{384} (3 + 3mT + m^2T^2) = \frac{18 + 18mT + 6m^2T^2}{384}$.
Adding the numerators: $(18 + 18mT + 6m^2T^2) + (15 + 15mT + 6m^2T^2 + m^3T^3) = 33 + 33mT + 12m^2T^2 + m^3T^3$.
So,
$$G_2^{c(2)}(t_1, t_2) = \frac{g^2 e^{-m|t_1-t_2|}}{384m^7} (33 + 33m|t_1-t_2| + 12m^2(t_1-t_2)^2 + m^3|t_1-t_2|^3).$$

## Final answer

**(i) Leading term of the 1-point function $G_1(t) = \langle q(t) \rangle$:**

$$\boxed{\;G_1(t) = \frac{g}{4m^3} + O(g^2).\;}$$

**(ii) Connected 2-point function $G_2^c(t_1, t_2) = \langle q(t_1) q(t_2) \rangle_c$ modulo $g^3$:**

Let $T = |t_1 - t_2|$. The connected 2-point function is given by:
$$\boxed{\;G_2^c(t_1, t_2) = \frac{1}{2m} e^{-mT} + \frac{g^2 e^{-mT}}{384m^7} \left(33 + 33mT + 12m^2T^2 + m^3T^3\right) + O(g^3).\;}$$
