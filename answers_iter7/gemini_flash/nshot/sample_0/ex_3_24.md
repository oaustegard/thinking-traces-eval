$$S(x) = \frac{x^2}{2} - g x^2 - \frac{g x^3}{6} = \frac{x^2}{2}(1 - 2g) - \frac{g x^3}{6}.$$

$$S''(x) = (1 - 2g) - g x.$$
At $x_c = 0$, we have $S''(0) = 1 - 2g$.
The general formula for the 1-loop contribution to $\log Z$ (relative to the free theory) in 0D QFT is $\log(Z/Z_0)_{\text{1-loop}} = -\frac{1}{2}\log(S''(x_c))$.

### Part 2: Number of labeled $n$-vertex 1-loop graphs with 1-valent and 3-valent vertices only.

2.  **Constraints on $n$ and vertex counts**:

    $n + 2V_3 = 2n$
    $2V_3 = n \implies V_3 = n/2$.
    This implies that $n$ must be an even number. Also, $V_1 = n - V_3 = n - n/2 = n/2$.
    So, any such graph must have exactly $n/2$ 1-valent vertices and $n/2$ 3-valent vertices.

3.  **Structure of such graphs**:
    A 1-loop graph contains exactly one cycle. Let the length of this cycle be $k$.
    Since all vertices must be 1-valent or 3-valent, no vertex can have degree 2. This means that every vertex on the cycle must have degree 3 (as it connects to two other vertices in the cycle, plus one more edge).
    Each of these $k$ 3-valent cycle vertices has two edges within the cycle. Its third edge must connect to another vertex. Since all other vertices must be 1-valent (leaves), each of these $k$ third edges must connect to a unique 1-valent vertex.
    Therefore, the graph must consist of a $k$-cycle where each of the $k$ cycle vertices is connected to exactly one distinct 1-valent vertex (leaf).
    This structure yields $k$ 3-valent vertices (on the cycle) and $k$ 1-valent vertices (the leaves).
    The total number of vertices is $n = k + k = 2k$. This confirms our earlier finding that $k = n/2$.

4.  **Minimum $n$ and enumeration**:
    The minimum cycle length is $k=3$. Therefore, the smallest possible number of vertices is $n = 2k = 2 \cdot 3 = 6$.
