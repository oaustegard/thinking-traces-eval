## Reasoning trace

The problem has two parts:

### Part 1: 1-loop contribution to $\log(Z/Z_0)$

The action is given as $S(x) = \tfrac{x^2}{2} - g(x^2 + \tfrac{x^3}{6})$.
We can rewrite this as $S(x) = \tfrac{1-g}{2}x^2 - \tfrac{g}{6}x^3$.
This is a zero-dimensional quantum field theory. The partition function is $Z = \int e^{-S(x)/\hbar} dx$. The free partition function $Z_0$ corresponds to $g=0$, so $S_0(x) = x^2/2$.
The 1-loop contribution to $\log Z$ for a zero-dimensional QFT is given by $-\tfrac{1}{2}\log(S''(x_c))$, where $x_c$ is the classical solution (saddle point) for $J=0$.

First, find $x_c$:
$S'(x) = \frac{d}{dx}\left(\frac{1-g}{2}x^2 - \frac{g}{6}x^3\right) = (1-g)x - \frac{g}{2}x^2$.
Setting $S'(x_c) = 0$:
$x_c(1-g - \frac{g}{2}x_c) = 0$.
One solution is $x_c = 0$. We evaluate $S''$ at this trivial classical solution.

Next, find $S''(x)$:
$S''(x) = \frac{d}{dx}((1-g)x - \frac{g}{2}x^2) = (1-g) - gx$.
Evaluate at $x_c=0$:
$S''(0) = 1-g$.

The 1-loop contribution to $\log Z$ is $-\tfrac{1}{2}\log(S''(0)) = -\tfrac{1}{2}\log(1-g)$.
The free partition function $Z_0$ has $S_0(x) = x^2/2$, so $S_0''(0) = 1$. Its 1-loop contribution is $-\tfrac{1}{2}\log(1) = 0$.
Therefore, the 1-loop contribution to $\log(Z/Z_0)$ is simply $-\tfrac{1}{2}\log(1-g)$.

This result can also be obtained diagrammatically. The interaction part of the action is $-S_{\text{int}}(x) = g(x^2 + x^3/6)$. This means we have two types of vertices: a 2-valent vertex with factor $g$ and a 3-valent vertex with factor $g$. The bare propagator is $1$.
A 1-loop vacuum diagram must be a cycle. For a cycle, let $V_2$ be the number of 2-valent vertices and $V_3$ be the number of 3-valent vertices. The total number of vertices $V = V_2 + V_3$. The total number of edges $E = V$ (for a single loop). The sum of valences is $2V_2 + 3V_3$. Since all edges are internal, $2E = 2V_2 + 3V_3$.
Substituting $E=V$: $2(V_2+V_3) = 2V_2 + 3V_3 \implies 2V_2 + 2V_3 = 2V_2 + 3V_3 \implies V_3 = 0$.
This means 1-loop vacuum diagrams can only consist of 2-valent vertices. A cycle of $k$ 2-valent vertices (a $k$-gon) has a value of $g^k$ and a symmetry factor of $2k$ (for $k \ge 1$).
Summing these contributions: $\sum_{k=1}^\infty \frac{g^k}{2k} = -\frac{1}{2}\log(1-g)$. This matches the determinant calculation.

### Part 2: Number of labeled $n$-vertex 1-loop graphs with 1-valent and 3-valent vertices only

Let $a_n$ be the number of such graphs.
A 1-loop graph consists of a simple cycle with trees attached to its vertices.
The condition "1-valent and 3-valent vertices only" is crucial:
-   1-valent vertices must be leaves (they cannot be on the cycle).
-   Vertices on the cycle must have degree at least 2. If a cycle vertex is 3-valent, two of its edges are part of the cycle, and the third must connect to a branch (a subtree). This branch must be non-empty, otherwise the cycle vertex would be 2-valent, violating the "3-valent" condition.
-   Thus, all vertices on the cycle must be 3-valent, and each must have a non-empty branch attached.
-   The total number of vertices $n = V_1 + V_3$. The sum of degrees is $V_1 \cdot 1 + V_3 \cdot 3 = 2E$. This implies $V_1 + 3V_3$ must be even. Since $V_1 + 3V_3 = (V_1+V_3) + 2V_3 = n + 2V_3$, $n + 2V_3$ must be even. This means $n$ must be even. If $n$ is odd, $a_n=0$.

Let $k$ be the number of vertices on the cycle. Since a cycle must have at least 3 vertices, $k \ge 3$.
We choose $k$ vertices out of $n$ to form the cycle: $\binom{n}{k}$ ways.
These $k$ chosen vertices can be arranged into a labeled cycle in $\frac{(k-1)!}{2}$ ways (for undirected cycles).
The remaining $n-k$ vertices must be distributed among the $k$ branches. Each branch must be non-empty. This means $n-k \ge k$, or $n \ge 2k$.

Let $B(z)$ be the exponential generating function (EGF) for labeled trees rooted at an edge, where all vertices (except the implicit parent of the root edge) are 1-valent or 3-valent.
If the root vertex of such a branch is 1-valent, it's a single vertex (a leaf), contributing $z$.
If the root vertex of such a branch is 3-valent, one edge connects to the cycle, and the remaining two edges connect to two sub-branches. Since the children sub-branches are unordered, this contributes $z \frac{B(z)^2}{2}$.
So, $B(z)$ satisfies the equation $B(z) = z + \frac{z B(z)^2}{2}$.
The $z$ factor in each term counts the vertex that is the root of the branch.

The $k$ branches are distinct (attached to distinct cycle vertices). The number of ways to distribute $n-k$ labeled vertices among $k$ ordered branches is given by $(n-k)! \cdot [z^{n-k}] B(z)^k$.
We use Lagrange Inversion Theorem to find $[z^{n-k}] B(z)^k$.
From $z = B/(1+B^2/2)$, let $f(w) = w/(1+w^2/2)$. Then $B(z)$ is the inverse function of $f(w)$.
For $\phi(w) = w^k$, the coefficient $[z^j] \phi(B(z))$ is $\frac{1}{j} [w^{j-1}] \phi'(w) \left(\frac{w}{f(w)}\right)^j = \frac{1}{j} [w^{j-1}] \phi'(w) (1+w^2/2)^j$.
Here, $j=n-k$ and $\phi(w)=w^k$, so $\phi'(w)=kw^{k-1}$.
$[z^{n-k}] B(z)^k = \frac{1}{n-k} [w^{n-k-1}] (k w^{k-1} (1+w^2/2)^{n-k})$.
$= \frac{k}{n-k} [w^{n-2k}] (1+w^2/2)^{n-k}$.
Expand $(1+w^2/2)^{n-k} = \sum_{p=0}^{n-k} \binom{n-k}{p} (w^2/2)^p$.
We need the coefficient of $w^{n-2k}$, so $2p = n-2k \implies p = (n-2k)/2$.
This requires $n-2k$ to be a non-negative even integer. As $n$ must be even, $n-2k$ is always even. So we only need $n \ge 2k$.
The coefficient is $\frac{k}{n-k} \binom{n-k}{(n-2k)/2} (1/2)^{(n-2k)/2}$.

Combining these factors, the total number of graphs $a_n$ is:
$a_n = \sum_{k=3}^{\lfloor n/2 \rfloor} \binom{n}{k} \frac{(k-1)!}{2} \cdot (n-k)! \cdot \left(\frac{k}{n-k} \binom{n-k}{(n-2k)/2} (1/2)^{(n-2k)/2}\right)$.
This simplifies to:
$a_n = \sum_{k=3}^{\lfloor n/2 \rfloor} \frac{n!}{k!(n-k)!} \frac{(k-1)!}{2} (n-k)! \frac{k}{n-k} \binom{n-k}{(n-2k)/2} (1/2)^{(n-2k)/2}$.
$a_n = \sum_{k=3}^{\lfloor n/2 \rfloor} \frac{n!}{2(n-k)} \binom{n-k}{(n-2k)/2} (1/2)^{(n-2k)/2}$.
This sum is taken over $k$ from $3$ up to $\lfloor n/2 \rfloor$. As established, $a_n=0$ if $n$ is odd.

Let's check for small even $n$:
-   For $n=4$: The sum runs from $k=3$ to $\lfloor 4/2 \rfloor=2$. Since $k \ge 3$, the sum is empty. So $a_4=0$. This is correct: $n=4, k=3$ means $n-k=1$. We need 3 non-empty branches to attach to the cycle, but only 1 vertex is available. So no such graph exists.
-   For $n=6$: The sum runs for $k=3$. Here $n=2k$.
    $a_6 = \frac{6!}{2(6-3)} \binom{6-3}{(6-6)/2} (1/2)^{(6-6)/2} = \frac{6!}{2 \cdot 3} \binom{3}{0} (1/2)^0 = \frac{720}{6} \cdot 1 \cdot 1 = 120$.
    This corresponds to a triangle of 3-valent vertices, each having a single 1-valent leaf attached. Number of ways to choose 3 vertices for the cycle is $\binom{6}{3}=20$. Number of ways to form a labeled cycle is $1$. Number of ways to attach the remaining 3 labeled vertices as leaves is $3!=6$. So $20 \cdot 1 \cdot 6 = 120$. This matches.

### Part 1: 1-loop contribution to $\log(Z/Z_0)$

For the action $S(x) = \tfrac{x^2}{2} - g(x^2 + \tfrac{x^3}{6})$, the 1-loop contribution to $\log(Z/Z_0)$ is:

$$\boxed{-\frac{1}{2}\log(1-g)}$$

This is derived from the formula $\log Z_{1\text{-loop}} = -\tfrac{1}{2}\log(S''(x_c))$, where $x_c=0$ is the classical solution and $S''(0) = 1-g$.

### Part 2: Number of labeled $n$-vertex 1-loop graphs with 1-valent and 3-valent vertices only

Let $a_n$ be the number of such graphs.
The graphs must have an even number of vertices ($a_n=0$ if $n$ is odd).
For $n$ even, let $n=2m$. The number of graphs is given by:

$$\boxed{a_n = \sum_{k=3}^{m} \frac{n!}{2(n-k)} \binom{n-k}{(n-2k)/2} \left(\frac{1}{2}\right)^{(n-2k)/2}}$$

where the sum is over $k$ from $3$ to $m=\lfloor n/2 \rfloor$. Each term in the sum corresponds to a graph with a $k$-cycle, where each of the $k$ cycle vertices is 3-valent and has a non-empty branch attached. All other vertices in the branches are either 1-valent or 3-valent.
Examples:
-   $a_n=0$ for odd $n$.
-   $a_4=0$.
-   $a_6 = 120$.
-   $a_8 = 15120$.
