## Reasoning trace

### Part 1: 1-loop contribution to $\log(Z/Z_0)$

The given action is $S(x) = \frac{x^2}{2} - g(x^2 + \frac{x^3}{6})$.
We can rewrite this as $S(x) = \frac{(1-2g)x^2}{2} - \frac{g x^3}{6}$.
The partition function is $Z = \int e^{-S(x)/\hbar} dx$.
The free partition function $Z_0$ is for $g=0$, so $S_0(x) = \frac{x^2}{2}$.
$Z_0 = \int e^{-x^2/(2\hbar)} dx = \sqrt{2\pi\hbar}$.

We want to compute the 1-loop contribution to $\log(Z/Z_0)$. In zero-dimensional QFT, this is given by $W_1(0)/\hbar$, where $W_1(J)$ is the 1-loop contribution to the connected generating functional.
A standard result for the 1-loop effective action (which at $J=0$ corresponds to $-W_1(0)$) is $S_1(x_c) = \frac{1}{2}\log S''(x_c)$, where $x_c$ is the classical solution (saddle point) for $J=0$.
The classical equation of motion $S'(x) = 0$ determines $x_c$.
$S'(x) = (1-2g)x - \frac{g x^2}{2}$.
Setting $S'(x_c) = 0$: $x_c((1-2g) - \frac{g x_c}{2}) = 0$.
Two solutions: $x_c = 0$ or $x_c = \frac{2(1-2g)}{g}$.
For the integral to converge, we need $S''(x_c) > 0$.
$S''(x) = (1-2g) - gx$.
At $x_c=0$, $S''(0) = 1-2g$. If $1-2g > 0$, then $x_c=0$ is the relevant saddle point.
At $x_c = \frac{2(1-2g)}{g}$, $S''(x_c) = (1-2g) - g \frac{2(1-2g)}{g} = (1-2g) - 2(1-2g) = -(1-2g)$. This would be the saddle point if $1-2g < 0$.
Assuming $g < 1/2$, the relevant saddle point is $x_c=0$.
Then $S_1(0) = \frac{1}{2}\log S''(0) = \frac{1}{2}\log(1-2g)$.
Since $W(0) = \hbar \log Z$ and $S_{\text{eff}}(0) = -W(0)$, we have $S_{\text{eff}}(0) = -(\hbar \log Z_0 + \hbar S_1(0) + O(\hbar^2))$.
So $\log(Z/Z_0) = -S_1(0) = -\frac{1}{2}\log(1-2g)$.

Alternatively, by Feynman diagrams:
The interaction part of the action is $S_{\text{int}}(x) = -g x^2 - g x^3/6$.
The interaction terms in the exponent $e^{-S_{\text{int}}(x)/\hbar}$ are $g x^2/\hbar$ and $g x^3/(6\hbar)$.
The bare propagator is $\langle x x \rangle_0 = \hbar$.
A 1-loop vacuum graph (no external legs) with $V_2$ 2-valent vertices and $V_3$ 3-valent vertices must satisfy $V_3=0$. This is because the sum of degrees $2V_2 + 3V_3$ must be $2E$, and for a 1-loop graph $E=V_2+V_3$. So $2V_2+3V_3 = 2(V_2+V_3) \implies V_3=0$.
Therefore, only 2-valent vertices contribute to 1-loop vacuum graphs.
Each 2-valent vertex carries a factor $g/\hbar$. Each propagator carries a factor $\hbar$.
A $k$-cycle of 2-valent vertices has $k$ vertices and $k$ propagators. Its value is $(g/\hbar)^k \hbar^k = g^k$.
The symmetry factor for a $k$-cycle is $1/(2k)$.
So the sum of all 1-loop vacuum graphs is $\sum_{k=1}^\infty \frac{g^k}{2k} = -\frac{1}{2}\log(1-g)$.
This is the contribution to $\log(Z/Z_0)$.
**Correction**: The action is $S(x) = x^2/2 - g(x^2 + x^3/6) = (1-g)x^2/2 - gx^3/6$.
So $S''(0) = 1-g$.
Thus $\log(Z/Z_0) = -\frac{1}{2}\log(1-g)$.
The calculation via Feynman diagrams for 2-valent vertices from $S_{\text{int}}(x) = -gx^2$ is $\sum_{k=1}^\infty \frac{g^k}{2k} = -\frac{1}{2}\log(1-g)$. This matches.

### Part 2: Number of labeled $n$-vertex 1-loop graphs with 1-valent and 3-valent vertices only

Let $a_n$ be the number of such graphs on $n$ labeled vertices.
For a connected graph, the number of loops is $L = E-V+1$. For a 1-loop graph, $L=1$, so $E=V$.
Here $V=n$. So the number of edges is $E=n$.
Let $V_1$ be the number of 1-valent vertices and $V_3$ be the number of 3-valent vertices.
Total vertices $n = V_1 + V_3$.
The sum of degrees is $1 \cdot V_1 + 3 \cdot V_3$. This must equal $2E = 2n$.
So $V_1 + 3V_3 = 2n$.
Substitute $V_1 = n - V_3$: $(n - V_3) + 3V_3 = 2n \implies n + 2V_3 = 2n \implies 2V_3 = n$.
This implies that $n$ must be an even number. Let $n=2k$ for some integer $k \ge 1$.
Then $V_3 = k$ and $V_1 = n - V_3 = 2k - k = k$.
So any such graph must have $k$ 1-valent vertices and $k$ 3-valent vertices.

The structure of a connected graph with $E=V$ edges (a 1-loop graph) is a single cycle with trees attached to its vertices.
Since 1-valent vertices must be leaves (degree 1), they cannot be part of the cycle.
This means all $k$ 1-valent vertices are leaves of trees attached to the cycle.
The cycle must therefore be formed entirely by 3-valent vertices. Let the cycle have $m$ vertices. These $m$ vertices are all 3-valent.
Each of these $m$ cycle vertices uses two edges for the cycle. It has one remaining edge, which must connect to a tree.
The total number of 3-valent vertices is $k$. So $m \le k$.
If a 3-valent vertex is in a tree (not on the cycle), it uses one edge to its parent and two to its children.
However, to satisfy the valence constraint, the simplest and only possible structure is that the $k$ 3-valent vertices form the cycle, and each of them has its remaining single edge connected to one of the $k$ 1-valent vertices.
This implies $m=k$.
So the graph consists of a cycle of $k$ 3-valent vertices, and each 3-valent vertex is connected to exactly one 1-valent vertex.

Let's count these labeled graphs for $n=2k$:
1.  Choose $k$ vertices out of $n$ to be 3-valent: $\binom{n}{k}$ ways. The remaining $k$ vertices are 1-valent.
2.  Form a cycle on the $k$ chosen 3-valent vertices. The number of ways depends on $k$:
    *   For $k=1$ ($n=2$): A single 3-valent vertex forms a self-loop. 1 way.
    *   For $k=2$ ($n=4$): Two 3-valent vertices form a double edge. 1 way.
    *   For $k \ge 3$: The number of distinct Hamiltonian cycles on $k$ labeled vertices is $\frac{(k-1)!}{2}$.
3.  Connect each of the $k$ 1-valent vertices to a unique 3-valent vertex. There are $k!$ ways to assign these pairings.

Let's compute $a_n$ for specific $n$:
*   **$n=2$ ($k=1$):**
    $V_1=1, V_3=1$.
    $\binom{2}{1}$ ways to choose the 3-valent vertex. (e.g., $v_1$ is 3-valent, $v_2$ is 1-valent).
    1 way to form a cycle on $v_1$ (self-loop).
    $1!$ way to connect $v_2$ to $v_1$.
    $a_2 = \binom{2}{1} \cdot 1 \cdot 1! = 2 \cdot 1 \cdot 1 = 2$.

*   **$n=4$ ($k=2$):**
    $V_1=2, V_3=2$.
    $\binom{4}{2}$ ways to choose the 3-valent vertices.
    1 way to form a cycle on these two vertices (a double edge between them).
    $2!$ ways to connect the two 1-valent vertices to the two 3-valent vertices.
    $a_4 = \binom{4}{2} \cdot 1 \cdot 2! = 6 \cdot 1 \cdot 2 = 12$.

*   **$n \ge 6$ (general $k \ge 3$):**
    $V_1=k, V_3=k$.
    $\binom{n}{k}$ ways to choose the 3-valent vertices.
    $\frac{(k-1)!}{2}$ ways to form a cycle on these $k$ vertices.
    $k!$ ways to connect the $k$ 1-valent vertices to the $k$ 3-valent vertices.
    $a_n = \binom{n}{k} \cdot \frac{(k-1)!}{2} \cdot k! = \frac{n!}{k!(n-k)!} \cdot \frac{(k-1)!}{2} \cdot k!$.
    Since $n=2k$, we have $n-k=k$.
    $a_n = \frac{n!}{k!k!} \cdot \frac{(k-1)!}{2} \cdot k! = \frac{n!}{k!} \cdot \frac{(k-1)!}{2} = \frac{n!}{k(k-1)!} \cdot \frac{(k-1)!}{2} = \frac{n!}{2k}$.
    Since $k=n/2$, $a_n = \frac{n!}{2(n/2)} = \frac{n!}{n} = (n-1)!$.

The "use this" connection: The exponential generating function (EGF) for $a_n$ is $F(z) = \sum_{n=0}^\infty a_n \frac{z^n}{n!}$.
Since $a_n=0$ for odd $n$, $F(z) = \sum_{k=1}^\infty a_{2k} \frac{z^{2k}}{(2k)!}$.
Using the computed values for $a_n$:
$F(z) = a_2 \frac{z^2}{2!} + a_4 \frac{z^4}{4!} + \sum_{k=3}^\infty (2k-1)! \frac{z^{2k}}{(2k)!}$.
$F(z) = 2 \frac{z^2}{2} + 12 \frac{z^4}{24} + \sum_{k=3}^\infty \frac{z^{2k}}{2k}$.
$F(z) = z^2 + \frac{z^4}{2} + \left( \sum_{k=1}^\infty \frac{z^{2k}}{2k} - \frac{z^2}{2} - \frac{z^4}{4} \right)$.
$F(z) = \sum_{k=1}^\infty \frac{(z^2)^k}{2k} + \frac{z^2}{2} + \frac{z^4}{4}$.
The sum $\sum_{k=1}^\infty \frac{x^k}{k} = -\log(1-x)$. So $\sum_{k=1}^\infty \frac{(z^2)^k}{2k} = -\frac{1}{2}\log(1-z^2)$.
Thus, $F(z) = -\frac{1}{2}\log(1-z^2) + \frac{z^2}{2} + \frac{z^4}{4}$.
This EGF has a form similar to the 1-loop contribution to $\log(Z/Z_0)$ from Part 1, $-\frac{1}{2}\log(1-g)$, but with $g$ replaced by $z^2$ and with additional polynomial correction terms.

## Final answer

### Part 1: 1-loop contribution to $\log(Z/Z_0)$

The 1-loop contribution to $\log(Z/Z_0)$ is given by $-\frac{1}{2}\log S''(x_c)$, where $x_c$ is the classical minimum of $S(x)$.
Given $S(x) = \frac{x^2}{2} - g(x^2 + \frac{x^3}{6}) = \frac{(1-g)x^2}{2} - \frac{g x^3}{6}$.
The first derivative is $S'(x) = (1-g)x - \frac{g x^2}{2}$.
Setting $S'(x_c)=0$ yields $x_c=0$ (assuming $1-g>0$).
The second derivative is $S''(x) = (1-g) - gx$.
At $x_c=0$, $S''(0) = 1-g$.
Thus, the 1-loop contribution to $\log(Z/Z_0)$ is:

$$\boxed{-\frac{1}{2}\log(1-g)}$$

### Part 2: Number of labeled $n$-vertex 1-loop graphs with 1-valent and 3-valent vertices only

Let $a_n$ be the number of such graphs on $n$ labeled vertices.
A 1-loop graph has $E=V$ edges. If all vertices are 1-valent or 3-valent, then $n$ must be even, say $n=2k$.
Such a graph must consist of $k$ 1-valent vertices and $k$ 3-valent vertices. The only possible topology is a cycle formed by the $k$ 3-valent vertices, with each 3-valent vertex connected to a unique 1-valent vertex.

The number of such labeled graphs $a_n$ is:
*   **For odd $n$:** $a_n = 0$.
*   **For $n=2$ ($k=1$):** $a_2 = 2$.
    (There are two graphs: $v_1$ (3-valent, with self-loop) connected to $v_2$ (1-valent); and $v_2$ (3-valent, with self-loop) connected to $v_1$ (1-valent).)
*   **For $n=4$ ($k=2$):** $a_4 = 12$.
    (Choose 2 vertices to be 3-valent ($\binom{4}{2}=6$ ways). They form a double edge. The remaining 2 1-valent vertices are connected to them in $2! = 2$ ways. Total $6 \times 1 \times 2 = 12$.)
*   **For $n \ge 6$ (and $n$ is even, $k=n/2 \ge 3$):** $a_n = (n-1)!$.
    (Choose $k$ vertices to be 3-valent ($\binom{n}{k}$ ways). Form a cycle on these $k$ vertices ($\frac{(k-1)!}{2}$ ways). Connect the remaining $k$ 1-valent vertices to the $k$ 3-valent vertices ($k!$ ways). This simplifies to $\frac{n!}{k!(n-k)!} \frac{(k-1)!}{2} k! = \frac{n!}{2k} = (n-1)!$.)

The full sequence of coefficients $a_n$ is:
$$\boxed{a_n = \begin{cases} 0 & \text{if } n \text{ is odd} \\ 2 & \text{if } n=2 \\ 12 & \text{if } n=4 \\ (n-1)! & \text{if } n \ge 6 \text{ and } n \text{ is even} \end{cases}}$$

The exponential generating function for these numbers is $F(z) = \sum_{n=0}^\infty a_n \frac{z^n}{n!} = -\frac{1}{2}\log(1-z^2) + \frac{z^2}{2} + \frac{z^4}{4}$.
