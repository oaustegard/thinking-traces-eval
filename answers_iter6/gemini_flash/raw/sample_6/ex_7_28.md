The problem asks for the leading term of the 1-point function and the connected 2-point function up to order $g^2$ for a cubic anharmonic oscillator.

**Part (b): Solution for the Cubic Anharmonic Oscillator**

**Setup:**
The Euclidean action is $S[q] = \int dt \bigl(\tfrac12 \dot q^2 + \tfrac{m^2}{2} q^2 - \tfrac{g}{6} q^3\bigr)$.
The interaction term is $S_{\text{int}}[q] = -\int dt \tfrac{g}{6} q^3$.
In the path integral formulation, we expand $e^{-S_{\text{int}}} = 1 - S_{\text{int}} + \frac{1}{2} S_{\text{int}}^2 - \dots$.
Each interaction vertex comes from $S_{\text{int}}$. If $S_{\text{int}} = \int dt \frac{\lambda}{3!} q^3$, then the vertex factor is $-\lambda$. Here, $\lambda = -g$, so the vertex factor is $g$.

**Feynman Rules:**
1.  **Free Propagator:** $G_0(t_1, t_2) = \langle q(t_1) q(t_2)\rangle_0 = \frac{1}{2m} e^{-m|t_1 - t_2|}$. Its Fourier transform is $G_0(\omega) = \frac{1}{\omega^2 + m^2}$.
2.  **Vertex:** A 3-valent vertex at time $t_v$ carries a factor of $g$. We integrate over $t_v$. The $1/3!$ in the action cancels the $3!$ ways to connect legs to the vertex.

---

**(i) Calculate the leading term of the 1-point function $G_1(t) = \langle q(t)\rangle$ with respect to $g$.**

The 1-point function is the sum of all connected Feynman diagrams with one external leg at time $t$.
*   **Order $g^0$:** No vertices. No diagrams possible with one external leg. So $G_1^{(0)}(t) = 0$.
*   **Order $g^1$:** One $q^3$ vertex.
    *   The vertex has 3 half-edges. One connects to the external leg at $t$. The remaining two must contract with each other to form a loop.
    *   Diagram: A vertex at $t_v$, with one leg to $t$, and the other two forming a self-loop.
    *   Vertex factor: $g$.
    *   Propagator: $G_0(t_v, t_v)$ for the self-loop.
    *   Symmetry factor: The self-loop has a $\mathbb{Z}_2$ symmetry (flipping the loop), so $S=2$.
    *   Contribution: $\frac{1}{S} \cdot g \int dt_v \delta(t-t_v) G_0(t_v, t_v)$.
    *   $G_1^{(1)}(t) = \frac{g}{2} G_0(t,t)$.
    *   Using $G_0(t,t) = \frac{1}{2m}$:
    *   $G_1^{(1)}(t) = \frac{g}{2} \cdot \frac{1}{2m} = \frac{g}{4m}$.

This is constant in time, as expected for a time-translation invariant system.
The leading term of the 1-point function is:
$$\boxed{\;G_1(t) = \frac{g}{4m} + O(g^2).\;}$$

---

**(ii) Calculate the connected 2-point function $G_2^c(t_1, t_2) = \langle q(t_1) q(t_2)\rangle_c$ modulo $g^3$.**

The connected 2-point function is the sum of all connected Feynman diagrams with two external legs at $t_1$ and $t_2$. Let $\Delta t = |t_1 - t_2|$.

*   **Order $g^0$:** No vertices. This is the free propagator.
    *   $G_2^{c,(0)}(t_1, t_2) = G_0(t_1, t_2) = \frac{1}{2m} e^{-m\Delta t}$.

*   **Order $g^1$:** One vertex. A $q^3$ vertex has 3 half-edges. If two are external (to $t_1, t_2$), only one internal half-edge remains. This cannot form a loop. Thus, there are no 1-vertex diagrams for $G_2^c$.
    *   $G_2^{c,(1)}(t_1, t_2) = 0$.

*   **Order $g^2$:** Two vertices. Each vertex contributes $g$, so total $g^2$. Each vertex has 3 legs, so 6 half-edges total. Two are external (to $t_1, t_2$), leaving 4 internal half-edges to connect.

    There are two types of connected diagrams at $O(g^2)$:

    1.  **Sunset Diagram (1PI contribution):** This diagram involves two vertices ($V_a, V_b$) connected by three internal propagators. It is 1-particle irreducible (1PI) because cutting any single internal line does not disconnect the diagram.
        *   Diagram structure: $t_1 - V_a - (G_0(V_a, V_b))^3 - V_b - t_2$.
        *   Vertex factors: $g^2$.
        *   Propagators: $[G_0(t_a, t_b)]^3$.
        *   Symmetry factor: The three internal propagators are identical, so $S=3! = 6$.
        *   This diagram contributes to the 1PI self-energy $\Sigma^{(2)}(t_1, t_2)$.
        *   $\Sigma^{(2)}(t_1, t_2) = \frac{1}{S} g^2 \int dt_a dt_b \delta(t_1-t_a) \delta(t_2-t_b) [G_0(t_a, t_b)]^3 = \frac{g^2}{6} [G_0(t_1, t_2)]^3$.
        *   Substituting $G_0(t_1, t_2) = \frac{1}{2m} e^{-m\Delta t}$:
        *   $\Sigma^{(2)}(t_1, t_2) = \frac{g^2}{6} \left(\frac{1}{2m}\right)^3 e^{-3m\Delta t} = \frac{g^2}{48m^3} e^{-3m\Delta t}$.
        *   The contribution to $G_2^c$ from this 1PI diagram is given by $G_0 \star \Sigma^{(2)} \star G_0$. In Fourier space, this is $G_0(\omega) \Sigma^{(2)}(\omega) G_0(\omega)$.
        *   $\Sigma^{(2)}(\omega) = \mathcal{F}[\Sigma^{(2)}(t)] = \frac{g^2}{48m^3} \mathcal{F}[e^{-3m|t|}] = \frac{g^2}{48m^3} \frac{2(3m)}{\omega^2+(3m)^2} = \frac{g^2}{8m^2} \frac{1}{\omega^2+9m^2}$.
        *   So, $G_{2, \text{sunset}}^{c,(2)}(\omega) = \frac{1}{\omega^2+m^2} \cdot \frac{g^2}{8m^2} \frac{1}{\omega^2+9m^2} \cdot \frac{1}{\omega^2+m^2} = \frac{g^2}{8m^2} \frac{1}{(\omega^2+m^2)^2 (\omega^2+9m^2)}$.
        *   Using partial fraction decomposition and inverse Fourier transforms:
            *   $\mathcal{F}^{-1}\left[\frac{1}{\omega^2+a^2}\right] = \frac{1}{2a} e^{-a|t|}$.
            *   $\mathcal{F}^{-1}\left[\frac{1}{(\omega^2+a^2)^2}\right] = \frac{1}{4a^3} (1+a|t|) e^{-a|t|}$.
            *   The decomposition is $\frac{1}{(X+m^2)^2 (X+9m^2)} = -\frac{1}{64m^4(X+m^2)} + \frac{1}{8m^2(X+m^2)^2} + \frac{1}{64m^4(X+9m^2)}$.
            *   $G_{2, \text{sunset}}^{c,(2)}(\Delta t) = \frac{g^2}{8m^2} \left[ -\frac{1}{64m^4} \frac{1}{2m} e^{-m\Delta t} + \frac{1}{8m^2} \frac{1}{4m^3}(1+m\Delta t) e^{-m\Delta t} + \frac{1}{64m^4} \frac{1}{2(3m)} e^{-3m\Delta t} \right]$.
            *   $G_{2, \text{sunset}}^{c,(2)}(\Delta t) = \frac{g^2}{8m^2} \left[ -\frac{1}{128m^5} e^{-m\Delta t} + \frac{1}{32m^5}(1+m\Delta t) e^{-m\Delta t} + \frac{1}{384m^5} e^{-3m\Delta t} \right]$.
            *   $G_{2, \text{sunset}}^{c,(2)}(\Delta t) = \frac{g^2}{256m^7} \left[ -\frac{1}{2} e^{-m\Delta t} + (1+m\Delta t) e^{-m\Delta t} + \frac{1}{12} e^{-3m\Delta t} \right]$.
            *   $G_{2, \text{sunset}}^{c,(2)}(\Delta t) = \frac{g^2}{256m^7} \left[ \left(\frac{1}{2} + m\Delta t\right) e^{-m\Delta t} + \frac{1}{12} e^{-3m\Delta t} \right]$.

    2.  **Tadpole-on-a-Line Diagram:** This diagram consists of two vertices ($V_a, V_b$) connected by a single internal propagator, and each vertex has a self-loop. This diagram is *not* 1PI (it can be disconnected by cutting the central propagator).
        *   Diagram structure: $t_1 - G_0(t_1, V_a) - G_0(V_a, V_a) - G_0(V_a, V_b) - G_0(V_b, V_b) - G_0(V_b, t_2)$.
        *   Vertex factors: $g^2$.
        *   Symmetry factors: Each self-loop has $S=2$. Since there are two self-loops, the total symmetry factor is $S=2 \times 2 = 4$.
        *   Contribution: $\frac{1}{4} g^2 \int dt_a dt_b G_0(t_1,t_a) G_0(t_a,t_a) G_0(t_a,t_b) G_0(t_b,t_b) G_0(t_b,t_2)$.
        *   Since $G_0(t_a,t_a) = G_0(t_b,t_b) = \frac{1}{2m}$, this simplifies to:
        *   $\frac{g^2}{4} \left(\frac{1}{2m}\right)^2 \int dt_a dt_b G_0(t_1,t_a) G_0(t_a,t_b) G_0(t_b,t_2)$.
        *   $= \frac{g^2}{16m^2} (G_0 \star G_0 \star G_0)(t_1, t_2)$.
        *   The Fourier transform of $(G_0 \star G_0 \star G_0)(t)$ is $(G_0(\omega))^3 = \frac{1}{(\omega^2+m^2)^3}$.
        *   The inverse Fourier transform is $\mathcal{F}^{-1}\left[\frac{1}{(\omega^2+m^2)^3}\right] = \frac{1}{128m^5} (1 + 6m\Delta t + 12m^2(\Delta t)^2)e^{-m\Delta t}$.
        *   $G_{2, \text{tadpole}}^{c,(2)}(\Delta t) = \frac{g^2}{16m^2} \frac{1}{128m^5} (1 + 6m\Delta t + 12m^2(\Delta t)^2)e^{-m\Delta t}$.
        *   $G_{2, \text{tadpole}}^{c,(2)}(\Delta t) = \frac{g^2}{2048m^7} (1 + 6m\Delta t + 12m^2(\Delta t)^2)e^{-m\Delta t}$.

*   **Total Connected 2-point function up to $O(g^2)$:**
    Summing the $O(g^0)$ and two $O(g^2)$ contributions:
    $G_2^c(t_1, t_2) = G_0(t_1, t_2) + G_{2, \text{sunset}}^{c,(2)}(t_1, t_2) + G_{2, \text{tadpole}}^{c,(2)}(t_1, t_2) + O(g^3)$.
    $G_2^c(t_1, t_2) = \frac{1}{2m} e^{-m\Delta t} + \frac{g^2}{256m^7} \left[ \left(\frac{1}{2} + m\Delta t\right) e^{-m\Delta t} + \frac{1}{12} e^{-3m\Delta t} \right] + \frac{g^2}{2048m^7} (1 + 6m\Delta t + 12m^2(\Delta t)^2)e^{-m\Delta t} + O(g^3)$.
    To combine terms, use common denominator $2048m^7$:
    $G_2^c(t_1, t_2) = \frac{1}{2m} e^{-m\Delta t} + \frac{g^2}{2048m^7} \left[ 8\left(\frac{1}{2} + m\Delta t\right) e^{-m\Delta t} + \frac{8}{12} e^{-3m\Delta t} + (1 + 6m\Delta t + 12m^2(\Delta t)^2)e^{-m\Delta t} \right] + O(g^3)$.
    $G_2^c(t_1, t_2) = \frac{1}{2m} e^{-m\Delta t} + \frac{g^2}{2048m^7} \left[ (4 + 8m\Delta t + 1 + 6m\Delta t + 12m^2(\Delta t)^2)e^{-m\Delta t} + \frac{2}{3} e^{-3m\Delta t} \right] + O(g^3)$.
    $G_2^c(t_1, t_2) = \frac{1}{2m} e^{-m\Delta t} + \frac{g^2}{2048m^7} \left[ (5 + 14m\Delta t + 12m^2(\Delta t)^2)e^{-m\Delta t} + \frac{2}{3} e^{-3m\Delta t} \right] + O(g^3)$.

Where $\Delta t = |t_1 - t_2|$.

The final answer is:
$$\boxed{
\begin{align*}
G_1(t) &= \frac{g}{4m} + O(g^2) \\
G_2^c(t_1, t_2) &= \frac{1}{2m} e^{-m|t_1-t_2|} + \frac{g^2}{2048m^7} \left[ \left(5 + 14m|t_1-t_2| + 12m^2(t_1-t_2)^2\right) e^{-m|t_1-t_2|} + \frac{2}{3} e^{-3m|t_1-t_2|} \right] + O(g^3).
\end{align*}
}$$
