The problem asks for two quantities related to a Euclidean cubic anharmonic oscillator, defined by the potential $U(q) := m^2 q^2 / 2 - g q^3 / 6$. We will use Euclidean path integral formalism and perturbation theory in $g$.

The Euclidean action is $S_E[q] = \int dt \left[ \frac{1}{2} (\frac{dq}{dt})^2 + U(q) \right]$.
Substituting $U(q)$:
$S_E[q] = \int dt \left[ \frac{1}{2} (\frac{dq}{dt})^2 + \frac{m^2}{2} q^2 - \frac{g}{6} q^3 \right]$.
We split the action into a free part $S_0[q]$ and an interaction part $S_{int}[q]$:
$S_0[q] = \int dt \left[ \frac{1}{2} (\frac{dq}{dt})^2 + \frac{m^2}{2} q^2 \right]$
$S_{int}[q] = - \int dt \frac{g}{6} q^3(t)$.

The expectation value of an observable $O[q]$ is given by:
$<O[q]> = \frac{1}{Z} \int Dq O[q] e^{-S_E[q]}$
where $Z = \int Dq e^{-S_E[q]}$ is the partition function.
We expand $e^{-S_E[q]} = e^{-S_0[q]} e^{-S_{int}[q]} = e^{-S_0[q]} \left( 1 - S_{int}[q] + \frac{1}{2!} (S_{int}[q])^2 - \dots \right)$.
Substituting $S_{int}[q]$:
$e^{-S_E[q]} = e^{-S_0[q]} \left( 1 + \frac{g}{6} \int dt' q^3(t') + \frac{1}{2} \left(\frac{g}{6}\right)^2 \left(\int dt' q^3(t')\right)^2 + O(g^3) \right)$.
$= e^{-S_0[q]} \left( 1 + \frac{g}{6} \int dt' q^3(t') + \frac{g^2}{72} \int dt' dt'' q^3(t') q^3(t'') + O(g^3) \right)$.

The free propagator $G_0(t_1, t_2) = <q(t_1) q(t_2)>_0$ satisfies the equation $(-\partial_t^2 + m^2) G_0(t, t') = \delta(t-t')$. For an infinite time interval, its solution is $G_0(t_1, t_2) = \frac{1}{2m} e^{-m|t_1-t_2|}$.
In the free theory, all odd correlation functions are zero, i.e., $<q(t_1) \dots q(t_n)>_0 = 0$ if $n$ is odd.
Also, $G_0(t,t) = \frac{1}{2m}$.

---

**(i) Calculate the leading term of the 1-point function G_1(t) := <q(t)> with respect to g.**

$G_1(t) = <q(t)> = \frac{\int Dq q(t) e^{-S_E[q]}}{\int Dq e^{-S_E[q]}}$.
Expanding the numerator and denominator:
Numerator: $\int Dq q(t) e^{-S_0[q]} \left( 1 + \frac{g}{6} \int dt' q^3(t') + O(g^2) \right)$
$= Z_0 \left( <q(t)>_0 + \frac{g}{6} \int dt' <q(t) q^3(t')>_0 + O(g^2) \right)$.
Denominator: $Z = Z_0 \left( 1 + \frac{g}{6} \int dt' <q^3(t')>_0 + O(g^2) \right)$.

Since $n=1$ and $n=3$ are odd, $<q(t)>_0 = 0$ and $<q^3(t')>_0 = 0$.
So, $G_1(t) = \frac{Z_0 \left( \frac{g}{6} \int dt' <q(t) q^3(t')>_0 + O(g^2) \right)}{Z_0 \left( 1 + O(g^2) \right)} = \frac{g}{6} \int dt' <q(t) q^3(t')>_0 + O(g^2)$.

Now, we calculate $<q(t) q^3(t')>_0$ using Wick's theorem. This is $<q(t) q(t') q(t') q(t')>_0$.
There are 3 ways to contract $q(t)$ with one of the $q(t')$ fields. The remaining two $q(t')$ fields must contract with each other.
$<q(t) q^3(t')>_0 = 3 \cdot G_0(t,t') \cdot G_0(t',t') \cdot G_0(t',t')$.
$= 3 G_0(t,t') G_0(t',t')^2$.

Substitute this back into the expression for $G_1(t)$:
$G_1(t) = \frac{g}{6} \int dt' \left[ 3 G_0(t,t') G_0(t',t')^2 \right] + O(g^2)$.
$G_1(t) = \frac{g}{2} \int dt' G_0(t,t') G_0(t',t')^2 + O(g^2)$.

Now, substitute $G_0(t_1, t_2) = \frac{1}{2m} e^{-m|t_1-t_2|}$.
$G_0(t',t') = \frac{1}{2m} e^{-m|t'-t'|} = \frac{1}{2m}$.
$G_1(t) = \frac{g}{2} \int dt' \left( \frac{1}{2m} e^{-m|t-t'|} \right) \left( \frac{1}{2m} \right)^2 + O(g^2)$.
$G_1(t) = \frac{g}{2} \frac{1}{2m} \frac{1}{4m^2} \int dt' e^{-m|t-t'|} + O(g^2)$.
$G_1(t) = \frac{g}{16m^3} \int dt' e^{-m|t-t'|} + O(g^2)$.

The integral $\int_{-\infty}^{\infty} dt' e^{-m|t-t'|}$ can be split:
$\int_{-\infty}^{t} dt' e^{-m(t-t')} + \int_{t}^{\infty} dt' e^{-m(t'-t)}$.
$= \left[ \frac{1}{m} e^{-m(t-t')} \right]_{-\infty}^{t} + \left[ -\frac{1}{m} e^{-m(t'-t)} \right]_{t}^{\infty}$.
$= \frac{1}{m} (e^0 - 0) - \frac{1}{m} (0 - e^0) = \frac{1}{m} + \frac{1}{m} = \frac{2}{m}$.

So, $G_1(t) = \frac{g}{16m^3} \frac{2}{m} + O(g^2) = \frac{g}{8m^4} + O(g^2)$.

**(ii) Calculate the connected 2-point function G_2^c(t_1, t_2) := <q(t_1) q(t_2)>_c modulo g^3.**

The connected 2-point function is defined as $G_2^c(t_1, t_2) = <q(t_1) q(t_2)> - <q(t_1)> <q(t_2)>$.
We need to calculate this up to and including order $g^2$.

From part (i), we have $<q(t)> = \frac{g}{8m^4} + O(g^2)$.
So, $<q(t_1)> <q(t_2)> = \left( \frac{g}{8m^4} + O(g^2) \right) \left( \frac{g}{8m^4} + O(g^2) \right) = \frac{g^2}{64m^8} + O(g^3)$.

Now, let's calculate $<q(t_1) q(t_2)>$ up to $O(g^2)$.
$<q(t_1) q(t_2)> = \frac{\int Dq q(t_1) q(t_2) e^{-S_E[q]}}{\int Dq e^{-S_E[q]}}$.
Numerator (N): $Z_0 \left( <q(t_1)q(t_2)>_0 + \frac{g}{6} \int dt' <q(t_1)q(t_2)q^3(t')>_0 + \frac{g^2}{72} \int dt'dt'' <q(t_1)q(t_2)q^3(t')q^3(t'')>_0 + O(g^3) \right)$.
Denominator (Z): $Z_0 \left( 1 + \frac{g}{6} \int dt' <q^3(t')>_0 + \frac{g^2}{72} \int dt'dt'' <q^3(t')q^3(t'')>_0 + O(g^3) \right)$.

Using $<q^3(t')>_0 = 0$ and $<q(t_1)q(t_2)q^3(t')>_0 = 0$ (because it's an odd number of fields, $2+3=5$):
$N = Z_0 \left( G_0(t_1,t_2) + \frac{g^2}{72} \int dt'dt'' <q(t_1)q(t_2)q^3(t')q^3(t'')>_0 + O(g^3) \right)$.
$Z = Z_0 \left( 1 + \frac{g^2}{72} \int dt'dt'' <q^3(t')q^3(t'')>_0 + O(g^3) \right)$.

So, $<q(t_1) q(t_2)> = \left[ G_0(t_1,t_2) + \frac{g^2}{72} \int dt'dt'' <q(t_1)q(t_2)q^3(t')q^3(t'')>_0 \right] \left[ 1 - \frac{g^2}{72} \int dt'dt'' <q^3(t')q^3(t'')>_0 \right] + O(g^3)$.
$<q(t_1) q(t_2)> = G_0(t_1,t_2) + \frac{g^2}{72} \int dt'dt'' \left[ <q(t_1)q(t_2)q^3(t')q^3(t'')>_0 - G_0(t_1,t_2) <q^3(t')q^3(t'')>_0 \right] + O(g^3)$.

The term in the square brackets represents the sum of all Wick contractions of $q(t_1)q(t_2)q^3(t')q^3(t'')$ that are connected to $q(t_1)$ and $q(t_2)$, excluding the disconnected term where $q(t_1)$ and $q(t_2)$ contract with each other, and the remaining fields form a vacuum bubble. This is precisely the connected part of the correlation function.
The only connected diagram at $O(g^2)$ connecting $t_1$ and $t_2$ is the "bubble" diagram (one-loop self-energy insertion):
$t_1 \xrightarrow{G_0} t' \xrightarrow{V} t'' \xrightarrow{G_0} t_2$ where $t'$ and $t''$ are connected by two internal propagators forming a loop.
The diagrammatic contribution corresponds to the term:
$\frac{g^2}{72} \int dt'dt'' \left[ \text{number of contractions} \cdot G_0(t_1,t') G_0(t',t'')^2 G_0(t'',t_2) \right]$.
The number of contractions for $q(t_1)q(t_2)q(t')^3q(t'')^3$ to form this diagram is:
1.  Choose one $q(t')$ to connect to $q(t_1)$ (3 ways).
2.  Choose one $q(t'')$ to connect to $q(t_2)$ (3 ways).
3.  Choose one of the remaining two $q(t')$ to connect to one of the remaining two $q(t'')$ (2 ways).
4.  The last remaining $q(t')$ connects to the last remaining $q(t'')$ (1 way).
Total number of contractions = $3 \times 3 \times 2 \times 1 = 18$.

So, the $O(g^2)$ term for $<q(t_1) q(t_2)>$ is:
$\frac{g^2}{72} \cdot 18 \int dt'dt'' G_0(t_1,t') G_0(t',t'')^2 G_0(t'',t_2) = \frac{g^2}{4} \int dt'dt'' G_0(t_1,t') G_0(t',t'')^2 G_0(t'',t_2)$.

Combining all terms for $G_2^c(t_1, t_2)$:
$G_2^c(t_1, t_2) = <q(t_1) q(t_2)> - <q(t_1)> <q(t_2)>$.
$G_2^c(t_1, t_2) = G_0(t_1,t_2) + \frac{g^2}{4} \int dt'dt'' G_0(t_1,t') G_0(t',t'')^2 G_0(t'',t_2) - \frac{g^2}{64m^8} + O(g^3)$.

Substituting $G_0(t_1, t_2) = \frac{1}{2m} e^{-m|t_1-t_2|}$:
$G_2^c(t_1, t_2) = \frac{1}{2m} e^{-m|t_1-t_2|} + \frac{g^2}{4} \int dt'dt'' \left(\frac{1}{2m} e^{-m|t_1-t'|}\right) \left(\frac{1}{2m} e^{-m|t'-t''|}\right)^2 \left(\frac{1}{2m} e^{-m|t''-t_2|}\right) - \frac{g^2}{64m^8} + O(g^3)$.
$G_2^c(t_1, t_2) = \frac{1}{2m} e^{-m|t_1-t_2|} + \frac{g^2}{4} \frac{1}{2m} \frac{1}{4m^2} \frac{1}{2m} \int dt'dt'' e^{-m|t_1-t'|} e^{-2m|t'-t''|} e^{-m|t''-t_2|} - \frac{g^2}{64m^8} + O(g^3)$.
$G_2^c(t_1, t_2) = \frac{1}{2m} e^{-m|t_1-t_2|} + \frac{g^2}{64m^4} \int dt'dt'' e^{-m|t_1-t'|} e^{-2m|t'-t''|} e^{-m|t''-t_2|} - \frac{g^2}{64m^8} + O(g^3)$.

Note: The integral in the $O(g^2)$ term is known to be logarithmically divergent in 1D for a $\phi^3$ theory (quantum mechanics). The problem statement does not specify a regularization scheme or a finite time interval, so we present the expression in its integral form.

The final answers are:
(i) The leading term of the 1-point function $G_1(t)$ is:
$\boxed{G_1(t) = \frac{g}{8m^4} + O(g^2)}$

(ii) The connected 2-point function $G_2^c(t_1, t_2)$ modulo $g^3$ is:
$\boxed{G_2^c(t_1, t_2) = \frac{1}{2m} e^{-m|t_1-t_2|} + \frac{g^2}{64m^4} \int dt'dt'' e^{-m|t_1-t'|} e^{-2m|t'-t''|} e^{-m|t''-t_2|} - \frac{g^2}{64m^8} + O(g^3)}$
