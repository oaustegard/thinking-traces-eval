The problem asks for two quantities related to a Euclidean cubic anharmonic oscillator, whose potential is $U(q) = ¥frac{m^2 q^2}{2} - ¥frac{g q^3}{6}$. We will use the path integral formulation and perturbation theory in the coupling constant $g$. The Euclidean action is given by $S_E = ¥int dt ¥left(¥frac{1}{2} ¥dot{q}^2 + U(q)¥right)$. We split the action into a free part $S_0$ and an interacting part $S_{int}$:
$S_0 = ¥int dt ¥left(¥frac{1}{2} ¥dot{q}^2 + ¥frac{m^2 q^2}{2}¥right)$
$S_{int} = ¥int dt ¥left(-¥frac{g q^3}{6}¥right) = -¥frac{g}{6} ¥int dt q^3(t)$

The expectation value of an observable $O[q]$ is given by:
$¥langle O[q] ¥rangle = ¥frac{¥int ¥mathcal{D}q O[q] e^{-S_E}}{¥int ¥mathcal{D}q e^{-S_E}} = ¥frac{¥langle O[q] e^{-S_{int}} ¥rangle_0}{¥langle e^{-S_{int}} ¥rangle_0}$
where $¥langle ¥dots ¥rangle_0$ denotes the expectation value with respect to the free theory ($g=0$).
We expand $e^{-S_{int}}$ in powers of $g$:
$e^{-S_{int}} = 1 - S_{int} + ¥frac{1}{2} S_{int}^2 - ¥dots = 1 + ¥frac{g}{6} ¥int dt' q^3(t') + ¥frac{g^2}{72} ¥int dt' dt'' q^3(t') q^3(t'') + O(g^3)$

The free propagator $G_0(t-t') = ¥langle q(t) q(t') ¥rangle_0$ satisfies $(-¥partial_t^2 + m^2) G_0(t-t') = ¥delta(t-t')$. Its solution is $G_0(t-t') = ¥frac{1}{2m} e^{-m|t-t'|}$.
From this, we have $G_0(0) = ¥frac{1}{2m}$ and $¥int_{-¥infty}^¥infty dt G_0(t) = ¥frac{1}{m^2}$.
For a Gaussian theory (the free theory), expectation values of an odd number of fields are zero. Thus, $¥langle q^3(t') ¥rangle_0 = 0$, which implies $¥langle S_{int} ¥rangle_0 = 0$.

The denominator is $Z = Z_0 ¥langle e^{-S_{int}} ¥rangle_0 = Z_0 ¥left(1 + ¥frac{g^2}{72} ¥int dt' dt'' ¥langle q^3(t') q^3(t'') ¥rangle_0 + O(g^4)¥right)$.
Using Wick's theorem, $¥langle q^3(t') q^3(t'') ¥rangle_0 = ¥langle q(t')q(t')q(t') q(t'')q(t'')q(t'') ¥rangle_0 = 3! G_0^3(t'-t'') = 6 G_0^3(t'-t'')$.
So, $¥langle e^{-S_{int}} ¥rangle_0 = 1 + ¥frac{g^2}{12} ¥int dt' dt'' G_0^3(t'-t'') + O(g^4)$.
And $¥frac{1}{¥langle e^{-S_{int}} ¥rangle_0} = 1 - ¥frac{g^2}{12} ¥int dt' dt'' G_0^3(t'-t'') + O(g^4)$.
