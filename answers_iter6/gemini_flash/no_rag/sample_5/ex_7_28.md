The problem asks for two quantities related to a Euclidean cubic anharmonic oscillator with potential $U(q) = \frac{m^2 q^2}{2} - \frac{g q^3}{6}$. We need to calculate the leading term of the 1-point function $G_1(t) = <q(t)>$ with respect to $g$, and the connected 2-point function $G_2^c(t_1, t_2) = <q(t_1) q(t_2)>_c$ modulo $g^3$.

The Euclidean action is given by $S[q] = \int dt \left( \frac{1}{2} (\dot{q})^2 + U(q) \right)$.
Substituting $U(q)$:
$S[q] = \int dt \left( \frac{1}{2} (\dot{q})^2 + \frac{m^2 q^2}{2} - \frac{g q^3}{6} \right)$.
We split the action into a free part $S_0[q]$ and an interaction part $S_I[q]$:
$S_0[q] = \int dt \left( \frac{1}{2} (\dot{q})^2 + \frac{m^2 q^2}{2} \right)$
$S_I[q] = \int dt \left( - \frac{g q^3}{6} \right)$
The expectation value of an operator $O(q)$ is given by the path integral:
$<O(q)> = \frac{\int Dq O(q) e^{-S[q]}}{\int Dq e^{-S[q]}} = \frac{\int Dq O(q) e^{-S_0[q]} e^{-S_I[q]}}{\int Dq e^{-S_0[q]} e^{-S_I[q]}}$.
We use perturbation theory by expanding $e^{-S_I[q]} = e^{\int dt \frac{g q^3(t)}{6}}$:
$e^{-S_I[q]} = 1 + \int dt \frac{g q^3(t)}{6} + \frac{1}{2!} \left( \int dt \frac{g q^3(t)}{6} \right)^2 + O(g^3)$.
Let $< \cdot >_0$ denote the expectation value with respect to the free action $S_0[q]$.
The free propagator for the Euclidean harmonic oscillator is $D(t_1, t_2) = <q(t_1) q(t_2)>_0 = \frac{e^{-m|t_1-t_2|}}{2m}$.
A key property of Gaussian integrals (Wick's theorem) is that $<q(t_1) \dots q(t_N)>_0 = 0$ if $N$ is odd. Also, for coinciding points, $D(t,t) = \frac{1}{2m}$.

**Part (i): Leading term of G_1(t) = <q(t)>**

$G_1(t) = <q(t)> = \frac{<q(t) (1 + \int dt' \frac{g q^3(t')}{6} + O(g^2))>_0}{<1 + \int dt' \frac{g q^3(t')}{6} + O(g^2)>_0}$.

Let's evaluate the numerator and denominator up to $O(g^2)$:
Numerator: $<q(t)>_0 + \frac{g}{6} <q(t) \int dt' q^3(t')>_0 + O(g^2)$.
Denominator: $1 + \frac{g}{6} <\int dt' q^3(t')>_0 + O(g^2)$.

Using Wick's theorem:
$<q(t)>_0 = 0$ (odd number of fields).
$<\int dt' q^3(t')>_0 = \int dt' <q^3(t')>_0 = 0$ (odd number of fields).

So, the denominator is $1 + O(g^2)$.
The numerator simplifies to $\frac{g}{6} \int dt' <q(t) q^3(t')>_0 + O(g^2)$.
Now we calculate $<q(t) q^3(t')>_0 = <q(t) q(t') q(t') q(t')>_0$. By Wick's theorem, we pair $q(t)$ with one of the $q(t')$ terms, and the remaining two $q(t')$ terms with each other. There are 3 ways to choose which $q(t')$ pairs with $q(t)$:
$<q(t) q^3(t')>_0 = 3 \times D(t,t') D(t',t') D(t',t') = 3 D(t,t') D(t',t')^2$.
Substitute $D(t',t') = \frac{1}{2m}$:
$<q(t) q^3(t')>_0 = 3 D(t,t') \left(\frac{1}{2m}\right)^2 = \frac{3}{4m^2} D(t,t')$.

Thus, $G_1(t) = \frac{g}{6} \int dt' \frac{3}{4m^2} D(t,t') + O(g^2) = \frac{g}{8m^2} \int dt' D(t,t') + O(g^2)$.
The integral $\int dt' D(t,t')$ is evaluated over all Euclidean time.
$\int_{-\infty}^{\infty} dt' D(t,t') = \int_{-\infty}^{\infty} dt' \frac{e^{-m|t-t'|}}{2m} = \frac{1}{2m} \left( \int_{-\infty}^{t} e^{m(t'-t)} dt' + \int_{t}^{\infty} e^{-m(t'-t)} dt' \right)$.
$= \frac{1}{2m} \left( \frac{1}{m} [e^{m(t'-t)}]_{-\infty}^{t} - \frac{1}{m} [e^{-m(t'-t)}]_{t}^{\infty} \right) = \frac{1}{2m} \left( \frac{1}{m}(1-0) - \frac{1}{m}(0-1) \right) = \frac{1}{2m} \left( \frac{2}{m} \right) = \frac{1}{m^2}$.
So, $G_1(t) = \frac{g}{8m^2} \frac{1}{m^2} + O(g^2) = \frac{g}{8m^4} + O(g^2)$.

The leading term of $G_1(t)$ is $\frac{g}{8m^4}$.

---
**Part (ii): Connected 2-point function G_2^c(t_1, t_2) modulo g^3**

The connected 2-point function is defined as $G_2^c(t_1, t_2) = <q(t_1) q(t_2)>_c = \left. \frac{\delta^2 \ln Z[J]}{\delta J(t_1) \delta J(t_2)} \right|_{J=0}$, where $Z[J]$ is the generating functional with a source term $J(t)q(t)$.
$Z[J] = \int Dq e^{-S_0[q] + \int dt \frac{g q^3(t)}{6} + \int dt J(t)q(t)}$.
$Z[J] = Z_0 < e^{\int dt \frac{g q^3(t)}{6} + \int dt J(t)q(t)} >_0$.
Expand the exponential up to $O(g^2)$ and $O(J^2)$:
$e^{\int dt \frac{g q^3(t)}{6} + \int dt J(t)q(t)} = 1 + \int dt \frac{g q^3(t)}{6} + \int dt J(t)q(t) + \frac{1}{2} \left( \int dt \frac{g q^3(t)}{6} \right)^2 + \frac{1}{2} \left( \int dt J(t)q(t) \right)^2 + \left( \int dt \frac{g q^3(t)}{6} \right) \left( \int dt J(t)q(t) \right) + O(g^3, gJ^2, J^3)$.
Taking the expectation value $< \cdot >_0$:
$Z[J] = Z_0 \left( 1 + \frac{g}{6} <\int dt' q^3(t')>_0 + \int dt J(t)<q(t)>_0 + \frac{g^2}{72} <\int dt' dt'' q^3(t')q^3(t'')>_0 \right.$
$\left. + \frac{1}{2} \int dt_a dt_b J(t_a) J(t_b) <q(t_a)q(t_b)>_0 + \frac{g}{6} \int dt' dt J(t) <q(t)q^3(t')>_0 + O(g^3, gJ^2, J^3) \right)$.
Using $<q(t)>_0=0$ and $<q^3(t')>_0=0$:
$Z[J] = Z_0 \left( 1 + \frac{g^2}{72} \int dt' dt'' <q^3(t')q^3(t'')>_0 + \frac{1}{2} \int dt_a dt_b J(t_a) J(t_b) D(t_a,t_b) \right.$
$\left. + \frac{g}{6} \int dt' dt J(t) (3 D(t,t') D(t',t')^2) + O(g^3, gJ^2, J^3) \right)$.
Let $Z[J] = Z_0 (1 + A + B J^2 + C J)$.
$\ln Z[J] = \ln Z_0 + \ln(1 + A + B J^2 + C J) = \ln Z_0 + (A + B J^2 + C J) - \frac{1}{2} (C J)^2 + O(g^3, gJ^2, J^3)$.
(Note: $A$ is $O(g^2)$, $B$ is $O(g^0)$, $C$ is $O(g)$).
The term $A$ does not depend on $J$, so it won't contribute to $G_2^c$.
The term $C J$ leads to $G_1(t)$. From part (i), $G_1(t) = C = \frac{g}{6} \int dt' <q(t)q^3(t')>_0$.
$\ln Z[J] = \ln Z_0 + \frac{g^2}{72} \int dt' dt'' <q^3(t')q^3(t'')>_0 + \frac{1}{2} \int dt_a dt_b J(t_a) J(t_b) D(t_a,t_b) $
$+ \frac{g}{6} \int dt dt' J(t) (3 D(t,t') D(t',t')^2) - \frac{1}{2} \left( \frac{g}{6} \int dt dt' J(t) (3 D(t,t') D(t',t')^2) \right)^2 + O(g^3)$.
$G_2^c(t_1, t_2) = \left. \frac{\delta^2 \ln Z[J]}{\delta J(t_1) \delta J(t_2)} \right|_{J=0}$.
The $O(g^0)$ term: $\frac{1}{2} \int dt_a dt_b J(t_a) J(t_b) D(t_a,t_b) \implies D(t_1,t_2)$.
The $O(g^1)$ term: There are no terms of order $g J^2$. (The $g J$ term is for $G_1$).
The $O(g^2)$ terms:
1.  From $B J^2$: $\frac{g^2}{72} \int dt' dt'' <q(t_1)q(t_2)q^3(t')q^3(t'')>_0$. (This is by including $O(g^2 J^2)$ terms in expansion of $Z[J]$ from $e^{-S_I+ \int Jq}$ = $... + \frac{1}{2} (\int Jq)^2 (-S_I)^2 + ... $ )
    More accurately, it is $\frac{1}{2} \int dt_a dt_b J(t_a) J(t_b) \frac{g^2}{72} \int dt' dt'' <q(t_a)q(t_b)q^3(t')q^3(t'')>_0$.
    So it contributes $\frac{g^2}{72} \int dt' dt'' <q(t_1)q(t_2)q^3(t')q^3(t'')>_0$.
2.  From $-\frac{1}{2} (C J)^2$:
    $-\frac{1}{2} \left( \frac{g}{6} \int dt_a dt'_a J(t_a) <q(t_a)q^3(t'_a)>_0 \right) \left( \frac{g}{6} \int dt_b dt'_b J(t_b) <q(t_b)q^3(t'_b)>_0 \right)$.
    Taking $\frac{\delta^2}{\delta J(t_1)\delta J(t_2)}$ gives:
    $- \left( \frac{g}{6} \right)^2 \int dt' dt'' <q(t_1)q^3(t')>_0 <q(t_2)q^3(t'')>_0$.

So, $G_2^c(t_1, t_2) = D(t_1,t_2) + \frac{g^2}{72} \int dt' dt'' \left( <q(t_1)q(t_2)q^3(t')q^3(t'')>_0 - <q(t_1)q^3(t')>_0 <q(t_2)q^3(t'')>_0 \right) + O(g^3)$.
The term in the parenthesis is precisely the connected part of $<q(t_1)q(t_2)q^3(t')q^3(t'')>_0$.
Let's evaluate the connected contributions diagrammatically. Each interaction vertex is $\frac{g}{6} q^3$.
The $O(g^2)$ correction involves two vertices. Let the interaction points be $t'$ and $t''$.
The connected diagrams for $G_2^c(t_1, t_2)$ at $O(g^2)$ are:
1.  Both external lines $q(t_1)$ and $q(t_2)$ attach to the same vertex (say $t'$). The remaining line from $t'$ connects to $t''$, and the $q^3(t'')$ forms a loop (two lines connect to $t'$, one line forms a loop).
    The diagram is $t_1 - t' - t_2$ (two lines from $t'$ used for $t_1, t_2$).
    The remaining line from $t'$ connects to $t''$.
    The remaining two lines from $t''$ connect to each other ($D(t'',t'')$).
    Contribution: $D(t_1,t') D(t_2,t') D(t',t'') D(t'',t'')$.
    Number of ways to contract:
    -   $q(t_1)$ to $q(t')$: 3 ways.
    -   $q(t_2)$ to $q(t')$: 2 ways.
    -   Remaining $q(t')$ to $q(t'')$: 3 ways.
    -   Remaining $q(t'')$ to $q(t'')$: 1 way.
    Total: $3 \times 2 \times 3 \times 1 = 18$.
    This implies $18 D(t_1,t') D(t_2,t') D(t',t'') D(t'',t'')$.
    Due to symmetry $t' \leftrightarrow t''$, we also have $18 D(t_1,t'') D(t_2,t'') D(t'',t') D(t',t')$.
    Sum: $18 (D(t_1,t') D(t_2,t') D(t',t'') D(t'',t'') + D(t_1,t'') D(t_2,t'') D(t'',t') D(t',t'))$.
    Substitute $D(t',t') = D(t'',t'') = \frac{1}{2m}$:
    $18 (D(t_1,t') D(t_2,t') + D(t_1,t'') D(t_2,t'')) D(t',t'') \frac{1}{2m}$.

2.  External lines $q(t_1)$ and $q(t_2)$ attach to different vertices ($t_1$ to $t'$, $t_2$ to $t''$). The remaining two lines from $t'$ and two lines from $t''$ connect to each other.
    (a) The remaining $q(t')$ pair with each other, and remaining $q(t'')$ pair with each other.
        Contribution: $D(t_1,t') D(t_2,t'') D(t',t') D(t'',t'')$.
        Number of ways to contract:
        -   $q(t_1)$ to $q(t')$: 3 ways.
        -   $q(t_2)$ to $q(t'')$: 3 ways.
        -   Remaining $q(t')$ to $q(t')$: 1 way.
        -   Remaining $q(t'')$ to $q(t'')$: 1 way.
        Total: $3 \times 3 \times 1 \times 1 = 9$.
        This implies $9 D(t_1,t') D(t_2,t'') D(t',t') D(t'',t'')$.
    (b) The remaining $q(t')$ connect to $q(t'')$ in two pairs.
        Contribution: $D(t_1,t') D(t_2,t'') D(t',t'') D(t',t'')$.
        Number of ways to contract:
        -   $q(t_1)$ to $q(t')$: 3 ways.
        -   $q(t_2)$ to $q(t'')$: 3 ways.
        -   Remaining $q(t')$ to $q(t'')$: 2 ways.
        -   Remaining $q(t')$ to $q(t'')$: 1 way.
        Total: $3 \times 3 \times 2 \times 1 = 18$.
        This implies $18 D(t_1,t') D(t_2,t'') D(t',t'')^2$.
    Summing (a) and (b), and including the $t_1 \leftrightarrow t_2$ symmetry:
    $9 (D(t_1,t') D(t_2,t'') + D(t_1,t'') D(t_2,t')) D(t',t') D(t'',t'')$
    $+ 18 (D(t_1,t') D(t_2,t'') + D(t_1,t'') D(t_2,t')) D(t',t'')^2$.
    Substitute $D(t',t') = D(t'',t'') = \frac{1}{2m}$:
    $\frac{9}{4m^2} (D(t_1,t') D(t_2,t'') + D(t_1,t'') D(t_2,t')) + 18 (D(t_1,t') D(t_2,t'') + D(t_1,t'') D(t_2,t')) D(t',t'')^2$.

Now, we combine all terms and include the $\frac{g^2}{72}$ coefficient from the expansion:
$G_2^c(t_1, t_2) = D(t_1, t_2) + \frac{g^2}{72} \int dt' dt'' \left[ \frac{18}{2m} (D(t_1,t') D(t_2,t') + D(t_1,t'') D(t_2,t'')) D(t',t'') \right.$
$\left. + \frac{9}{4m^2} (D(t_1,t') D(t_2,t'') + D(t_1,t'') D(t_2,t')) \right.$
$\left. + 18 (D(t_1,t') D(t_2,t'') + D(t_1,t'') D(t_2,t')) D(t',t'')^2 \right] + O(g^3)$.
Simplifying the coefficients:
$\frac{18}{72 \times 2m} = \frac{1}{8m}$.
$\frac{9}{72 \times 4m^2} = \frac{1}{32m^2}$.
$\frac{18}{72} = \frac{1}{4}$.

$G_2^c(t_1, t_2) = D(t_1, t_2) + g^2 \int dt' dt'' \left[ \frac{1}{8m} (D(t_1,t') D(t_2,t') + D(t_1,t'') D(t_2,t'')) D(t',t'') \right.$
$\left. + \frac{1}{32m^2} (D(t_1,t') D(t_2,t'') + D(t_1,t'') D(t_2,t')) \right.$
$\left. + \frac{1}{4} (D(t_1,t') D(t_2,t'') + D(t_1,t'') D(t_2,t')) D(t',t'')^2 \right] + O(g^3)$.

The integral terms can be simplified further:
1.  The term $\frac{1}{8m} \int dt' dt'' (D(t_1,t') D(t_2,t') + D(t_1,t'') D(t_2,t'')) D(t',t'')$.
    Since $\int dt'' D(t',t'') = \frac{1}{m^2}$, this becomes:
    $\frac{1}{8m^3} \int dt' (D(t_1,t') D(t_2,t') + D(t_1,t'') D(t_2,t''))$. (The $t''$ in the second term is a dummy variable, rename to $t'$).
    $= \frac{1}{4m^3} \int dt' D(t_1,t') D(t_2,t')$.
2.  The term $\frac{1}{32m^2} \int dt' dt'' (D(t_1,t') D(t_2,t'') + D(t_1,t'') D(t_2,t'))$.
    Since $\int dt' D(t,t') = \frac{1}{m^2}$, this becomes:
    $\frac{1}{32m^2} \left( (\int dt' D(t_1,t')) (\int dt'' D(t_2,t'')) + (\int dt'' D(t_1,t'')) (\int dt' D(t_2,t')) \right)$.
    $= \frac{1}{32m^2} \left( \frac{1}{m^2} \frac{1}{m^2} + \frac{1}{m^2} \frac{1}{m^2} \right) = \frac{1}{32m^2} \frac{2}{m^4} = \frac{1}{16m^6}$.
3.  The term $\frac{1}{4} \int dt' dt'' (D(t_1,t') D(t_2,t'') + D(t_1,t'') D(t_2,t')) D(t',t'')^2$.
    By symmetry, $D(t_1,t') D(t_2,t'') D(t',t'')^2$ and $D(t_1,t'') D(t_2,t') D(t',t'')^2$ contribute equally to the integral.
    $= \frac{1}{2} \int dt' dt'' D(t_1,t') D(t_2,t'') D(t',t'')^2$.

Combining these simplified terms:

The final answers are:
(i) The leading term of the 1-point function $G_1(t)$ with respect to $g$:
$\boxed{G_1(t) = \frac{g}{8m^4} + O(g^2)}$

(ii) The connected 2-point function $G_2^c(t_1, t_2)$ modulo $g^3$:
$\boxed{G_2^c(t_1, t_2) = D(t_1, t_2) + g^2 \left[ \frac{1}{4m^3} \int dt' D(t_1,t') D(t_2,t') + \frac{1}{16m^6} + \frac{1}{2} \int dt' dt'' D(t_1,t') D(t_2,t'') D(t',t'')^2 \right] + O(g^3)}$
