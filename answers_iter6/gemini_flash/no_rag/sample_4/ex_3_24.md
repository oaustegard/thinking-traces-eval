**Part 1: Compute the 1-loop contribution to $\log(Z/Z_0)$**

The action is $S(x) = \frac{1}{2}x^2 - g\left(\frac{1}{2}x^2 + \frac{1}{6}x^3\right)$.
We can rewrite this as $S(x) = \frac{1}{2}(1-g)x^2 - g\frac{1}{6}x^3$.
The partition function is $Z = \int e^{-S(x)/\hbar} dx$.
The free partition function $Z_0$ corresponds to $g=0$, so $S_0(x) = \frac{1}{2}x^2$.

The 1-loop contribution in quantum field theory is typically obtained from the saddle-point approximation (or Gaussian integral approximation) to the path integral. For a 0-dimensional theory (an ordinary integral), this approximation is:
$\log Z \approx -\frac{1}{\hbar}S(x_0) + \frac{1}{2}\log\left(\frac{2\pi\hbar}{S''(x_0)}\right)$,
where $x_0$ is the classical solution (saddle point) found by setting $S'(x_0)=0$.

First, let's find the saddle point $x_0$ for $S(x)$:
$S'(x) = \frac{d}{dx}\left(\frac{1}{2}(1-g)x^2 - g\frac{1}{6}x^3\right) = (1-g)x - g\frac{1}{2}x^2$.
Setting $S'(x_0)=0$: $x_0\left((1-g) - g\frac{1}{2}x_0\right) = 0$.
The solutions are $x_0=0$ or $x_0 = \frac{2(1-g)}{g}$. For small $g$, $x_0=0$ is the relevant saddle point.
At $x_0=0$, $S(0) = 0$.

Next, we compute the second derivative of $S(x)$ at $x_0=0$:
$S''(x) = (1-g) - gx$.
$S''(0) = 1-g$.

Now we can compute $\log Z$:
$\log Z = \frac{1}{2}\log\left(\frac{2\pi\hbar}{1-g}\right)$.

For the free partition function $Z_0$, the action is $S_0(x) = \frac{1}{2}x^2$.
The saddle point is $x_0=0$. $S_0(0)=0$.
The second derivative is $S_0''(x)=1$, so $S_0''(0)=1$.
$\log Z_0 = \frac{1}{2}\log(2\pi\hbar)$.

Finally, the 1-loop contribution to $\log(Z/Z_0)$ is:
$\log(Z/Z_0) = \log Z - \log Z_0 = \frac{1}{2}\log\left(\frac{2\pi\hbar}{1-g}\right) - \frac{1}{2}\log(2\pi\hbar)$
$\log(Z/Z_0) = \frac{1}{2}\log\left(\frac{1}{1-g}\right)$.
Using the Taylor expansion $\log(1/(1-y)) = -\log(1-y) = \sum_{k=1}^\infty \frac{y^k}{k}$ for $|y|<1$:
$\log(Z/Z_0) = \frac{1}{2}\sum_{k=1}^\infty \frac{g^k}{k} = \frac{g}{2} + \frac{g^2}{4} + \frac{g^3}{6} + \dots$.
This is the 1-loop contribution to $\log(Z/Z_0)$.
