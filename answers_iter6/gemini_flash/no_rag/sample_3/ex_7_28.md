The problem asks for two quantities related to a Euclidean cubic anharmonic oscillator with potential $U(q) = \frac{m^2 q^2}{2} - \frac{g q^3}{6}$. We will use the path integral formalism in Euclidean time. The action is $S[q] = \int dt' \left( \frac{1}{2} \dot{q}(t')^2 + U(q(t')) \right)$. We split the action into a free part $S_0[q]$ and an interaction part $S_{int}[q]$:
$S_0[q] = \int dt' \left( \frac{1}{2} \dot{q}(t')^2 + \frac{m^2}{2} q(t')^2 \right)$
$S_{int}[q] = -\frac{g}{6} \int dt' q(t')^3$

The Euclidean free propagator for the harmonic oscillator is $D(t) = \frac{1}{2m} e^{-m|t|}$. In Fourier space, $\tilde{D}(p) = \frac{1}{p^2+m^2}$.

---

**(i) Calculate the leading term of the 1-point function $G_1(t) := <q(t)>$ with respect to g.**

The 1-point function is given by:
$G_1(t) = <q(t)> = \frac{\int Dq \, q(t) e^{-S[q]}}{\int Dq \, e^{-S[q]}}$
We expand $e^{-S[q]} = e^{-S_0[q]} e^{-S_{int}[q]} = e^{-S_0[q]} \left( 1 + \frac{g}{6} \int dt' q(t')^3 + O(g^2) \right)$.

The numerator $N$ becomes:
$N = \int Dq \, q(t) e^{-S_0[q]} \left( 1 + \frac{g}{6} \int dt' q(t')^3 + O(g^2) \right)$
$N = Z_0 \left( <q(t)>_0 + \frac{g}{6} \int dt' <q(t) q(t')^3>_0 + O(g^2) \right)$
where $Z_0 = \int Dq \, e^{-S_0[q]}$ is the free partition function, and $<...>_0$ denotes expectation values with respect to $S_0$.

The denominator $D$ becomes:
$D = \int Dq \, e^{-S_0[q]} \left( 1 + \frac{g}{6} \int dt' q(t')^3 + O(g^2) \right)$
$D = Z_0 \left( 1 + \frac{g}{6} \int dt' <q(t')^3>_0 + O(g^2) \right)$

For a harmonic oscillator, the ground state is symmetric, so expectation values of odd powers of $q$ are zero:
$<q(t)>_0 = 0$
$<q(t')^3>_0 = 0$

Therefore, the 1-point function simplifies to:
$G_1(t) = \frac{Z_0 \left( 0 + \frac{g}{6} \int dt' <q(t) q(t')^3>_0 + O(g^2) \right)}{Z_0 \left( 1 + 0 + O(g^2) \right)} = \frac{g}{6} \int dt' <q(t) q(t')^3>_0 + O(g^2)$

This is the leading term. We calculate $<q(t) q(t')^3>_0$ using Wick's theorem:
$<q(t) q(t') q(t') q(t')>_0$. There are 3 ways to contract $q(t)$ with one of the $q(t')$, and the remaining two $q(t')$ must contract with each other.
$<q(t) q(t')^3>_0 = 3 <q(t) q(t')>_0 <q(t') q(t')>_0 = 3 D(t-t') D(0)$.

Substitute this back into the expression for $G_1(t)$:
$G_1(t) = \frac{g}{6} \int dt' \, 3 D(t-t') D(0) = \frac{g}{2} D(0) \int dt' D(t-t')$.

Now, we use the specific form of the propagator: $D(t) = \frac{1}{2m} e^{-m|t|}$.
$D(0) = \frac{1}{2m}$.
The integral $\int dt' D(t-t')$ is:
$\int_{-\infty}^\infty \frac{1}{2m} e^{-m|t-t'|} dt' = \frac{1}{2m} \int_{-\infty}^\infty e^{-m|u|} du$ (where $u = t-t'$)
$= \frac{1}{2m} \left( 2 \int_0^\infty e^{-mu} du \right) = \frac{1}{2m} \left( 2 \left[ -\frac{1}{m} e^{-mu} \right]_0^\infty \right) = \frac{1}{2m} \left( \frac{2}{m} \right) = \frac{1}{m^2}$.

Plugging these values in:
$G_1(t) = \frac{g}{2} \left(\frac{1}{2m}\right) \left(\frac{1}{m^2}\right) = \frac{g}{4m^3}$.
The leading term is independent of $t$, as expected for a vacuum expectation value in a time-translationally invariant system.

**(ii) Calculate the connected 2-point function $G_2^c(t_1, t_2) := <q(t_1) q(t_2)>_c$ modulo g^3.**

The connected 2-point function can be directly calculated from the generating functional for connected correlation functions, $W[J] = \ln Z[J]$, where $Z[J]$ is the generating functional for all correlation functions:
$G_2^c(t_1, t_2) = \frac{\delta^2 W[J]}{\delta J(t_1) \delta J(t_2)} \Big|_{J=0}$.

We use the perturbation expansion for $W[J]$:
$W[J] = W_0[J] + \frac{g}{6} \int dt_a <q(t_a)^3>_J^0 + \frac{g^2}{72} \int dt_a dt_b \left( <q(t_a)^3 q(t_b)^3>_J^0 - <q(t_a)^3>_J^0 <q(t_b)^3>_J^0 \right) + O(g^3)$.
Here, $W_0[J] = \ln Z_0[J] = \frac{1}{2} \int dt' dt'' J(t') D(t'-t'') J(t'')$, and $<O>_J^0$ denotes expectation values in the free theory with a source $J$.

1.  **Leading term (Order $g^0$):**
    $\frac{\delta^2 W_0[J]}{\delta J(t_1) \delta J(t_2)} \Big|_{J=0} = \frac{\delta}{\delta J(t_1)} \left( \int dt'' D(t_2-t'') J(t'') \right) \Big|_{J=0} = D(t_1-t_2)$.
    This is the free propagator.

2.  **Order $g$ term:**
    We need to calculate $\frac{g}{6} \int dt_a \frac{\delta^2}{\delta J(t_1) \delta J(t_2)} <q(t_a)^3>_J^0 \Big|_{J=0}$.
    $<q(t_a)^3>_J^0 = (\int dt' D(t_a-t') J(t'))^3 + 3 D(0) (\int dt' D(t_a-t') J(t'))$.
    Let $A_a = \int dt' D(t_a-t') J(t')$. Then $<q(t_a)^3>_J^0 = A_a^3 + 3 D(0) A_a$.
    $\frac{\delta}{\delta J(t_1)} (A_a^3 + 3 D(0) A_a) = 3 A_a^2 D(t_a-t_1) + 3 D(0) D(t_a-t_1)$.
    $\frac{\delta^2}{\delta J(t_1) \delta J(t_2)} (A_a^3 + 3 D(0) A_a) = \frac{\delta}{\delta J(t_2)} (3 A_a^2 D(t_a-t_1) + 3 D(0) D(t_a-t_1))$.
    $= 3 D(t_a-t_1) (2 A_a D(t_a-t_2)) = 6 A_a D(t_a-t_1) D(t_a-t_2)$.
    At $J=0$, $A_a=0$. Thus, the $O(g)$ term is 0.

3.  **Order $g^2$ term:**
    This term is $\frac{g^2}{72} \int dt_a dt_b \frac{\delta^2}{\delta J(t_1) \delta J(t_2)} \left( <q(t_a)^3 q(t_b)^3>_J^0 - <q(t_a)^3>_J^0 <q(t_b)^3>_J^0 \right) \Big|_{J=0}$.
    This corresponds to the sum of all connected Feynman diagrams with two external lines $t_1, t_2$ and two interaction vertices at $t_a, t_b$. In $\phi^3$ theory, the lowest-order non-trivial contribution to the connected 2-point function (beyond the free propagator) comes from the self-energy insertion into the propagator.
    The connected 2-point function in momentum space is $G_2^c(p) = \frac{1}{p^2+m^2 - \Sigma(p)}$.
    Expanding this to $O(g^2)$: $G_2^c(p) = \tilde{D}(p) + \tilde{D}(p) \Sigma(p) \tilde{D}(p) + O(g^3)$.
    The lowest-order 1-particle irreducible (1PI) self-energy $\Sigma(p)$ for a $\phi^3$ theory is given by the one-loop diagram (a loop of two propagators connecting two vertices):
    $\Sigma(p) = \frac{g^2}{2} \int \frac{dk}{2\pi} \frac{1}{(k^2+m^2)((p-k)^2+m^2)}$.
    In position space, this corresponds to:
    $\Sigma(t_x-t_y) = \frac{g^2}{2} \int dt_z D(t_x-t_z) D(t_z-t_y)$. (The factor $1/2$ is a symmetry factor for the loop).

    So, the $O(g^2)$ contribution to $G_2^c(t_1, t_2)$ is:
    $\int dt_x dt_y D(t_1-t_x) \Sigma(t_x-t_y) D(t_y-t_2)$
    $= \int dt_x dt_y D(t_1-t_x) \left( \frac{g^2}{2} \int dt_z D(t_x-t_z) D(t_z-t_y) \right) D(t_y-t_2)$
    $= \frac{g^2}{2} \int dt_x dt_y dt_z D(t_1-t_x) D(t_x-t_z) D(t_z-t_y) D(t_y-t_2)$.

Combining all terms, the connected 2-point function up to $O(g^2)$ is:
$G_2^c(t_1, t_2) = D(t_1-t_2) + \frac{g^2}{2} \int dt_x dt_y dt_z D(t_1-t_x) D(t_x-t_z) D(t_z-t_y) D(t_y-t_2) + O(g^3)$.

The integral term can be expressed in terms of Fourier transforms. Let $D_n(t) = \int \frac{dp}{2\pi} \frac{e^{ipt}}{(p^2+m^2)^n}$.
Then the $O(g^2)$ term is $\frac{g^2}{2} D_4(t_1-t_2)$.

The final answers are:

(i) The leading term of the 1-point function $G_1(t)$ is:
$\boxed{G_1(t) = \frac{g}{4m^3}}$

(ii) The connected 2-point function $G_2^c(t_1, t_2)$ modulo $g^3$ is:
$\boxed{G_2^c(t_1, t_2) = D(t_1-t_2) + \frac{g^2}{2} \int dt_x dt_y dt_z D(t_1-t_x) D(t_x-t_z) D(t_z-t_y) D(t_y-t_2) + O(g^3)}$
where $D(t) = \frac{1}{2m} e^{-m|t|}$ is the free Euclidean propagator.
