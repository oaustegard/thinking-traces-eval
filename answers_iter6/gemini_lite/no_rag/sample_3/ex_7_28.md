Here's a step-by-step solution to the problem.

We are given the potential for a Euclidean cubic anharmonic oscillator:
$U(q) = \frac{m^2 q^2}{2} - \frac{g q^3}{6}$

The expectation value of an observable $O(q)$ in the Euclidean path integral formulation is given by:
$<O(q)> = \frac{\int \mathcal{D}q \, O(q) e^{-S[q]}}{\int \mathcal{D}q \, e^{-S[q]}}$
where the Euclidean action is $S[q] = \int dt \, (\frac{1}{2} (\frac{dq}{dt})^2 + U(q))$.
For simplicity, we will work in the context of a quantum mechanical system where the path integral is over configurations $q(t)$. The partition function is $Z = \int \mathcal{D}q \, e^{-S[q]}$.

The 1-point function is $G_1(t) = <q(t)>$.
The 2-point function is $G_2(t_1, t_2) = <q(t_1) q(t_2)>$.
The connected 2-point function is $G_2^c(t_1, t_2) = <q(t_1) q(t_2)>_c = <q(t_1) q(t_2)> - <q(t_1)><q(t_2)>$.

We can express these expectation values using the generating functional $Z[J]$ for currents $J(t)$:
$Z[J] = \int \mathcal{D}q \, e^{-S[q] + \int dt \, J(t)q(t)}$

Then, the expectation values can be obtained by functional differentiation:
$<q(t)> = \frac{1}{Z} \frac{\delta Z}{\delta J(t)} \Big|_{J=0}$
$<q(t_1) q(t_2)> = \frac{1}{Z} \frac{\delta^2 Z}{\delta J(t_1) \delta J(t_2)} \Big|_{J=0}$

Let's expand the action in terms of the "free" part and the "interaction" part:
$S[q] = S_0[q] + S_I[q]$
where $S_0[q] = \int dt \, \frac{1}{2} (\frac{dq}{dt})^2 + \frac{m^2 q^2}{2}$ is the action for the harmonic oscillator, and $S_I[q] = \int dt \, (-\frac{g q^3}{6})$.

The generating functional can be written as:
$Z[J] = \int \mathcal{D}q \, e^{-S_0[q] - S_I[q] + \int dt \, J(t)q(t)}$
$Z[J] = \int \mathcal{D}q \, e^{-S_0[q] + \int dt \, J(t)q(t)} e^{-S_I[q]}$

We can expand the exponential of the interaction term in a power series:
$e^{-S_I[q]} = 1 - S_I[q] + \frac{1}{2!} (S_I[q])^2 - \dots$
$e^{-S_I[q]} = 1 + \int dt \, \frac{g q^3(t)}{6} + \frac{1}{2} \left( \int dt \, \frac{g q^3(t)}{6} \right)^2 + \dots$

The partition function $Z$ is obtained by setting $J=0$.
$Z = \int \mathcal{D}q \, e^{-S_0[q]} (1 + \int dt \, \frac{g q^3(t)}{6} + O(g^2))$
$Z = Z_0 (1 + \int dt \, \frac{g}{6} <q^3(t)>_0 + O(g^2))$
where $Z_0 = \int \mathcal{D}q \, e^{-S_0[q]}$ is the partition function of the free harmonic oscillator, and $< \cdot >_0$ denotes expectation values with respect to $S_0$.

Similarly, for the generating functional with sources:
$Z[J] = \int \mathcal{D}q \, e^{-S_0[q] + \int dt \, J(t)q(t)} (1 + \int dt' \, \frac{g q^3(t')}{6} + O(g^2))$
$Z[J] = Z_0[J] (1 + \int dt' \, \frac{g}{6} <q^3(t')>_J + O(g^2))$
where $Z_0[J] = \int \mathcal{D}q \, e^{-S_0[q] + \int dt \, J(t)q(t)}$ is the generating functional for the free harmonic oscillator.

For the free harmonic oscillator, we know:
$Z_0[J] = Z_0 e^{\frac{1}{2} \int dt dt' J(t) G_0(t, t') J(t')}$
where $G_0(t, t')$ is the 2-point function of the free harmonic oscillator.
The 1-point function of the free oscillator is $<q(t)>_0 = \frac{1}{Z_0} \frac{\delta Z_0}{\delta J(t)} \Big|_{J=0} = \int dt' G_0(t, t') J(t') \Big|_{J=0} = 0$.
This is because the potential $U(q)$ is symmetric about $q=0$.

(i) Calculate the leading term of the 1-point function $G_1(t) = <q(t)>$ with respect to $g$.

$G_1(t) = <q(t)> = \frac{1}{Z} \frac{\delta Z}{\delta J(t)} \Big|_{J=0}$
$Z = Z_0 [1 + \frac{g}{6} \int dt' <q^3(t')>_0 + O(g^2)]$
$\frac{\delta Z}{\delta J(t)} = \frac{\delta Z_0}{\delta J(t)} [1 + \frac{g}{6} \int dt' <q^3(t')>_0] + Z_0 [\frac{g}{6} \int dt' \frac{\delta <q^3(t')>_0}{\delta J(t)}]$

Setting $J=0$:
$\frac{\delta Z}{\delta J(t)} \Big|_{J=0} = \frac{\delta Z_0}{\delta J(t)} \Big|_{J=0} [1 + \frac{g}{6} \int dt' <q^3(t')>_0] + Z_0 [\frac{g}{6} \int dt' \frac{\delta <q^3(t')>_0}{\delta J(t)} \Big|_{J=0}]$

We have $\frac{\delta Z_0}{\delta J(t)} \Big|_{J=0} = Z_0 <q(t)>_0 = 0$.
So, $\frac{\delta Z}{\delta J(t)} \Big|_{J=0} = Z_0 [\frac{g}{6} \int dt' \frac{\delta <q^3(t')>_0}{\delta J(t)} \Big|_{J=0}]$

Now, let's calculate $\frac{\delta <q^3(t')>_0}{\delta J(t)} \Big|_{J=0}$.
$<q^3(t')>_0 = \frac{1}{Z_0} \int \mathcal{D}q \, q^3(t') e^{-S_0[q]}$
$\frac{\delta <q^3(t')>_0}{\delta J(t)} = \frac{\delta}{\delta J(t)} \left( \frac{\int \mathcal{D}q \, q^3(t') e^{-S_0[q] + \int dt'' J(t'')q(t'')}}{\int \mathcal{D}q \, e^{-S_0[q] + \int dt'' J(t'')q(t'')} } \right)$
Using the formula $\frac{\delta <A>_J}{\delta J(t)} = <A q(t)>_J - <A>_J <q(t)>_J$.
So, $\frac{\delta <q^3(t')>_0}{\delta J(t)} \Big|_{J=0} = <q^3(t') q(t)>_0 - <q^3(t')>_0 <q(t)>_0$.
Since $<q(t)>_0 = 0$ and $<q^3(t')>_0 = 0$ (due to symmetry of the harmonic oscillator potential), this simplifies to:
$\frac{\delta <q^3(t')>_0}{\delta J(t)} \Big|_{J=0} = <q^3(t') q(t)>_0$.

Using Wick's theorem for the free oscillator (Gaussian integral):
$<q(t_1) q(t_2) q(t_3) q(t_4)>_0 = <q(t_1)q(t_2)>_0 <q(t_3)q(t_4)>_0 + \text{3 permutations}$
$<q^3(t') q(t)>_0 = <q(t')q(t') q(t') q(t)>_0$
$= <q(t')q(t')>_0 <q(t')q(t)>_0 + <q(t')q(t')>_0 <q(t')q(t)>_0 + <q(t')q(t)>_0 <q(t')q(t')>_0$
$= 3 <q(t')q(t')>_0 <q(t')q(t)>_0$

The 2-point function of the free harmonic oscillator is $G_0(t, t') = \frac{e^{-m|t-t'|}}{2m}$ (this is for the time-independent case, if we consider a time-dependent problem, it would be a Green's function). Assuming we are in a situation where $G_0(t, t')$ is well-defined and normalized, a common convention for correlation functions in Euclidean path integrals relates to the propagator. For the purposes of perturbation theory, we often consider $G_0(t, t')$ to be the propagator of the free theory. If $m^2 > 0$, the frequency is $\omega = m$. The propagator is $G_0(t, t') = \frac{1}{2\omega} e^{-\omega|t-t'|}$ in infinite time. However, the problem statement is general. Let's assume that the context implies the standard propagator for the free oscillator. Let's denote the free 2-point function as $G_0(t,t')$.

$<q(t')q(t)>_0 = G_0(t', t)$.
$<q(t')q(t')>_0 = G_0(t', t')$.

So, $\frac{\delta <q^3(t')>_0}{\delta J(t)} \Big|_{J=0} = 3 G_0(t', t') G_0(t', t)$.

Substituting back into the expression for $\frac{\delta Z}{\delta J(t)} \Big|_{J=0}$:
$\frac{\delta Z}{\delta J(t)} \Big|_{J=0} = Z_0 \left[ \frac{g}{6} \int dt' (3 G_0(t', t') G_0(t', t)) \right]$
$\frac{\delta Z}{\delta J(t)} \Big|_{J=0} = Z_0 \frac{g}{2} \int dt' G_0(t', t') G_0(t', t)$

Now, $G_1(t) = \frac{1}{Z} \frac{\delta Z}{\delta J(t)} \Big|_{J=0} = \frac{1}{Z_0 (1 + O(g))} Z_0 \frac{g}{2} \int dt' G_0(t', t') G_0(t', t)$
$G_1(t) = \frac{g}{2} \int dt' G_0(t', t') G_0(t', t) (1 + O(g))$
The leading term with respect to $g$ is obtained by taking the $O(g^0)$ term from the expansion of $Z$.
$G_1(t) = \frac{g}{2} \int dt' G_0(t', t') G_0(t', t) + O(g^2)$.

The term $\int dt' G_0(t', t') G_0(t', t)$ is a tadpole diagram.

(ii) Calculate the connected 2-point function $G_2^c(t_1, t_2) := <q(t_1) q(t_2)>_c$ modulo $g^3$.

$G_2^c(t_1, t_2) = <q(t_1) q(t_2)> - <q(t_1)><q(t_2)>$
We need $<q(t_1) q(t_2)>$ up to $O(g^2)$.

$<q(t_1) q(t_2)> = \frac{1}{Z} \frac{\delta^2 Z}{\delta J(t_1) \delta J(t_2)} \Big|_{J=0}$
$Z = Z_0 [1 + \frac{g}{6} \int dt' <q^3(t')>_0 + \frac{g^2}{2 \cdot 6^2} \int dt' dt'' <q^3(t')q^3(t'')>_0 + O(g^3)]$
$Z = Z_0 [1 + \frac{g}{6} \int dt' <q^3(t')>_0 + \frac{g^2}{72} \int dt' dt'' <q^3(t')q^3(t'')>_0 + O(g^3)]$
Since $<q^3(t')>_0 = 0$, the $g$ term vanishes.
$Z = Z_0 [1 + \frac{g^2}{72} \int dt' dt'' <q^3(t')q^3(t'')>_0 + O(g^3)]$

$\frac{\delta Z}{\delta J(t_1)} = \frac{\delta Z_0}{\delta J(t_1)} [1 + \frac{g^2}{72} \int dt' dt'' <q^3(t')q^3(t'')>_0] + Z_0 [\frac{g^2}{72} \int dt' dt'' \frac{\delta <q^3(t')q^3(t'')>_0}{\delta J(t_1)}]$
Set $J=0$:
$\frac{\delta Z}{\delta J(t_1)} \Big|_{J=0} = \frac{\delta Z_0}{\delta J(t_1)} \Big|_{J=0} [1 + O(g^2)] + Z_0 [\frac{g^2}{72} \int dt' dt'' \frac{\delta <q^3(t')q^3(t'')>_0}{\delta J(t_1)} \Big|_{J=0}]$
Since $\frac{\delta Z_0}{\delta J(t_1)} \Big|_{J=0} = Z_0 <q(t_1)>_0 = 0$, we have:
$\frac{\delta Z}{\delta J(t_1)} \Big|_{J=0} = Z_0 [\frac{g^2}{72} \int dt' dt'' \frac{\delta <q^3(t')q^3(t'')>_0}{\delta J(t_1)} \Big|_{J=0}]$

Now differentiate again with respect to $J(t_2)$:
$\frac{\delta^2 Z}{\delta J(t_1) \delta J(t_2)} \Big|_{J=0} = \frac{\delta Z_0}{\delta J(t_1)} \Big|_{J=0} [\frac{g^2}{72} \int dt' dt'' \frac{\delta <q^3(t')q^3(t'')>_0}{\delta J(t_2)} \Big|_{J=0}]$
$+ Z_0 [\frac{g^2}{72} \int dt' dt'' \frac{\delta^2 <q^3(t')q^3(t'')>_0}{\delta J(t_1) \delta J(t_2)} \Big|_{J=0}]$

The first term vanishes because $\frac{\delta Z_0}{\delta J(t_1)} \Big|_{J=0} = 0$.
So, $\frac{\delta^2 Z}{\delta J(t_1) \delta J(t_2)} \Big|_{J=0} = Z_0 [\frac{g^2}{72} \int dt' dt'' \frac{\delta^2 <q^3(t')q^3(t'')>_0}{\delta J(t_1) \delta J(t_2)} \Big|_{J=0}]$

We need $\frac{\delta^2 <q^3(t')q^3(t'')>_0}{\delta J(t_1) \delta J(t_2)} \Big|_{J=0}$.
Using $\frac{\delta <A>_J}{\delta J(t)} = <A q(t)>_J - <A>_J <q(t)>_J$, and setting $J=0$ where $<q>_0=0$:
$\frac{\delta <q^3(t')q^3(t'')>_0}{\delta J(t_1)} \Big|_{J=0} = <q^3(t')q^3(t'')q(t_1)>_0$.
$\frac{\delta^2 <q^3(t')q^3(t'')>_0}{\delta J(t_1) \delta J(t_2)} \Big|_{J=0} = <q^3(t')q^3(t'')q(t_1)q(t_2)>_0$.

So, $\frac{\delta^2 Z}{\delta J(t_1) \delta J(t_2)} \Big|_{J=0} = Z_0 \frac{g^2}{72} \int dt' dt'' <q^3(t')q^3(t'')q(t_1)q(t_2)>_0$.

Now, we need to evaluate $<q^3(t')q^3(t'')q(t_1)q(t_2)>_0$ using Wick's theorem.
This is a 6-point correlation function.
$<q_a q_b q_c q_d q_e q_f>_0 = \sum_{pairs} G_0(q_i, q_j)$. There are $5!! = 15$ terms.

Let's consider the term $<q(t_1) q(t_2)>$ to the order $g^2$.
$<q(t_1) q(t_2)> = \frac{1}{Z} \frac{\delta^2 Z}{\delta J(t_1) \delta J(t_2)} \Big|_{J=0}$
$Z = Z_0 (1 + \frac{g}{6} \int dt' <q^3(t')>_0 + \frac{1}{2} (\frac{g}{6})^2 \int dt' dt'' <q^3(t')q^3(t'')>_0 + O(g^3))$
$Z = Z_0 (1 + 0 + \frac{g^2}{72} \int dt' dt'' <q^3(t')q^3(t'')>_0 + O(g^3))$

$\frac{\delta Z}{\delta J(t_1)} = \frac{\delta Z_0}{\delta J(t_1)} (1 + O(g^2)) + Z_0 \frac{g^2}{72} \int dt' dt'' \frac{\delta <q^3(t')q^3(t'')>_0}{\delta J(t_1)}$
At $J=0$: $\frac{\delta Z}{\delta J(t_1)} = Z_0 \frac{g^2}{72} \int dt' dt'' <q^3(t')q^3(t'')q(t_1)>_0$.

$\frac{\delta^2 Z}{\delta J(t_1) \delta J(t_2)} = \frac{\delta}{\delta J(t_2)} \left( Z_0 \frac{g^2}{72} \int dt' dt'' <q^3(t')q^3(t'')q(t_1)>_0 \right)$
At $J=0$:
$\frac{\delta^2 Z}{\delta J(t_1) \delta J(t_2)} = Z_0 \frac{g^2}{72} \int dt' dt'' <q^3(t')q^3(t'')q(t_1)q(t_2)>_0$
(The terms involving $\frac{\delta Z_0}{\delta J}$ vanish).

So, $<q(t_1) q(t_2)> = \frac{1}{Z} Z_0 \frac{g^2}{72} \int dt' dt'' <q^3(t')q^3(t'')q(t_1)q(t_2)>_0 (1 + O(g^2))$
$<q(t_1) q(t_2)> = \frac{g^2}{72} \int dt' dt'' <q^3(t')q^3(t'')q(t_1)q(t_2)>_0 + O(g^4)$.

Let's evaluate $<q^3(t')q^3(t'')q(t_1)q(t_2)>_0$.
This is the expectation of $q(t')q(t')q(t') q(t'')q(t'')q(t'') q(t_1)q(t_2)$.
Using Wick's theorem, we pair them up. The terms will be products of $G_0$.
The structure of the terms will be:
$G_0(t_1, t_2) \times (\text{terms from } q^3(t')q^3(t''))$.
Consider the contribution from the $g^2$ term in the expansion of $Z$.
$Z = Z_0 \left(1 + \frac{g^2}{2} \int dt_1 dt_2 \frac{1}{36} <q^3(t_1)q^3(t_2)>_0 + O(g^3) \right)$
$<q(t_1)q(t_2)> = \frac{1}{Z} \frac{\delta^2 Z}{\delta J(t_1)\delta J(t_2)} \Big|_{J=0}$.
The term $\frac{\delta^2 Z}{\delta J(t_1)\delta J(t_2)} \Big|_{J=0}$ from the $g^2$ expansion of $Z$ is:
$Z_0 \frac{1}{2} \int dt_1' dt_2' \frac{1}{36} \frac{\delta^2}{\delta J(t_1)\delta J(t_2)} <q^3(t_1')q^3(t_2')>_0 \Big|_{J=0}$
$= Z_0 \frac{1}{72} \int dt_1' dt_2' <q^3(t_1')q^3(t_2')q(t_1)q(t_2)>_0$.

So, $<q(t_1)q(t_2)> = \frac{1}{Z} Z_0 \frac{1}{72} \int dt_1' dt_2' <q^3(t_1')q^3(t_2')q(t_1)q(t_2)>_0 (1+O(g^2))$
$= \frac{1}{72} \int dt_1' dt_2' <q^3(t_1')q^3(t_2')q(t_1)q(t_2)>_0 + O(g^4)$.

Let's evaluate $<q^3(t')q^3(t'')q(t_1)q(t_2)>_0$.
This is a 6-point correlation function.
The pairings can be:
1. $G_0(t_1, t_2)$ paired with pairings from $q^3(t')q^3(t'')$.
2. $G_0(t_1, t')$ paired with pairings from $q(t_2)q^3(t')q^3(t'')$.
3. $G_0(t_1, t'')$ paired with pairings from $q(t_2)q^3(t')q^3(t'')$.
And so on.

Let's use Feynman diagrams.
The 2-point function $<q(t_1)q(t_2)>$ can be represented by diagrams.
Order $g^0$: A line connecting $t_1$ and $t_2$, representing $G_0(t_1, t_2)$.
Order $g^1$: No connected diagrams. The interaction term is cubic, so it cannot connect two external points directly.
Order $g^2$: We need to connect $t_1$ and $t_2$ using two cubic vertices.
The interaction term is $-\frac{g}{6} \int dt q^3(t)$.
The vertices are cubic.
To get a 2-point function, we can have:
- A line from $t_1$ to a vertex, two lines from the vertex to internal points, and then lines from internal points to $t_2$. This requires at least two vertices.
- A line from $t_1$ to a vertex, then a line from that vertex to another vertex, and then a line from the second vertex to $t_2$. This is a "chain" of interactions.

The contribution to $<q(t_1)q(t_2)>$ up to $g^2$:
$<q(t_1)q(t_2)> = G_0(t_1, t_2) + \frac{g^2}{2!} \int dt' dt'' G_0(t_1, t') G_0(t', t'') G_0(t'', t_2) \left(-\frac{1}{6}\right)^2 \times (\text{number of ways to connect})$. This is not the correct way to sum up.

Let's go back to the expansion of Z.
$Z = Z_0 \left( 1 + \frac{g}{6} \int dt <q^3(t)>_0 + \frac{1}{2} (\frac{g}{6})^2 \int dt_1 dt_2 <q^3(t_1)q^3(t_2)>_0 + O(g^3) \right)$
$<q(t_1)q(t_2)> = \frac{1}{Z} \frac{\delta^2 Z}{\delta J(t_1)\delta J(t_2)} \Big|_{J=0}$

We need to consider the terms that contribute to $<q(t_1)q(t_2)>$ up to $g^2$.
The expansion of $Z[J]$:
$Z[J] = Z_0[J] \exp\left( \sum_{n=1}^\infty \frac{1}{n!} \int dt_1 \dots dt_n \left(\frac{-g}{6}\right)^n <q(t_1)\dots q(t_n)>_{0, \text{connected}} J(t_1)\dots J(t_n) \right)$
This is incorrect. The expansion of the exponential of the interaction term is what we used.

Let's use the formula:
$G_2^c(t_1, t_2) = <q(t_1) q(t_2)> - <q(t_1)><q(t_2)>$
We need $<q(t_1) q(t_2)>$ up to $g^2$.
$Z = Z_0 \left( 1 + \frac{g}{6} \int dt <q^3(t)>_0 + \frac{g^2}{2} \int dt_1 dt_2 \frac{1}{36} <q^3(t_1)q^3(t_2)>_0 + O(g^3) \right)$
Since $<q^3(t)>_0 = 0$,
$Z = Z_0 \left( 1 + \frac{g^2}{72} \int dt_1 dt_2 <q^3(t_1)q^3(t_2)>_0 + O(g^3) \right)$

$\frac{\delta^2 Z}{\delta J(t_1)\delta J(t_2)} \Big|_{J=0} = \frac{\delta^2 Z_0}{\delta J(t_1)\delta J(t_2)} \Big|_{J=0} \left(1 + O(g^2)\right) + Z_0 \frac{g^2}{72} \int dt_1' dt_2' \frac{\delta^2 <q^3(t_1')q^3(t_2')>_0}{\delta J(t_1)\delta J(t_2)} \Big|_{J=0}$
$= Z_0 G_0(t_1, t_2) (1+O(g^2)) + Z_0 \frac{g^2}{72} \int dt_1' dt_2' <q^3(t_1')q^3(t_2')q(t_1)q(t_2)>_0$.

So, $<q(t_1)q(t_2)> = \frac{1}{Z} \left( Z_0 G_0(t_1, t_2) + Z_0 \frac{g^2}{72} \int dt_1' dt_2' <q^3(t_1')q^3(t_2')q(t_1)q(t_2)>_0 \right) (1+O(g^2))$
$= G_0(t_1, t_2) (1+O(g^2)) + \frac{g^2}{72} \int dt_1' dt_2' <q^3(t_1')q^3(t_2')q(t_1)q(t_2)>_0 (1+O(g^2))$.

We need to evaluate $<q^3(t_1')q^3(t_2')q(t_1)q(t_2)>_0$.
This is the expectation of 8 powers of q.
The terms that contribute to $<q(t_1)q(t_2)>$ up to $g^2$ are:
1. $G_0(t_1, t_2)$ (order $g^0$).
2. Diagrams with two cubic vertices.
   - A vertex at $t_1'$ connected to $t_1$ and two internal points, and a vertex at $t_2'$ connected to $t_2$ and two internal points, and these internal points are connected. This is not how it works.

Let's consider the structure of the interaction term $H_I = -\frac{g}{6} q^3$.
The 2-point function $<q(t_1)q(t_2)>$ up to $g^2$ will have the form $G_0(t_1, t_2) + (\text{terms with } g^2)$.
The terms with $g^2$ will involve two cubic vertices.
The cubic vertex connects three fields.
To get a 2-point function, we need external points $t_1$ and $t_2$.
Consider a diagram with two vertices $V_1$ at $t_1'$ and $V_2$ at $t_2'$.
The lines from the vertices connect to external points or other vertices.
The interaction term is $\int dt (-\frac{g}{6}) q^3(t)$.

The contribution to $<q(t_1)q(t_2)>$ from the $g^2$ term in the expansion of $Z$ is:
$\frac{1}{Z_0} \frac{\delta^2}{\delta J(t_1)\delta J(t_2)} \left( Z_0 \frac{g^2}{72} \int dt_1' dt_2' <q^3(t_1')q^3(t_2')>_0 \right) \Big|_{J=0}$
$= \frac{g^2}{72} \int dt_1' dt_2' <q^3(t_1')q^3(t_2')q(t_1)q(t_2)>_0$.

Let's evaluate $<q^3(t_1')q^3(t_2')q(t_1)q(t_2)>_0$.
The terms are:
