Let the potential be $U(q) = \frac{1}{2} m^2 q^2 - \frac{g}{6} q^3$. This is the Euclidean potential for a cubic anharmonic oscillator. We are asked to calculate the 1-point function $G_1(t) = \langle q(t) \rangle$ and the connected 2-point function $G_2^c(t_1, t_2) = \langle q(t_1) q(t_2) \rangle_c$.

The expectation value $\langle O \rangle$ in Euclidean quantum mechanics is given by the path integral:
$\langle O \rangle = \frac{\int \mathcal{D}q \, O e^{-S[q]}}{\int \mathcal{D}q \, e^{-S[q]}}$, where $S[q] = \int_0^\beta dq \, \left(\frac{1}{2}(\frac{dq}{d\tau})^2 + U(q)\right)$ is the Euclidean action. Here, $\beta$ is the inverse temperature, but for time-independent potentials, the temporal extent is often taken to be infinite for correlation functions. Let's assume we are working in a context where the time evolution is governed by a Hamiltonian, and the expectation values are thermal averages. However, the problem statement uses "t" which suggests real time evolution. For correlation functions in Euclidean spacetime, the integral is over $\tau \in [0, \beta]$. If we are considering real-time correlation functions, the path integral is over paths from $t=-\infty$ to $t=+\infty$ or similar. Given the notation $G_1(t)$ and $G_2(t_1, t_2)$, it's more likely we are dealing with real-time correlation functions, and the Euclidean action is used to define the path integral. In the context of quantum field theory, $e^{-S}$ often refers to $e^{-\int d^d x \mathcal{L}}$, where $\mathcal{L}$ is the Lagrangian density. For a single particle, this would be $e^{-\int dt (\frac{1}{2}(\frac{dq}{dt})^2 - U(q))}$. However, the problem statement explicitly mentions "Euclidean cubic anharmonic oscillator" and $U(q)$, which strongly suggests a Euclidean formulation. Let's assume the path integral is over paths $q(\tau)$ from $\tau=-\infty$ to $\tau=+\infty$ with the measure $\mathcal{D}q$ and the action $S[q] = \int_{-\infty}^{\infty} d\tau \left(\frac{1}{2}(\frac{dq}{d\tau})^2 + U(q)\right)$.

The generating functional for correlation functions is given by:
$Z[J] = \int \mathcal{D}q \, e^{-S[q] + \int d\tau J(\tau) q(\tau)}$
Then $\langle q(\tau_1) \dots q(\tau_n) \rangle = \frac{1}{Z[0]} \frac{\delta^n Z[J]}{\delta J(\tau_1) \dots \delta J(\tau_n)} \bigg|_{J=0}$.
