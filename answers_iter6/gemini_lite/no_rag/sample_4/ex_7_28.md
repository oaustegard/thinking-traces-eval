**Understanding the Problem**

We are given a potential for a particle, $U(q) = \frac{m^2 q^2}{2} - \frac{g q^3}{6}$. This is a Euclidean version of a cubic anharmonic oscillator. We need to calculate two quantities:
1.  The 1-point function $\langle q(t) \rangle$, which represents the average value of the position at time $t$, up to the leading order in the coupling constant $g$.
2.  The connected 2-point function $\langle q(t_1) q(t_2) \rangle_c$, which represents the correlation between the position at two different times, up to order $g^2$.

The notation $\langle \dots \rangle$ typically refers to the thermal average or, in a quantum field theory context, the vacuum expectation value. Since we are dealing with a potential and correlation functions, this problem likely originates from a statistical mechanics or quantum field theory context. We will assume a quantum mechanical framework where these correlation functions are defined via path integrals or perturbation theory.

**Methodology: Perturbation Theory**

The potential $U(q)$ can be split into a "free" part and an "interaction" part:
$U_0(q) = \frac{m^2 q^2}{2}$ (harmonic oscillator)
$U_{int}(q) = -\frac{g q^3}{6}$ (anharmonic perturbation)

We can use perturbation theory to calculate the correlation functions. The generating functional for correlation functions in Euclidean spacetime is given by:
$Z[J] = \int \mathcal{D}q \exp\left(-\int_0^\beta d\tau \left( \frac{1}{2}(\partial_\tau q)^2 + U(q) - J(\tau)q \right) \right)$
where $\beta$ is the inverse temperature (or Euclidean time duration). For correlation functions, we often consider the vacuum expectation value, which can be obtained by setting $J(\tau) = 0$.

The correlation functions can be computed as derivatives of the logarithm of the generating functional with respect to the source $J$. However, a more direct approach for calculating $\langle q(t) \rangle$ and $\langle q(t_1) q(t_2) \rangle_c$ is to use Wick's theorem and Feynman diagrams in perturbation theory.

The average of an observable $\mathcal{O}(q)$ is given by:
$\langle \mathcal{O}(q) \rangle = \frac{\int \mathcal{D}q \mathcal{O}(q) e^{-S[q]}}{\int \mathcal{D}q e^{-S[q]}}$, where $S[q] = \int d\tau (\frac{1}{2}(\partial_\tau q)^2 + U(q))$.

In perturbation theory, we expand $e^{-S[q]} = e^{-S_0[q]} e^{-S_{int}[q]}$ where $S_0[q] = \int d\tau \frac{1}{2}(\partial_\tau q)^2$ and $S_{int}[q] = \int d\tau (-\frac{g q^3}{6})$.

The average of $q(t)$ can be written as:
$\langle q(t) \rangle = \frac{\langle q(t) e^{-S_{int}[q]} \rangle_0}{\langle e^{-S_{int}[q]} \rangle_0}$, where $\langle \dots \rangle_0$ denotes the average with respect to the free theory (harmonic oscillator).

**Part (i): Leading term of the 1-point function $G_1(t) := \langle q(t) \rangle$ with respect to $g$.**

We need to expand $\langle q(t) \rangle$ in powers of $g$.
$\langle q(t) \rangle = \langle q(t) (1 - \frac{g}{6} \int d\tau' q(\tau')^3 + \dots) \rangle_0 / (1 - \frac{g}{6} \langle \int d\tau' q(\tau')^3 \rangle_0 + \dots)$

To the leading order in $g$ (i.e., order $g^1$), we have:
$\langle q(t) \rangle \approx \langle q(t) (1 - \frac{g}{6} \int d\tau' q(\tau')^3) \rangle_0$
$\langle q(t) \rangle \approx \langle q(t) \rangle_0 - \frac{g}{6} \langle q(t) \int d\tau' q(\tau')^3 \rangle_0$

In the free theory (harmonic oscillator), the average of $q$ is zero due to symmetry: $\langle q(\tau) \rangle_0 = 0$. This is because the potential $U_0(q) = \frac{m^2 q^2}{2}$ is an even function of $q$, and the path integral measure $\mathcal{D}q$ is symmetric.

Therefore, the leading term of $\langle q(t) \rangle$ with respect to $g$ is zero.
