**Part 1: 1-loop contribution to log(Z/Z_0)**

We are given the action $S(x) = \frac{x^2}{2} - g\left(\frac{x^2}{2} + \frac{x^3}{6}\right)$.
The partition function is $Z = \int dx \, e^{-S(x)/\hbar}$.
The free partition function is $Z_0$, which is obtained by setting $g=0$. So, $S_0(x) = \frac{x^2}{2}$.
$Z_0 = \int dx \, e^{-x^2/(2\hbar)}$. This is a Gaussian integral. To evaluate it, we can complete the square in the exponent. Let $x/\sqrt{\hbar} = y$, so $dx = \sqrt{\hbar} dy$.
$Z_0 = \int d(\sqrt{\hbar}y) \, e^{-y^2} = \sqrt{\hbar} \int dy \, e^{-y^2} = \sqrt{\hbar} \sqrt{\pi} = \sqrt{2\pi\hbar}$.

Now, let's consider the full partition function $Z$.
$S(x) = S_0(x) - g\left(\frac{x^2}{2} + \frac{x^3}{6}\right)$.
$Z = \int dx \, e^{-(S_0(x) - g(x^2/2 + x^3/6))/\hbar} = \int dx \, e^{-S_0(x)/\hbar} e^{g(x^2/2 + x^3/6)/\hbar}$.
$Z = Z_0 \left\langle e^{g(x^2/2 + x^3/6)/\hbar} \right\rangle_0$, where $\langle \cdot \rangle_0$ denotes the expectation value with respect to the free theory ($S_0$).

We want to compute $\log(Z/Z_0)$.
$\log(Z/Z_0) = \log \left\langle e^{g(x^2/2 + x^3/6)/\hbar} \right\rangle_0$.
For small $g$, we can expand the exponential:
$e^A \approx 1 + A + \frac{A^2}{2!} + \dots$
$\log(Z/Z_0) \approx \log \left\langle 1 + \frac{g}{\hbar}\left(\frac{x^2}{2} + \frac{x^3}{6}\right) + \frac{1}{2\hbar^2} g^2 \left(\frac{x^2}{2} + \frac{x^3}{6}\right)^2 + \dots \right\rangle_0$.

The 1-loop contribution comes from the term linear in $g$.

We need to compute $\langle x^2 \rangle_0$ and $\langle x^3 \rangle_0$. The free theory is a Gaussian distribution with probability density $P_0(x) = \frac{1}{Z_0} e^{-x^2/(2\hbar)}$.
The moments of a Gaussian $\mathcal{N}(0, \sigma^2)$ are:
$\langle x \rangle = 0$
$\langle x^2 \rangle = \sigma^2$
$\langle x^3 \rangle = 0$
$\langle x^4 \rangle = 3\sigma^4$

In our case, $e^{-x^2/(2\hbar)}$, so $\sigma^2 = \hbar$.
Therefore, $\langle x^2 \rangle_0 = \hbar$ and $\langle x^3 \rangle_0 = 0$.

$\log(Z/Z_0)^{(1)} = \frac{g}{\hbar} \left( \frac{1}{2}\hbar + \frac{1}{6}(0) \right) = \frac{g}{\hbar} \frac{\hbar}{2} = \frac{g}{2}$.

This is the 1-loop contribution to $\log(Z/Z_0)$.
