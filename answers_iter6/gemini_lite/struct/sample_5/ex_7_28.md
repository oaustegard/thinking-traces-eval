(a) Relevant examples:
Example 1: "Calculate the 1-particle irreducible 2-point function $\Sigma(t_1,t_2)$ for a quantum particle with potential $U(q) = \tfrac{m^2 q^2}{2} + \tfrac{g q^4}{24}$." This is relevant because it deals with calculating correlation functions and self-energy diagrams in quantum field theory, involving potentials with interaction terms. The steps involve setting up the action, identifying Feynman rules, and computing diagrams order by order.

Example 3: "EGF for Restricted-Valence Labeled Trees." This example is less directly relevant but touches upon generating functions and combinatorial structures arising from field theories, which can be related to correlation functions. However, Example 1 is much more directly applicable.

(b) Differences between relevant examples and the main problem:
The main problem concerns a *cubic* anharmonic oscillator ($U(q) = m^2 q^2 / 2 - g q^3 / 6$), whereas Example 1 deals with a *quartic* anharmonic oscillator ($U(q) = m^2 q^2 / 2 + g q^4 / 24$). This difference in the interaction term ($q^3$ vs. $q^4$) will lead to different Feynman rules and different types of diagrams contributing to the correlation functions. Specifically, the $q^3$ interaction vertex will have a different structure and contribution compared to the $q^4$ vertex. Also, the main problem asks for the 1-point function and the connected 2-point function up to $O(g^2)$, while Example 1 focuses on the 1-particle irreducible 2-point function (self-energy) for a different potential.

**Solution to the Main Problem:**

The potential is $U(q) = \tfrac{m^2 q^2}{2} - \tfrac{g q^3}{6}$. The Euclidean action is $S[q] = \int dt \left(\tfrac12 \dot q^2 + \tfrac{m^2}{2} q^2 - \tfrac{g}{6} q^3\right)$.
The free action is $S_0[q] = \int dt \left(\tfrac12 \dot q^2 + \tfrac{m^2}{2} q^2\right)$.
The free propagator is $G_0(t_1, t_2) = \langle q(t_1) q(t_2) \rangle_0 = \frac{1}{m} e^{-m|t_1 - t_2|}$.
The interaction term is $V(q) = -\tfrac{g}{6} q^3$. The Feynman rule for the $q^3$ vertex is $-g$.

**(i) Leading term of the 1-point function $G_1(t) := \langle q(t) \rangle$ with respect to $g$.**

The 1-point function is given by $\langle q(t) \rangle = \frac{\int \mathcal{D}q \, q(t) e^{-S[q]}}{\int \mathcal{D}q \, e^{-S[q]}}$.
Expanding $e^{-S[q]} = e^{-S_0[q] + \int dt \frac{g}{6} q^3} \approx e^{-S_0[q]} \left(1 + \int dt' \frac{g}{6} q(t')^3 + O(g^2)\right)$.
Then, $\langle q(t) \rangle = \frac{\int \mathcal{D}q \, q(t) e^{-S_0[q]} \left(1 + \int dt' \frac{g}{6} q(t')^3 + \dots\right)}{\int \mathcal{D}q \, e^{-S_0[q]} \left(1 + \int dt' \frac{g}{6} q(t')^3 + \dots\right)}$.
The denominator is $\approx \int \mathcal{D}q \, e^{-S_0[q]}$, as the $O(g)$ term is zero due to symmetry of the free theory.
The numerator is $\approx \int \mathcal{D}q \, q(t) e^{-S_0[q]} + \int \mathcal{D}q \, q(t) e^{-S_0[q]} \int dt' \frac{g}{6} q(t')^3$.
The first term is $\langle q(t) \rangle_0 = 0$ because the free theory is symmetric around $q=0$.

$\int dt' \frac{g}{6} \langle q(t) q(t')^3 \rangle_0$.
