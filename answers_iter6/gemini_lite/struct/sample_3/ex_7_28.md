The main problem concerns calculating correlation functions for a quantum field theory with a cubic interaction term.

(a) **Relevant Examples:**
*   **Example 1: 1-particle irreducible 2-point function $\Sigma(t_1,t_2)$ for a quantum particle with potential $U(q) = \tfrac{m^2 q^2}{2} + \tfrac{g q^4}{24}$.** This example is relevant because it deals with calculating correlation functions ($\Sigma$ is related to the full Green's function $G_2$ via Dyson's equation) in a quantum field theory with a polynomial interaction. Specifically, it demonstrates the use of Feynman diagrams and perturbation theory to compute these functions order by order in the coupling constant $g$. The calculation of $\Sigma^{(1)}$ and $\Sigma^{(2)}$ provides a template for how to approach diagrammatic calculations.

*   **Example 2: Etingof Exercise 2.20 (explicit Morse coordinates for a cubic).** This example is *not* directly relevant to the quantum field theory calculation. It deals with differential geometry and finding local coordinate transformations for a function, which is a different mathematical context.

*   **Example 3: EGF for Restricted-Valence Labeled Trees.** This example is also *not* directly relevant. It concerns combinatorial enumeration using exponential generating functions, which is a different branch of mathematics from quantum field theory perturbation theory.

Therefore, **Example 1** is the most relevant as it directly addresses the calculation of correlation functions in a perturbative quantum field theory.

(b) **Differences between Relevant Example and Main Problem:**
*   **Interaction Potential:** Example 1 has a quartic interaction ($q^4$), while the main problem has a cubic interaction ($-g q^3/6$). This difference is crucial because a cubic interaction is *odd* in $q$, leading to different properties for correlation functions, particularly the 1-point function. The quartic interaction in Example 1 is *even*, meaning the 1-point function in that theory would be zero due to symmetry.
*   **Correlation Function to Calculate:** Example 1 calculates the 1-particle irreducible 2-point function $\Sigma(t_1, t_2)$, which is related to the self-energy. The main problem asks for the 1-point function $G_1(t)$ and the connected 2-point function $G_2^c(t_1, t_2)$. While related, these are distinct quantities.
*   **Order of Calculation:** Example 1 calculates up to $O(g^2)$ for $\Sigma$. The main problem asks for $G_1$ to leading order in $g$ and $G_2^c$ up to $O(g^2)$ modulo $g^3$.

**Solution to the Main Problem:**

The potential is $U(q) = \tfrac{m^2 q^2}{2} - \tfrac{g q^3}{6}$. The Euclidean action is $S[q] = \int dt \left(\tfrac12 \dot q^2 + \tfrac{m^2}{2} q^2 - \tfrac{g}{6} q^3\right)$.
The free propagator is $G_0(t_1, t_2) = \langle q(t_1)q(t_2) \rangle_0 = \frac{1}{m} e^{-m|t_1-t_2|}$ (assuming mass $m$ and $\hbar=1$).
The Feynman rules are:
*   Propagator: $G_0(t_1, t_2)$
*   Vertex: $-g$ for each $q^3$ vertex.

**(i) Leading term of the 1-point function $G_1(t) = \langle q(t) \rangle$ with respect to $g$.**

The 1-point function is given by $\langle q(t) \rangle = \frac{\int \mathcal{D}q \, q(t) e^{-S[q]}}{\int \mathcal{D}q \, e^{-S[q]}}$.
In perturbation theory, this can be expanded using Feynman diagrams. The 1-point function is the sum of all connected diagrams with one external leg.

*   **Order $g^0$ (free theory):** $\langle q(t) \rangle_0 = 0$. This is because the free theory is invariant under $q \to -q$, and the integral of $q(t)$ over this symmetry is zero.
*   **Order $g^1$:** The only diagram is a single $q^3$ vertex connected to the external leg. This diagram is:
    $$ \langle q(t) \rangle^{(1)} = \int dt' \left\langle q(t) q(t') \right\rangle_0 (-g) \langle q(t')^2 \rangle_0 $$
    Wait, this is not how the 1-point function is calculated. The 1-point function $\langle q(t) \rangle$ is given by the sum of all 1PI diagrams with one external leg. Or, more directly, by applying the functional derivative to the partition function.
    Let $Z[J] = \int \mathcal{D}q \, e^{-S[q] + \int J(t)q(t)dt}$. Then $\langle q(t) \rangle = \frac{\delta \ln Z[J]}{\delta J(t)} \Big|_{J=0}$.
    $S[q] = S_0[q] + S_{int}[q]$, where $S_0[q] = \int (\tfrac12 \dot q^2 + \tfrac{m^2}{2} q^2) dt$ and $S_{int}[q] = -\int \tfrac{g}{6} q^3 dt$.
    $\ln Z[J] = \ln Z_0[J] + \ln \frac{Z[J]}{Z_0[J]}$.
    $\frac{\delta \ln Z[J]}{\delta J(t)} = \frac{\delta \ln Z_0[J]}{\delta J(t)} + \frac{\delta}{\delta J(t}} \ln \left\langle e^{\int J q' dt'} \right\rangle_0$
    $= \langle q(t) \rangle_0 + \frac{\delta}{\delta J(t)} \left\langle \sum_n \frac{1}{n!} (\int J q')^n \right\rangle_0$
    $= \langle q(t) \rangle_0 + \left\langle q(t) \sum_n \frac{1}{n!} (\int J q')^n \right\rangle_0 + \dots$
    At $J=0$, $\langle q(t) \rangle_0 = 0$.
    The first non-zero contribution to $\langle q(t) \rangle$ comes from diagrams with at least one vertex.
    The 1-point function is the sum of all connected diagrams with one external leg.
    The simplest diagram contributing to $\langle q(t) \rangle$ involves the $q^3$ vertex.
    The contribution to $\langle q(t) \rangle$ from one $q^3$ vertex is:
    $$ \langle q(t) \rangle^{(1)} = \int dt' \left\langle q(t) q(t') \right\rangle_0 \left(-\frac{g}{6}\right) \langle q(t')^2 \rangle_0 $$
    This is incorrect. The 1-point function is the sum of all connected diagrams with one external leg. The vertex is $-g q^3$. The external leg is $q$.
    So, a single vertex contributes to the 1-point function by having the external leg attach to one of the $q$'s in the vertex. The remaining two $q$'s in the vertex must be contracted with each other to form a loop.
    The diagram is:
    $$ \begin{tikzpicture}
      \draw (-1,0) node[left] {$t$} -- (0,0) node[circ] {};
      \draw (0,0) -- (1,1) node[circ] {};
      \draw (0,0) -- (1,-1) node[circ] {};
      \draw (0,0) -- (0.5,0) node[right] {$t'$};
      \draw (0.5,0) .. controls (0.7,0.2) and (0.7,-0.2) .. (0.5,0); % Loop
    \end{tikzpicture} $$
    The vertex weight is $-g$. The external leg is $q(t)$. The vertex is $-g q(t')^3/6$.
    The contribution to $\langle q(t) \rangle$ is:
    $$ \langle q(t) \rangle^{(1)} = \int dt' \left\langle q(t) q(t') \right\rangle_0 \left(-\frac{g}{6}\right) \langle q(t') q(t') \rangle_0 $$
    The term $\langle q(t') q(t') \rangle_0$ is the vacuum expectation value of $q^2$ in the free theory.
    $$ \langle q(t') q(t') \rangle_0 = G_0(t', t') = \frac{1}{m} e^{-m|t'-t'|} = \frac{1}{m} $$
    So,
    $$ \langle q(t) \rangle^{(1)} = \int dt' G_0(t, t') \left(-\frac{g}{6}\right) \frac{1}{m} $$
    $$ \langle q(t) \rangle^{(1)} = -\frac{g}{6m} \int dt' G_0(t, t') $$
    The integral $\int dt' G_0(t, t')$ is the integral of the propagator over all possible times. This integral diverges. This indicates that the vacuum is unstable or that we need to renormalize. However, for the *leading term* in $g$, we typically consider diagrams that are finite and non-zero.

    Let's reconsider the definition of $\langle q(t) \rangle$. It is the sum of all connected diagrams with one external leg.
    The $q^3$ vertex has weight $-g$.
    Diagram: External leg at $t$, connected to one leg of a $q^3$ vertex at $t'$. The other two legs of the vertex are contracted.
    $$ -g \int dt' q(t) q(t') q(t') $$
    The expectation value is:
    $$ \langle q(t) \rangle^{(1)} = -g \int dt' \langle q(t) q(t') \rangle_0 \langle q(t') q(t') \rangle_0 $$
    $$ \langle q(t) \rangle^{(1)} = -g \int dt' G_0(t, t') \frac{1}{m} $$
    This integral $\int dt' G_0(t, t')$ diverges. This divergence is related to the fact that the cubic term tends to push the field away from $q=0$.

    However, the question asks for the *leading term* with respect to $g$. This typically means the lowest order in $g$ that is non-zero.
    The expectation value of $q$ in the presence of the interaction $-g q^3/6$ can be computed by considering the derivative of the free partition function with respect to the source $J$.
    $\langle q(t) \rangle = \frac{\delta \ln Z[J]}{\delta J(t)}|_{J=0}$.
    $Z[J] = Z_0[J] \left\langle e^{-\int \frac{g}{6} q^3 dt} \right\rangle_0$.
    $\ln Z[J] = \ln Z_0[J] + \ln \left\langle e^{-\int \frac{g}{6} q^3 dt} \right\rangle_0$.
    $\frac{\delta \ln Z[J]}{\delta J(t)} = \frac{\delta \ln Z_0[J]}{\delta J(t)} + \frac{1}{\left\langle e^{-\int \frac{g}{6} q'^3 dt'} \right\rangle_0} \frac{\delta}{\delta J(t)} \left\langle e^{-\int \frac{g}{6} q'^3 dt'} \right\rangle_0$.
    At $J=0$:
    $\langle q(t) \rangle = \langle q(t) \rangle_0 + \frac{1}{\left\langle 1 - \int \frac{g}{6} q'^3 dt' + \dots \right\rangle_0} \left\langle (-\int \frac{g}{6} q'^3 dt') q(t) \right\rangle_0$
    $\langle q(t) \rangle = 0 + \frac{1}{1} \left\langle -\frac{g}{6} \int dt' q'(t')^3 q(t) \right\rangle_0$
    $\langle q(t) \rangle^{(1)} = -\frac{g}{6} \int dt' \langle q(t) q(t')^3 \rangle_0$.
    Using Wick's theorem for $\langle q(t) q(t')^3 \rangle_0$:
    $\langle q(t) q(t')^3 \rangle_0 = \langle q(t) q(t') \rangle_0 \langle q(t') q(t') \rangle_0 + \langle q(t) q(t') \rangle_0 \langle q(t') q(t') \rangle_0 + \langle q(t) q(t') \rangle_0 \langle q(t') q(t') \rangle_0$
    $= 3 \langle q(t) q(t') \rangle_0 \langle q(t') q(t') \rangle_0$
    $= 3 G_0(t, t') \frac{1}{m}$
    So, $\langle q(t) \rangle^{(1)} = -\frac{g}{6} \int dt' \left( 3 G_0(t, t') \frac{1}{m} \right) = -\frac{g}{2m} \int dt' G_0(t, t')$.
    This integral is indeed divergent. This indicates that the vacuum expectation value of $q$ itself is shifted due to the cubic interaction.

    In such cases, the usual interpretation of "leading term" refers to the lowest order diagram that contributes. If all such diagrams are divergent, it implies a need for renormalization. A common approach is to consider the contribution from loop diagrams that renormalize parameters.

    However, if we consider the *effective potential* approach, the cubic term can induce a non-zero mean field.
    Let $\langle q \rangle = v$. Expanding $U(q)$ around $v$: $U(v+\delta q) \approx U(v) + U'(v)\delta q + \frac{1}{2}U''(v)\delta q^2 + \frac{1}{6}U'''(v)\delta q^3$.
    $U'(q) = m^2 q - \frac{g}{2} q^2$.
    $U''(q) = m^2 - gq$.
    $U'''(q) = -g$.
    If we set the expectation value $v$ such that the linear term in the expansion of the *effective potential* is zero, this gives the vacuum expectation value.
    The one-loop correction to the potential is $\frac{1}{2} \int \frac{d^d p}{(2\pi)^d} \ln(p^2 + m^2 - gv)$.
    The effective potential $V_{eff}(v) = \frac{m^2 v^2}{2} - \frac{g v^3}{6} + \text{1-loop corrections}$.
    Setting $\frac{dV_{eff}}{dv} = 0$ to find the vacuum expectation value.
    $\frac{dV_{eff}}{dv} = m^2 v - \frac{g v^2}{2} + \text{corrections}$.
    At leading order in $g$, the 1-loop correction to the potential is $\frac{1}{2} \int d\omega \ln(\omega^2 + m^2)$. This is independent of $v$.
    So, at leading order in $g$, the equation for the vacuum expectation value $v$ is:
    $m^2 v - \frac{g v^2}{2} = 0$.
    This gives $v(m^2 - \frac{g}{2}v) = 0$.
    Solutions are $v=0$ and $v = \frac{2m^2}{g}$.
    The $v=0$ solution corresponds to the minimum of the original potential $U(q)$. However, the cubic term breaks the symmetry, and $v = \frac{2m^2}{g}$ is a stable minimum of the effective potential when $g$ is small and positive.

    If we interpret $\langle q(t) \rangle$ as the vacuum expectation value, then the leading order term is indeed non-zero. However, the question asks for the leading term *with respect to $g$*, implying an expansion of the connected diagram.

    Let's assume the question implies the lowest order non-zero contribution from a connected diagram. The divergence of the single-vertex diagram suggests that $G_1(t)$ is not simply given by this diagram.

    A different perspective: The 1-point function is the sum of all connected diagrams with one external leg.
    Diagrams for $\langle q(t) \rangle$:
    *   Order $g^0$: $\langle q(t) \rangle_0 = 0$.
    *   Order $g^1$: A single $q^3$ vertex with one leg attached to the external point, and the other two legs contracted. This is the diagram calculated above, which diverges.
    *   Order $g^2$: The external leg is connected to a $q^3$ vertex. The remaining two legs of this vertex are connected by a propagator $G_0$. The third leg of the vertex is contracted with itself to form a loop. This is not a 1-point function diagram.

    The $q^3$ term means that the potential is not symmetric. For a potential $V(q)$, $\langle q \rangle = \frac{\int q e^{-V(q)}}{\int e^{-V(q)}}$. If $V(q)$ is not symmetric, $\langle q \rangle$ can be non-zero.
    For $U(q) = \tfrac{m^2 q^2}{2} - \tfrac{g q^3}{6}$, the minimum is not at $q=0$ for $g \ne 0$.
    The equation $U'(q) = m^2 q - g q^2/2 = 0$ gives $q=0$ or $q = 2m^2/g$.
    The second derivative $U''(q) = m^2 - gq$. At $q=0$, $U''(0)=m^2 > 0$. At $q=2m^2/g$, $U''(2m^2/g) = m^2 - g(2m^2/g) = m^2 - 2m^2 = -m^2 < 0$.
    This means $q=0$ is a local maximum and $q=2m^2/g$ is a local minimum for the potential. The vacuum is at $q \approx 2m^2/g$.

    However, the question is about perturbation theory around $q=0$.
    The leading term of the 1-point function $\langle q(t) \rangle$ with respect to $g$ is the lowest order in $g$ that is non-zero.
    The diagram with one $q^3$ vertex and two contracted legs is:
    $$ \langle q(t) \rangle^{(1)} = -g \int dt' \langle q(t) q(t') \rangle_0 \langle q(t') q(t') \rangle_0 $$
    As calculated, this is $-g \int dt' G_0(t, t') \frac{1}{m}$. This integral diverges.

    If we are asked for the leading *finite* term, we might need to consider higher orders or renormalization. However, "leading term" usually implies the lowest order non-zero term in the expansion. The divergence suggests that the parameter $m$ or the "zero point" of $q$ is not properly defined.

    Let's assume the question implies the contribution from the simplest connected diagram that generates a non-zero expectation value. The cubic term breaks parity, so the expectation value of $q$ can be non-zero.
    The diagram corresponding to $-\frac{g}{2m} \int dt' G_0(t, t')$ is the lowest order contribution.
    The integral $\int dt' G_0(t, t') = \int dt' \frac{1}{m} e^{-m|t'-t'|} = \int dt' \frac{1}{m}$. This integral is $\frac{1}{m} T$, where $T$ is the total time. If we are in a finite time interval, it's finite. If it's infinite time, it diverges. In QFT, we usually consider infinite time.

    A more rigorous way to see the leading term for $\langle q(t) \rangle$:
    Consider the equation of motion for $q(t)$:
    $\ddot q + m^2 q - g q^2 = 0$.
    Taking expectation values: $\langle \ddot q \rangle + m^2 \langle q \rangle - g \langle q^2 \rangle = 0$.
    If we assume $\langle q \rangle$ is constant in time, then $\langle \ddot q \rangle = 0$.
    $m^2 \langle q \rangle - g \langle q^2 \rangle = 0$.
    $\langle q \rangle = \frac{g}{m^2} \langle q^2 \rangle$.
    In the free theory, $\langle q^2 \rangle_0 = \frac{1}{m}$.
    So, $\langle q \rangle^{(1)} = \frac{g}{m^2} \langle q^2 \rangle_0 = \frac{g}{m^2} \frac{1}{m} = \frac{g}{m^3}$.
    This approach is heuristic.

    Let's go back to the Feynman diagram:
    $\langle q(t) \rangle^{(1)} = -g \int dt' \langle q(t) q(t') \rangle_0 \langle q(t') q(t') \rangle_0$.
    The term $\langle q(t') q(t') \rangle_0 = G_0(t', t')$ is problematic because the propagator is defined for $t_1 \ne t_2$.
    However, if we consider the definition of the Green's function $G_0(t_1, t_2) = \frac{1}{m} e^{-m|t_1-t_2|}$, then $G_0(t,t) = \frac{1}{m}$.
    The integral is $\int dt' G_0(t, t') = \int_{-\infty}^{\infty} \frac{1}{m} e^{-m|t-t'|} dt'$. This integral is $\frac{1}{m} \left( \int_{-\infty}^t e^{m(t-t')} dt' + \int_t^\infty e^{-m(t-t')} dt' \right) = \frac{1}{m} \left( \frac{1}{m} + \frac{1}{m} \right) = \frac{2}{m^2}$.
    So, $\langle q(t) \rangle^{(1)} = -g \left(\frac{2}{m^2}\right) \frac{1}{m} = -\frac{2g}{m^3}$.
    This is a finite result if we interpret the integral of the propagator correctly.
    The leading term of the 1-point function is the order $g^1$ contribution.

    **(i) Answer:** The leading term of the 1-point function is $\langle q(t) \rangle^{(1)} = -\frac{2g}{m^3}$.

**(ii) Connected 2-point function $G_2^c(t_1, t_2) = \langle q(t_1) q(t_2) \rangle_c$ modulo $g^3$.**

The connected 2-point function is the sum of all connected diagrams with two external legs.
$G_2^c(t_1, t_2) = G_0(t_1, t_2) + \text{1PI diagrams with 2 external legs} + \text{non-1PI connected diagrams}$.
The 1PI diagrams with 2 external legs are the self-energy diagrams $\Sigma(t_1, t_2)$.
The full Green's function $G_2(t_1, t_2) = \langle q(t_1) q(t_2) \rangle = (G_0^{-1} - \Sigma)^{-1}$.
The connected part is $G_2^c = G_2 - G_0$.

We need to compute $G_2^c$ up to $O(g^2)$.
$G_2^c = \Sigma + \Sigma G_0 \Sigma + \dots$
Up to $O(g^2)$, $G_2^c$ is the sum of $G_0$ and all 1PI diagrams with two external legs up to order $g^2$.

*   **Order $g^0$:** The free propagator is $G_0(t_1, t_2) = \frac{1}{m} e^{-m|t_1-t_2|}$. This is the connected 2-point function in the free theory.

*   **Order $g^1$ (1PI diagrams):** There is one diagram: a single $q^3$ vertex connecting the two external legs.
    $$ \Sigma^{(1)}(t_1, t_2) = -g \int dt' \langle q(t_1) q(t') \rangle_0 \langle q(t_2) q(t') \rangle_0 $$
    $$ \Sigma^{(1)}(t_1, t_2) = -g \int dt' G_0(t_1, t') G_0(t_2, t') $$
    $$ \Sigma^{(1)}(t_1, t_2) = -g \int_{-\infty}^{\infty} dt' \left(\frac{1}{m} e^{-m|t_1-t'|}\right) \left(\frac{1}{m} e^{-m|t_2-t'|}\right) $$
    Let's assume $t_1 < t_2$.
    The integral is over $dt'$. The integrand is non-zero when $t'$ is between $t_1$ and $t_2$.
    $\int_{-\infty}^{\infty} dt' e^{-m|t_1-t'|} e^{-m|t_2-t'|} = \int_{-\infty}^{t_1} e^{m(t_1-t')} e^{-m(t_2-t')} dt' + \int_{t_1}^{t_2} e^{-m(t_1-t')} e^{-m(t_2-t')} dt' + \int_{t_2}^{\infty} e^{-m(t_1-t')} e^{m(t_2-t')} dt'$.
    This is not correct. The exponents are:
    For $t'<t_1$: $e^{-m(t_1-t')} e^{-m(t_2-t')} = e^{-m(t_1-t'+t_2-t')} = e^{m(t'-t_1)} e^{m(t'-t_2)}$ - This is wrong.
    Let's split the integral:
    $\int_{-\infty}^{\infty} dt' e^{-m|t_1-t'|} e^{-m|t_2-t'|}$.
    If $t_1 < t_2$:
    Case 1: $t' < t_1$. $|t_1-t'| = t_1-t'$, $|t_2-t'| = t_2-t'$. Integral: $\int_{-\infty}^{t_1} e^{-m(t_1-t')} e^{-m(t_2-t')} dt'$. This is not correct.
    The exponents are $|t_1-t'|$ and $|t_2-t'|$.
    Let $t_1 < t_2$.
    If $t' < t_1$: $|t_1-t'| = t_1-t'$, $|t_2-t'| = t_2-t'$. Term: $e^{-m(t_1-t')} e^{-m(t_2-t')}$. This is still wrong.
    Let's use the fact that $\int_{-\infty}^{\infty} e^{-a|x-y|} e^{-b|z-y|} dy = \frac{e^{-(|x-z|(a+b))}}{a+b}$ if $a,b>0$. No, this is not correct.

    Let's perform the integral directly: $\int_{-\infty}^{\infty} e^{-m|t_1-t'|} e^{-m|t_2-t'|} dt'$.
    Assume $t_1 < t_2$.
    If $t' < t_1$: $|t_1-t'| = t_1-t'$, $|t_2-t'| = t_2-t'$. Term: $e^{-m(t_1-t')}e^{-m(t_2-t')}$.
    If $t_1 < t' < t_2$: $|t_1-t'| = t'-t_1$, $|t_2-t'| = t_2-t'$. Term: $e^{-m(t'-t_1)}e^{-m(t_2-t')}$.
    If $t' > t_2$: $|t_1-t'| = t'-t_1$, $|t_2-t'| = t'-t_2$. Term: $e^{-m(t'-t_1)}e^{-m(t'-t_2)}$.

    Let's use the formula: $\int_{-\infty}^{\infty} e^{-a|x-y|} e^{-b|z-y|} dy = \frac{e^{-|x-z|(a+b)}}{a+b}$ is incorrect.
    The correct integral is:
    $\int_{-\infty}^{\infty} e^{-m|t_1-t'|} e^{-m|t_2-t'|} dt'$.
    Let $u=t'-t_1$, $v=t'-t_2$.
    Let's consider the Fourier transform of the propagator: $\tilde{G}_0(p) = \frac{1}{p^2+m^2}$.
    The integral $\int_{-\infty}^{\infty} G_0(t_1, t') G_0(t_2, t') dt'$ corresponds to the convolution of the propagator with itself.
    In momentum space, this is $\tilde{G}_0(p)^2$.
    $\tilde{G}_0(p)^2 = \left(\frac{1}{p^2+m^2}\right)^2$.
    The inverse Fourier transform of $\left(\frac{1}{p^2+m^2}\right)^2$ is $\frac{1}{2m} (1+m|t|) e^{-m|t|}$.
    So, $\int_{-\infty}^{\infty} G_0(t_1, t') G_0(t_2, t') dt' = \frac{1}{2m} (1+m|t_1-t_2|) e^{-m|t_1-t_2|}$.
    Therefore, $\Sigma^{(1)}(t_1, t_2) = -g \frac{1}{m^2} \left( \frac{1}{2m} (1+m|t_1-t_2|) e^{-m|t_1-t_2|} \right)$.
    $\Sigma^{(1)}(t_1, t_2) = -\frac{g}{2m^3} (1+m|t_1-t_2|) e^{-m|t_1-t_2|}$.

    This term is a correction to the propagator. It is a non-local function of time.

*   **Order $g^2$ (1PI diagrams):** There are two types of 1PI diagrams with two external legs at order $g^2$:
    1.  A "bubble" diagram: Two $q^3$ vertices, connected by a propagator $G_0$. Each vertex has one leg connected to an external point, and the other two contracted.
        $$ \Sigma^{(2)}_{\text{bubble}}(t_1, t_2) = \int dt' \int dt'' \left\langle q(t_1) q(t') \right\rangle_0 \left(-\frac{g}{6}\right) \langle q(t') q(t') \rangle_0 \left(-\frac{g}{6}\right) \langle q(t'') q(t'') \rangle_0 \left\langle q(t_2) q(t'') \right\rangle_0 $$
        This is not a 1PI diagram. A 1PI diagram with two external legs and two vertices must have the vertices connected by propagators, and the external legs attached to the vertices.

    Let's draw the 1PI diagrams for $\Sigma$ up to $g^2$:
    *   $\Sigma^{(0)} = 0$.
    *   $\Sigma^{(1)}$: Single $q^3$ vertex connecting the two external legs. This is the diagram calculated above.
    *   $\Sigma^{(2)}$:
        *   Diagram 1: Two $q^3$ vertices. One external leg attached to vertex 1, the other to vertex 2. The remaining two legs of vertex 1 are contracted. The remaining two legs of vertex 2 are contracted. The two vertices are connected by a $G_0$ propagator.
            $$ \Sigma^{(2)}_{1}(t_1, t_2) = \int dt' \int dt'' \langle q(t_1) q(t') \rangle_0 (-g) \langle q(t') q(t'') \rangle_0 (-g) \langle q(t'') q(t_2) \rangle_0 \times (\text{contractions}) $$
