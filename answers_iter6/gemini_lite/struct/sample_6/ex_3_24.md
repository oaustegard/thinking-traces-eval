**Part 2: Compute the number of labeled $n$-vertex 1-loop graphs with 1-valent and 3-valent vertices only (be careful with double edges and self-loops).**

This is a combinatorial problem. We are counting labeled graphs with $n$ vertices.
The graph is a 1-loop graph, meaning it contains exactly one cycle.
The vertices can only be 1-valent or 3-valent.

Let $V$ be the set of vertices, $|V|=n$.
Let $v_1$ be the number of 1-valent vertices, and $v_3$ be the number of 3-valent vertices.
So, $v_1 + v_3 = n$.

By the handshaking lemma, the sum of degrees of all vertices is equal to twice the number of edges ($2E$).
Sum of degrees = $v_1 \cdot 1 + v_3 \cdot 3 = v_1 + 3v_3$.
So, $2E = v_1 + 3v_3$.

Since it's a 1-loop graph, it has exactly one cycle. A graph with one cycle has $E = V$.
So, $2V = v_1 + 3v_3$.
Substituting $V=n$: $2n = v_1 + 3v_3$.

We have a system of two equations:
1.  $v_1 + v_3 = n$
2.  $v_1 + 3v_3 = 2n$

Subtracting equation (1) from equation (2):
$(v_1 + 3v_3) - (v_1 + v_3) = 2n - n$
$2v_3 = n$
$v_3 = n/2$.

This implies that $n$ must be an even number. If $n$ is odd, there are no such graphs.
If $n$ is even, then $v_1 = n - v_3 = n - n/2 = n/2$.
So, for a 1-loop graph with only 1-valent and 3-valent vertices, we must have $v_1 = n/2$ and $v_3 = n/2$. This means $n$ must be even.

The number of edges is $E = V = n$.
The number of edges is also $E = \frac{v_1 + 3v_3}{2} = \frac{n/2 + 3(n/2)}{2} = \frac{n/2 + 3n/2}{2} = \frac{4n/2}{2} = \frac{2n}{2} = n$. This is consistent.

If a vertex is not on the cycle, it must be 1-valent.
