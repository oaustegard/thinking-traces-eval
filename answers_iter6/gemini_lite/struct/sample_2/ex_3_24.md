**Part 1: 1-Loop Contribution to $\log(Z/Z_0)$**

The partition function is $Z = \int dx \, e^{-S(x)/\hbar}$.
The free partition function is $Z_0 = \int dx \, e^{-S_0(x)/\hbar}$, where $S_0(x) = \frac{x^2}{2}$.
We are interested in $\log(Z/Z_0)$.
Using perturbation theory, $\log(Z/Z_0) \approx \langle S_{\text{int}} - S_{\text{free,int}} \rangle_0$, where $S_{\text{int}} = -g(\frac{x^2}{2} + \frac{x^3}{6})$ and $S_{\text{free,int}} = 0$ (since $S_0(x) = x^2/2$ is the free action, not an interaction term added on top).
So, $\log(Z/Z_0) \approx \langle -g(\frac{x^2}{2} + \frac{x^3}{6}) \rangle_0$.

The expectation value is taken with respect to the free theory defined by $S_0(x) = x^2/2$. The partition function for the free theory is $Z_0 = \int dx \, e^{-x^2/(2\hbar)}$.
The generating function for correlation functions in the free theory is $W_0(J) = \log \int dx \, e^{-\frac{x^2}{2\hbar} + Jx} = \log \left( \sqrt{2\pi\hbar} e^{\hbar J^2/2} \right) = \frac{1}{2}\log(2\pi\hbar) + \frac{\hbar J^2}{2}$.
The connected correlation function is $\langle x^n \rangle_0 = \frac{d^n W_0(J)}{dJ^n} \Big|_{J=0}$.
$\langle x \rangle_0 = 0$.
$\langle x^2 \rangle_0 = \frac{d}{dJ} (\hbar J) \Big|_{J=0} = \hbar$.
$\langle x^3 \rangle_0 = 0$.

So, the 1-loop contribution to $\log(Z/Z_0)$ is:
$\log(Z/Z_0)^{(1)} = \langle -g(\frac{x^2}{2} + \frac{x^3}{6}) \rangle_0 = -g \left( \frac{1}{2} \langle x^2 \rangle_0 + \frac{1}{6} \langle x^3 \rangle_0 \right)$
$\log(Z/Z_0)^{(1)} = -g \left( \frac{1}{2} \hbar + \frac{1}{6} \cdot 0 \right) = -\frac{g\hbar}{2}$.

Alternatively, using the 1-loop effective action approach from Example 2:

The 1-loop effective action $S_1(x_c)$ is given by the sum of 1PI diagrams.
The free propagator is derived from $S_0(x) = x^2/2$. The equation of motion is $x=0$.
Let's re-evaluate the free theory. The problem statement implies $Z_0$ is for $S_0(x) = x^2/2$.

The generating function for connected correlation functions in the free theory is $W_0(J) = \log \int dx \, e^{-x^2/(2\hbar) + Jx}$.
Let $y = x/\sqrt{\hbar}$. $dy = dx/\sqrt{\hbar}$.
$W_0(J) = \log \int dy \sqrt{\hbar} \, e^{-y^2/2 + J\sqrt{\hbar}y} = \log \sqrt{\hbar} + \log \int dy \, e^{-y^2/2 + J\sqrt{\hbar}y}$.

$= -g \left( \frac{1}{2} \hbar + \frac{1}{6} \cdot 0 \right) = -\frac{g\hbar}{2}$.
