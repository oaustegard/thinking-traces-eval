**Specific Differences for the Main Problem:**

1.  **Potential:** The main problem is $U(q) = \tfrac{m^2 q^2}{2} - \tfrac{g q^3}{6}$. This is a **cubic potential**. The Euclidean action is $S[q] = \int dt (\tfrac12 \dot q^2 + \tfrac{m^2}{2} q^2 - \tfrac{g}{6} q^3)$.
    *   The $q^3$ term means vertices in the Feynman expansion will be 3-valent, with a coupling strength of $-g/6$ (or $g/6$ if we define $S_{int} = \int g q^3/6$). Let's use $S_{int} = -g q^3/6$. The Feynman rule for a $q^3$ vertex will be $-g/6$.
    *   Crucially, the cubic potential is **odd** under $q \to -q$. This has significant implications for correlation functions.

2.  **Target Quantities:**
    *   **(i) 1-point function:** $\langle q(t) \rangle$.
    *   **(ii) Connected 2-point function:** $\langle q(t_1) q(t_2) \rangle_c$.

## Solution to the Main Problem

**Setup:**
Euclidean action: $S[q] = \int dt \left(\tfrac12 \dot q^2 + \tfrac{m^2}{2} q^2 - \tfrac{g}{6} q^3\right)$.
Free propagator: $G_0(t_1, t_2) = \langle q(t_1) q(t_2) \rangle_0 = \frac{1}{2m} e^{-m|t_1 - t_2|}$.
Feynman rule for the cubic vertex: $-g/6$.

**(i) Leading term of the 1-point function $\langle q(t) \rangle$ with respect to $g$.**

The 1-point function $\langle q(t) \rangle$ can be computed using Feynman diagrams. A 1-point function corresponds to diagrams with one external leg (which is then amputated for the *connected* 1-point function, but here we mean the full correlation function). The simplest diagram for $\langle q(t) \rangle$ is just the external leg itself, but this is the free part. To get contributions involving $g$, we need to attach vertices.

The lowest order contribution to $\langle q(t) \rangle$ involving $g$ will come from diagrams with one $q^3$ vertex.
Diagram: A single $q^3$ vertex with one external leg.
The vertex has 3 half-edges. One connects to the external leg at time $t$. The other two must contract with each other to form a loop. This is a "tadpole" diagram attached to an external leg.

The contribution from this diagram is:
$\langle q(t) \rangle^{(g)} = \int dt_v \left(-\frac{g}{6}\right) \langle q(t) q(t_v) q(t_v) \rangle_0$.
The correlation function $\langle q(t) q(t_v) q(t_v) \rangle_0$ in the free theory is computed using Wick contractions. We have two $q(t_v)$ fields. One contracts with $q(t)$, and the other contracts with itself.
$\langle q(t) q(t_v) q(t_v) \rangle_0 = \langle q(t) q(t_v) \rangle_0 \langle q(t_v) \rangle_0 + \text{contractions involving } q(t_v) \text{ with itself}$.
The free theory is Gaussian, so $\langle q(t_v) \rangle_0 = 0$.
Thus, the only non-zero contraction is:
$\langle q(t) q(t_v) q(t_v) \rangle_0 = G_0(t, t_v) G_0(t_v, t_v)$.

So, the contribution is:
$\langle q(t) \rangle^{(g)} = \int dt_v \left(-\frac{g}{6}\right) G_0(t, t_v) G_0(t_v, t_v)$.

We know $G_0(t_v, t_v) = \langle q(t_v) q(t_v) \rangle_0 = \frac{1}{2m}$.
And $G_0(t, t_v) = \frac{1}{2m} e^{-m|t - t_v|}$.

$\langle q(t) \rangle^{(g)} = \int_{-\infty}^{\infty} dt_v \left(-\frac{g}{6}\right) \left(\frac{1}{2m} e^{-m|t - t_v|}\right) \left(\frac{1}{2m}\right)$
$\langle q(t) \rangle^{(g)} = -\frac{g}{24m^2} \int_{-\infty}^{\infty} dt_v \, e^{-m|t - t_v|}$.

The integral $\int_{-\infty}^{\infty} e^{-m|t - t_v|} dt_v$ is divergent. This indicates that the naive application of perturbation theory to the 1-point function of an odd potential leads to infinities, which are handled by renormalization. However, the question asks for the *leading term with respect to g*. This typically implies the first non-zero contribution in the perturbative expansion.

Alternatively, consider the symmetry of the potential. $U(q) = \tfrac{m^2 q^2}{2} - \tfrac{g q^3}{6}$. This potential is not symmetric around $q=0$. For a symmetric potential ($q^4$), the 1-point function is zero at all orders. For an asymmetric potential like this cubic one, the expectation value $\langle q(t) \rangle$ is generally non-zero and requires renormalization.

Let's consider the structure of the problem: we are looking for the *leading term*.
The generating functional for connected correlation functions is $W[J] = \ln Z[J]$, where $Z[J] = \int \mathcal{D}q \, e^{-S[q] + \int J(t) q(t) dt}$.
The $n$-point connected correlation function is given by $\langle q(t_1) \cdots q(t_n) \rangle_c = \frac{\delta^n W[J]}{\delta J(t_1) \cdots \delta J(t_n)} \Big|_{J=0}$.

For the 1-point function:
$\langle q(t) \rangle = \frac{\delta W[J]}{\delta J(t)} \Big|_{J=0}$.
$W[J] = \ln \int \mathcal{D}q \, \exp\left(-\int (\tfrac12\dot q^2 + \tfrac{m^2}{2} q^2 - \tfrac{g}{6}q^3) dt + \int J q dt\right)$.
Let $S_0 = \int (\tfrac12\dot q^2 + \tfrac{m^2}{2} q^2) dt$ and $S_{int} = -\int \tfrac{g}{6}q^3 dt$.
$Z[J] = \int \mathcal{D}q \, e^{-S_0 - S_{int} + \int J q dt}$.
$Z[J] = Z_0 \int \mathcal{D}q \, e^{-S_0} e^{-S_{int} + \int J q dt} / \int \mathcal{D}q \, e^{-S_0}$
$Z[J] = Z_0 \langle e^{-S_{int} + \int J q dt} \rangle_0$.
$W[J] = \ln Z_0 + \ln \langle e^{-S_{int} + \int J q dt} \rangle_0$.
$W[J] = \ln Z_0 + \ln \left\langle \exp\left(\int J q dt\right) \exp\left(\int \frac{g}{6} q^3 dt\right) \right\rangle_0$.

Expanding the exponential $\exp(\int \frac{g}{6} q^3 dt)$:
$\exp\left(\int \frac{g}{6} q^3 dt\right) = 1 + \int \frac{g}{6} q(t_1)^3 dt_1 + \frac{1}{2} \left(\int \frac{g}{6} q(t_1)^3 dt_1\right) \left(\int \frac{g}{6} q(t_2)^3 dt_2\right) + \cdots$.
Expanding $\exp(\int J q dt)$:
$\exp\left(\int J q dt\right) = 1 + \int J(t_1) q(t_1) dt_1 + \frac{1}{2} \int J(t_1) q(t_1) dt_1 \int J(t_2) q(t_2) dt_2 + \cdots$.

The 1-point function $\langle q(t) \rangle$ is $\frac{\delta W[J]}{\delta J(t)}|_{J=0}$.
$\frac{\delta}{\delta J(t)} \ln \langle \exp(\int J q) \exp(\int \frac{g}{6} q^3) \rangle_0 = \frac{1}{\langle \dots \rangle_0} \frac{\delta}{\delta J(t)} \langle \exp(\int J q) \exp(\int \frac{g}{6} q^3) \rangle_0$.
$\frac{\delta}{\delta J(t)} \langle \exp(\int J q') \exp(\int \frac{g}{6} q'^3) \rangle_0 = \langle q(t) \exp(\int J q') \exp(\int \frac{g}{6} q'^3) \rangle_0$.
Setting $J=0$: $\langle q(t) \exp(\int \frac{g}{6} q'^3) \rangle_0$.
$\langle q(t) (1 + \int \frac{g}{6} q(t_1)^3 dt_1 + \dots) \rangle_0 = \langle q(t) \rangle_0 + \int \frac{g}{6} \langle q(t) q(t_1)^3 \rangle_0 dt_1 + \dots$.
Since $\langle q(t) \rangle_0 = 0$, the first term is zero.
The next term is $\int \frac{g}{6} \langle q(t) q(t_1)^3 \rangle_0 dt_1$.
$\langle q(t) q(t_1)^3 \rangle_0 = \langle q(t) q(t_1) q(t_1) q(t_1) \rangle_0$.
Using Wick contractions:
There are 3 ways to contract $q(t)$ with one of the $q(t_1)$ fields:
$3 \times G_0(t, t_1) \langle q(t_1) q(t_1) q(t_1) \rangle_0$.
Since $\langle q(t_1) q(t_1) q(t_1) \rangle_0 = 0$ (odd number of fields), this is zero.
The other contractions are of the form $G_0(t, t_1) G_0(t_1, t_1) G_0(t_1, t_1)$, which is not correct.

Let's use the Feynman diagram for the 1-point function:
A vertex of degree 3 at time $t_v$ with coupling $-g/6$. One leg is external (at time $t$). The other two legs contract.
$\langle q(t) \rangle^{(g)} = \int dt_v \left(-\frac{g}{6}\right) \langle q(t) q(t_v) q(t_v) \rangle_0$.
As derived before: $\langle q(t) q(t_v) q(t_v) \rangle_0 = G_0(t, t_v) G_0(t_v, t_v)$.
$\langle q(t) \rangle^{(g)} = \int dt_v \left(-\frac{g}{6}\right) G_0(t, t_v) G_0(t_v, t_v)$.
This is the contribution of the "tadpole on an external leg" diagram.

The integral $\int_{-\infty}^{\infty} dt_v \, e^{-m|t - t_v|} = \frac{2}{m}$.
So, $\langle q(t) \rangle^{(g)} = \left(-\frac{g}{6}\right) \left(\frac{1}{2m}\right) \left(\frac{1}{2m}\right) \left(\frac{2}{m}\right) = -\frac{g}{6} \frac{2}{4m^3} = -\frac{g}{12m^3}$.

Wait, this is a specific value. The question asks for the leading term.
The *leading term* of $\langle q(t) \rangle$ with respect to $g$ is the first non-zero term in the perturbative expansion. As we saw, the $g$ term is:
$\langle q(t) \rangle^{(g)} = -\frac{g}{12m^3}$.
This term is constant in time.

Let's double check the symmetry argument. For an odd potential $U(q)$, the expectation value $\langle q \rangle$ is generally non-zero. The symmetry of the field $q$ under $q \to -q$ is broken.
If $S[q] = S_0[q] + S_{int}[q]$, then $Z[J] = \int \mathcal{D}q \, e^{-S_0[q] - S_{int}[q] + \int J q dt}$.
If $S_{int}[q]$ is odd, then $S_{int}[-q] = -S_{int}[q]$.
Let's consider $\langle q(t) \rangle$.
$\langle q(t) \rangle = \frac{\int \mathcal{D}q \, q(t) e^{-S[q] + \int J q dt}}{\int \mathcal{D}q \, e^{-S[q] + \int J q dt}} \Big|_{J=0}$.
$\langle q(t) \rangle = \frac{\int \mathcal{D}q \, q(t) e^{-S[q]}}{\int \mathcal{D}q \, e^{-S[q]}} \Big|_{J=0}$.
Let $q' = -q$. Then $\mathcal{D}q' = \mathcal{D}q$.
$\int \mathcal{D}q \, q(t) e^{-S[q]} = \int \mathcal{D}q' \, (-q'(t)) e^{-S[-q']} = \int \mathcal{D}q \, (-q(t)) e^{-S_0[q] + S_{int}[q]}$.
So $\langle q(t) \rangle = \frac{\int \mathcal{D}q \, (-q(t)) e^{-S_0[q] + S_{int}[q]}}{\int \mathcal{D}q \, e^{-S_0[q] - S_{int}[q]}}$.
This does not directly imply $\langle q(t) \rangle = 0$.

The leading term of $\langle q(t) \rangle$ is the lowest order non-zero term in $g$. This is indeed the tadpole diagram.
The calculation:
$\langle q(t) \rangle^{(g)} = \int dt_v \left(-\frac{g}{6}\right) G_0(t, t_v) G_0(t_v, t_v)$
$= -\frac{g}{6} \frac{1}{2m} \frac{1}{2m} \int_{-\infty}^{\infty} dt_v \, e^{-m|t - t_v|}$
$= -\frac{g}{24m^2} \left(\frac{2}{m}\right) = -\frac{g}{12m^3}$.
This term is constant in time.

**(ii) Connected 2-point function $\langle q(t_1) q(t_2) \rangle_c$ modulo $g^3$.**

The connected 2-point function is given by $G_2^c(t_1, t_2) = \langle q(t_1) q(t_2) \rangle_c$.
In terms of Feynman diagrams, this is the sum of all connected diagrams with two external legs, with all internal lines being free propagators and all vertices being interaction vertices. These diagrams are 1PI (one-particle irreducible) or reducible (built from 1PI parts connected by propagators).
The relation is $G_2 = G_0 + G_0 \star \Sigma \star G_0$, where $G_2$ is the full 2-point function and $\Sigma$ is the self-energy (sum of 1PI 2-point diagrams).
The connected 2-point function is $G_2^c = G_2 - G_0$. So $G_2^c = G_0 \star \Sigma \star G_0$.
We need contributions up to $g^2$.

**Order $g^0$ (free theory):**
$\langle q(t_1) q(t_2) \rangle_c^{(0)} = G_0(t_1, t_2) = \frac{1}{2m} e^{-m|t_1 - t_2|}$.
However, this term is *not* considered part of the $g$-expansion for the *connected* part if we define $G_2^c$ as the part that vanishes in the free theory. The standard definition of $G_2^c$ is the sum of all connected diagrams with two amputated external legs.

**Order $g^1$ (1-loop):**
The only diagram with one cubic vertex and two external legs is the "tadpole" diagram *without* the external leg attached. The vertex has 3 half-edges. Two contract to form a loop, and one is external.
$\langle q(t_1) q(t_2) \rangle_c^{(g)} = \int dt_v \left(-\frac{g}{6}\right) G_0(t_1, t_v) G_0(t_v, t_2) G_0(t_v, t_v)$.
$= -\frac{g}{6} \int dt_v \, G_0(t_1, t_v) G_0(t_v, t_2) \frac{1}{2m}$.
The integral $\int_{-\infty}^{\infty} dt_v \, e^{-m|t_1 - t_v|} e^{-m|t_v - t_2|} = e^{-m|t_1 - t_2|} \frac{2}{m}$.
So, $\langle q(t_1) q(t_2) \rangle_c^{(g)} = -\frac{g}{6} \left(\frac{1}{2m}\right) \left(\frac{2}{m}\right) G_0(t_1, t_2) = -\frac{g}{6m^2} G_0(t_1, t_2) = -\frac{g}{12m^3} e^{-m|t_1 - t_2|}$.

This term is a correction to the propagator, effectively modifying the mass.
The self-energy $\Sigma^{(1)}$ is related to this:
$\Sigma^{(1)}(t_1, t_2) = -\frac{g}{6} G_0(t_1, t_2) G_0(t_v, t_v) \delta(t_1 - t_2)$? No.
The tadpole diagram is $\Sigma^{(1)} = (-g/6) G_0(t,t)$. This is a local correction to the mass term in the inverse propagator.
$\Sigma^{(1)} = (-g/6) (1/2m) \delta(t_1 - t_2) = -g/(12m) \delta(t_1 - t_2)$.
Then $G_2^c = G_0 \star \Sigma^{(1)} \star G_0$.
Convolution of $G_0$ with $\delta$ is $G_0$. So $G_2^c = G_0 \star (-g/12m) \star G_0 = (-g/12m) G_0 \star G_0$.
$\int dt_v G_0(t_1, t_v) G_0(t_v, t_2) = \frac{1}{4m^2} \int dt_v e^{-m|t_1-t_v|} e^{-m|t_v-t_2|} = \frac{1}{4m^2} \frac{2}{m} e^{-m|t_1-t_2|} = \frac{1}{2m^3} e^{-m|t_1-t_2|}$.
So $G_2^c = (-g/12m) \frac{1}{2m^3} e^{-m|t_1-t_2|} = -g/(24m^4) e^{-m|t_1-t_2|}$.

Let's re-evaluate the tadpole diagram contribution to $\langle q(t_1) q(t_2) \rangle_c$.
The diagram is a $q^3$ vertex connected to $t_1$ and $t_2$, with the third leg of the vertex contracting with itself.
Contribution is $\int dt_v (-\frac{g}{6}) G_0(t_1, t_v) G_0(t_2, t_v) G_0(t_v, t_v)$.
$= -\frac{g}{6} G_0(t_v, t_v) \int dt_v G_0(t_1, t_v) G_0(t_2, t_v)$
$= -\frac{g}{6} \frac{1}{2m} \left(\frac{1}{2m^2} e^{-m|t_1-t_2|}\right)$
$= -\frac{g}{12m} \frac{1}{2m^2} e^{-m|t_1-t_2|} = -\frac{g}{24m^3} e^{-m|t_1-t_2|}$.
This is the $g^1$ contribution.

**Order $g^2$ (2-loops):**
We need to sum all connected diagrams with two $q^3$ vertices and two external legs, up to order $g^2$.
Possible diagrams:
1.  **Two tadpoles on separate legs:** This is reducible. Two $q^3$ vertices, each with one external leg and two internal legs contracting. The two vertices are not connected to each other. This is not 1PI.
2.  **"Bow-tie" or "butterfly" diagram:** Two $q^3$ vertices, each connected to one external leg ($t_1$ for vertex 1, $t_2$ for vertex 2). The remaining two legs of vertex 1 contract with each other, and the remaining two legs of vertex 2 contract with each other. This is two separate tadpoles. This diagram is **reducible** and does not contribute to the *connected* 2-point function at this order if we think in terms of $\Sigma$.
    Actually, the question asks for $G_2^c$. $G_2^c = G_0 \star \Sigma \star G_0$.
    $\Sigma$ contains 1PI diagrams.
    For $G_2^c$ up to $g^2$:
    - $g^0$: $G_0(t_1, t_2)$ (free propagator)
    - $g^1$: $G_0 \star \Sigma^{(1)} \star G_0$, where $\Sigma^{(1)}$ is the 1PI tadpole diagram.
    - $g^2$: $G_0 \star \Sigma^{(2)} \star G_0$. $\Sigma^{(2)}$ contains 2-loop 1PI diagrams.

Let's list the diagrams contributing to $G_2^c$ up to $g^2$:
*   **Order $g^0$:** $G_0(t_1, t_2)$. (This is the free propagator, often not explicitly written as a $g$-expansion term for $G_2^c$, but it's the base).
*   **Order $g^1$:** The "tadpole on an external leg" diagram. This is an interaction vertex with one external leg and a self-loop.
    Diagram: Vertex at $t_v$, external leg at $t_1$, external leg at $t_2$. Two legs of vertex $v$ contract.
    This is actually the contribution to $G_2$ from $G_0 + G_0 \star \Sigma \star G_0$. The $\Sigma^{(1)}$ is a local mass correction.
    $\Sigma^{(1)}(t_1, t_2) = -\frac{g}{12m} \delta(t_1 - t_2)$.
    $G_2^c(t_1, t_2)^{(g)} = G_0(t_1, t_2) + G_0 \star \Sigma^{(1)} \star G_0$.
    The first term $G_0$ is order $g^0$.
    The second term: $G_0 \star \Sigma^{(1)} \star G_0 = \int dt_v dt_u G_0(t_1, t_v) \Sigma^{(1)}(t_v, t_u) G_0(t_u, t_2)$
    $= \int dt_v dt_u G_0(t_1, t_v) \left(-\frac{g}{12m} \delta(t_v - t_u)\right) G_0(t_u, t_2)$
    $= -\frac{g}{12m} \int dt_v G_0(t_1, t_v) G_0(t_v, t_2) = -\frac{g}{12m} \left(\frac{1}{2m^2} e^{-m|t_1-t_2|}\right) = -\frac{g}{24m^3} e^{-m|t_1-t_2|}$.
    So, $G_2^c(t_1, t_2)^{(g)} = -\frac{g}{24m^3} e^{-m|t_1-t_2|}$.

*   **Order $g^2$:** We need diagrams contributing to $G_2^c$ that have two $q^3$ vertices.
    These can be built from $\Sigma^{(2)}$ and then convolved with $G_0$.
    $\Sigma^{(2)}$ contains 2-loop 1PI diagrams. For a cubic interaction:
    1.  **Two tadpoles connected by a propagator:** Two $q^3$ vertices, each with one self-loop. The two vertices are connected by a propagator. This is 1PI.
        Diagram: $t_1$ -- (vertex 1) -- $t_v$ -- $t_u$ -- (vertex 2) -- $t_2$. Vertex 1 has a loop, vertex 2 has a loop.
        $\Sigma^{(2)}_{\text{loop-link-loop}}(t_1, t_2) = \int dt_v dt_u (-\frac{g}{6})^2 \, G_0(t_1, t_v) G_0(t_v, t_v) \, G_0(t_v, t_u) \, G_0(t_u, t_u) G_0(t_u, t_2)$.
        $= \left(-\frac{g}{6}\right)^2 \left(\frac{1}{2m}\right) \left(\frac{1}{2m}\right) \int dt_v dt_u G_0(t_1, t_v) G_0(t_v, t_u) G_0(t_u, t_2)$.
        The integral is $\int dt_v dt_u G_0(t_1, t_v) G_0(t_v, t_u) G_0(t_u, t_2) = (\frac{1}{2m^3} e^{-m|t_1-t_2|})$.
        So, $\Sigma^{(2)}_{\text{loop-link-loop}} = \frac{g^2}{36} \frac{1}{4m^2} \frac{1}{2m^3} e^{-m|t_1-t_2|} = \frac{g^2}{72m^5} e^{-m|t_1-t_2|}$.
        This contribution to $G_2^c$ is $G_0 \star \Sigma^{(2)}_{\text{loop-link-loop}} \star G_0$. This will have higher powers of $e^{-m|\cdot|}$.

    Let's re-evaluate the structure of $G_2^c$ directly.
    $G_2^c(t_1, t_2) = \sum_{\text{connected diagrams with 2 external legs}}$.
    *   **Order $g^0$:** $G_0(t_1, t_2)$. This is the fundamental propagator.
    *   **Order $g^1$:** The diagram is a $q^3$ vertex connecting $t_1$ and $t_2$, with the third leg forming a loop.
        Diagram: $t_1$ -- (vertex at $t_v$) -- $t_2$, with $t_v$ connected to itself.
        $\langle q(t_1) q(t_2) \rangle_c^{(g)} = \int dt_v \left(-\frac{g}{6}\right) G_0(t_1, t_v) G_0(t_2, t_v) G_0(t_v, t_v)$.
        $= -\frac{g}{6} \frac{1}{2m} \int dt_v G_0(t_1, t_v) G_0(t_2, t_v)$
        $= -\frac{g}{12m} \left(\frac{1}{2m^2} e^{-m|t_1-t_2|}\right) = -\frac{g}{24m^3} e^{-m|t_1-t_2|}$. This agrees with earlier.

    *   **Order $g^2$:** Diagrams with two $q^3$ vertices.
