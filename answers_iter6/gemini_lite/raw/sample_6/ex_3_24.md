## Reasoning for the Main Problem

**Part 1: 1-loop contribution to $\log(Z/Z_0)$**

The partition function is given by $Z = \int e^{-S(x)/\hbar} dx$. The action is $S(x) = \tfrac{x^2}{2} - g(x^2 + \tfrac{x^3}{6})$.
We are interested in $\log(Z/Z_0)$, where $Z_0$ is the partition function for $g=0$.
For $g=0$, $S_0(x) = \tfrac{x^2}{2}$. This is a Gaussian integral.
$Z_0 = \int e^{-x^2/(2\hbar)} dx = \sqrt{2\pi\hbar}$.

Now consider $Z$ for $g \neq 0$. We can rewrite the action by completing the square for the $x^2$ terms:
$S(x) = \tfrac{x^2}{2} - gx^2 - \tfrac{g}{6}x^3 = \left(\tfrac12 - g\right)x^2 - \tfrac{g}{6}x^3 = \tfrac{1-2g}{2}x^2 - \tfrac{g}{6}x^3$.

For small $g$, the term $\tfrac{1-2g}{2}x^2$ is like a shifted mass-squared term. Let's define a new variable $y$ such that the quadratic part is canonical.
Let $x = c y$. Then $S(x) = \tfrac{1-2g}{2} c^2 y^2 - \tfrac{g}{6} c^3 y^3$.
We want $\tfrac{1-2g}{2} c^2 = \tfrac12$ to make it canonical. So $c^2 = \tfrac{1}{1-2g}$. Let $c = (1-2g)^{-1/2}$.
Then $x = (1-2g)^{-1/2} y$.
$S(x) = \tfrac12 y^2 - \tfrac{g}{6} (1-2g)^{-3/2} y^3$.

The effective action at 1-loop level is given by $S_{\text{eff}}(y_c) = S(y_c) + \hbar S_1(y_c)$, where $S_1$ is the sum of all 1PI 1-loop diagrams. For a theory with action $S(y)$, the 1-loop correction to the effective action is $S_1(y_c) = \tfrac{\hbar}{2} \log \det(S''(y_c))$, where $S''(y_c)$ is the second derivative of the *bare* action with respect to $y$ evaluated at $y_c$. Here, $S(y) = \tfrac12 y^2 - \tfrac{g}{6} (1-2g)^{-3/2} y^3$.

$S'(y) = y - \tfrac{g}{2} (1-2g)^{-3/2} y^2$.
$S''(y) = 1 - g (1-2g)^{-3/2} y$.

So, the 1-loop effective action for $y$ is:
$S_{\text{eff}}^{(1\text{-loop})}(y_c) = \tfrac{\hbar}{2} \log \left(1 - g (1-2g)^{-3/2} y_c\right)$.

We need to express this in terms of the original variable $x$. Since $x = c y$, $y = x/c = x (1-2g)^{1/2}$.
So, $y_c = x_c (1-2g)^{1/2}$.
Substitute this into $S_{\text{eff}}^{(1\text{-loop})}$:
$S_{\text{eff}}^{(1\text{-loop})}(x_c) = \tfrac{\hbar}{2} \log \left(1 - g (1-2g)^{-3/2} \cdot x_c (1-2g)^{1/2}\right)$
$S_{\text{eff}}^{(1\text{-loop})}(x_c) = \tfrac{\hbar}{2} \log \left(1 - g (1-2g)^{-1} x_c\right)$.

Now, we need to relate this to $\log(Z/Z_0)$.
$\log Z = \log \int e^{-S(x)/\hbar} dx$.
The 1-loop contribution to $\log Z$ (which is $-S_1/\hbar$) is given by $\tfrac{\hbar}{2} \log \det(\text{matrix of second derivatives of bare action})$.
The bare action is $S(x) = \tfrac{1-2g}{2}x^2 - \tfrac{g}{6}x^3$.
The second derivative is $S''(x) = 1-2g - gx$.
So, the 1-loop contribution to $\log Z$ (from the Gaussian integral over fluctuations) is:
$\log Z_{\text{1-loop}} = \tfrac{\hbar}{2} \log \left(\frac{1}{1-2g - gx}\right) = -\tfrac{\hbar}{2} \log (1 - (2g+gx))$.

This is the contribution from the vacuum fluctuations around $x=0$.
The problem asks for $\log(Z/Z_0)$.
$\log Z = \log Z_0 + \log(Z/Z_0)$.
$\log Z = \log \int e^{-S_0(x)/\hbar - g(\dots)/\hbar} dx$.
$\log Z = \log \left( \int e^{-S_0(x)/\hbar} e^{-g(\dots)/\hbar} dx \right)$.
Let $S_{\text{int}}(x) = x^2 + x^3/6$.
$Z = \int e^{-x^2/(2\hbar)} e^{g(x^2+x^3/6)/\hbar} dx$.
$Z = Z_0 \left\langle e^{g(x^2+x^3/6)/\hbar} \right\rangle_0$, where $\langle \cdot \rangle_0$ is the average w.r.t. $S_0$.
$\log(Z/Z_0) = \log \left\langle e^{g(x^2+x^3/6)/\hbar} \right\rangle_0$.
Expanding the exponential:
$\log(Z/Z_0) = \log \left\langle 1 + \frac{g}{\hbar}(x^2 + \tfrac{x^3}{6}) + \frac{g^2}{2\hbar^2}(x^2 + \tfrac{x^3}{6})^2 + \dots \right\rangle_0$.
The 1-loop contribution comes from the term linear in $g/\hbar$:

For $S_0(x) = x^2/(2\hbar)$, the Gaussian integral $\int e^{-x^2/(2\hbar)} x^n dx$ has:

$\langle x^3 \rangle_0 = 0$ (odd moment of Gaussian).

So, the 1-loop contribution to $\log(Z/Z_0)$ is:

For a zero-dimensional field, the determinant is just the value of the function. So,
